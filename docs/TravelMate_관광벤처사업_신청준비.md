### 3-2. 문제 인식 (관광 관점)

#### Problem 1: 1인 관광객 증가 vs 인프라 부재

```
[현황]
- 1인 여행 비율: 25% 이상 (2024 한국관광공사)
- 혼자 여행 시 불편함 경험: 67%
- 동행 있으면 좋겠다 응답: 72%

[문제점]
- 1인 메뉴 없는 식당
- 사진 찍어줄 사람 없음
- 안전 우려 (특히 여성)
- 현지 정보 교류 어려움

[TravelMate 해결]
→ 신뢰할 수 있는 동행 매칭 시스템
```

#### Problem 2: 수도권 집중 vs 지역 관광 침체

```
[현황]
- 서울/경기 관광객 집중: 전체의 65%
- 비수도권 관광 수입 감소
- 지역 관광 콘텐츠 부족

[TravelMate 해결]
→ 지역 한정 NFT로 소도시 방문 유도
→ "이 NFT는 전주에서만 수집 가능!"
```

#### Problem 3: 관광 기록의 휘발성

```
[현황]
- 여행 후 기록 남기는 비율: 23%
- SNS 포스팅 후 잊혀짐
- 방문 인증 시스템 부재

[TravelMate 해결]
→ 블록체인 기반 영구 보존
→ 디지털 수집품으로 여행 아카이빙
```
