## 2. 왜 정부가 창업지원을 하는가?

### 정부 창업지원의 본질적 목적

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  정부가 3.3조원을 창업지원에 쓰는 이유:                         │
│                                                                 │
│  1. 양질의 일자리 창출                                          │
│     → 청년 실업 해소                                            │
│     → 중장년 재취업 기회                                        │
│                                                                 │
│  2. 사회 문제 해결                                              │
│     → 고령화, 돌봄 공백, 지방 소멸                              │
│     → 시장이 해결 못하는 문제를 창업이 해결                     │
│                                                                 │
│  3. 지역 균형 발전                                              │
│     → 기초지자체 예산 45% 증가 (2025)                           │
│     → 지방소멸대응기금: 연 1조원 × 10년                         │
│                                                                 │
│  4. 경제 활성화                                                 │
│     → 새로운 산업 생태계                                        │
│     → 글로벌 경쟁력                                             │
│                                                                 │
│  핵심 인사이트:                                                 │
│  "정부가 돈을 주는 이유 = 그 문제가 심각하다는 뜻"              │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2025년 정부 중점 지원 분야

| 분야 | 왜 지원하는가? |
|------|---------------|
| AI/반도체 | 글로벌 경쟁력, 일자리 |
| 바이오/헬스 | 고령화 대응 |
| 기후/친환경 | 탄소중립 의무 |
| 지역/로컬 | 지방 소멸 대응 |
| 소셜벤처 | 사회 문제 해결 |

## 5. 창업 아이템 도출: 니즈 → 서비스

### 접근법: "확실히 증가하는 사용자" + "해결 안 된 니즈"

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  좋은 창업 아이템의 조건:                                       │
│                                                                 │
│  1. 사용자가 확실히 증가한다                                    │
│     → 고령자, 1인가구는 절대 줄지 않음                          │
│                                                                 │
│  2. 니즈가 해결되지 않고 있다                                   │
│     → 정부도 예산 부족, 시장도 안 풀린 문제                     │
│                                                                 │
│  3. 정부가 지원하고 싶어한다                                    │
│     → 사회 문제 해결 = 정부 지원 확률 높음                      │
│                                                                 │
│  4. 1인 창업자가 할 수 있다                                     │
│     → 플랫폼/매칭/중개 모델                                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 아이템 1: 시니어 생활 도우미 매칭 플랫폼

### 왜 이 아이템인가?

```
사용자층:
• 돌봄 필요 노인: 급증 (65세+ 1,012만명)
• 독거노인 가족: 안부 확인 원함
• 중장년 구직자: 일자리 원함 (187만명+)

니즈:
• 노인: 집에서 살고 싶다, 병원 가기 어렵다
• 가족: 부모님 돌볼 시간 없다, 죄책감
• 중장년: 경력 없어도 할 수 있는 일자리

현재 문제:
• 요양보호사 부족
• 정부 예산 부족
• 민간 서비스 거의 없음
```

### 서비스 모델

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  서비스명: 효도메이트 / 시니어헬퍼                              │
│                                                                 │
│  핵심 기능:                                                     │
│  1. 병원 동행 서비스                                            │
│     → 노인 혼자 병원 가기 어려움                                │
│     → 접수/대기/수납 동행                                       │
│                                                                 │
│  2. 장보기/외출 동행                                            │
│     → 마트, 은행, 관공서                                        │
│                                                                 │
│  3. 말벗/안부 확인                                              │
│     → 정기 방문, 화상 통화                                      │
│     → 자녀에게 리포트 전송                                      │
│                                                                 │
│  4. 집안일 도움                                                 │
│     → 청소, 정리, 간단한 수리                                   │
│                                                                 │
│  도우미:                                                        │
│  • 50~60대 중장년 (재취업 원하는)                               │
│  • 시간제 근무 (주 2~3회)                                       │
│  • 지역 기반 매칭                                               │
│                                                                 │
│  수익 모델:                                                     │
│  • 시간당 ₩20,000~30,000                                        │
│  • 플랫폼 수수료 15~20%                                         │
│  • 월 정기 구독 (주 2회 방문)                                   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 왜 정부가 지원하는가?

```
• 고령화 문제 해결
• 중장년 일자리 창출
• 돌봄 공백 메우기
• 고독사 예방
• 지역 기반 서비스

관련 지원:
• 소셜벤처 지원사업
• 예비창업패키지 (사회문제 해결형)
• 지역 돌봄 관련 기금
```

## 아이템 2: 지방 거주 청년/중장년 원격근무 일자리 플랫폼

### 왜 이 아이템인가?

```
사용자층:
• 지방 거주 청년: 일자리 없어 수도권 이동
• 지방 거주 중장년: 재취업 기회 없음
• 수도권 기업: 인력 부족, 비용 절감 원함

니즈:
• 지방 청년: 고향에서 일하고 싶다
• 지방 중장년: 서울 가지 않고 일하고 싶다
• 기업: 저렴한 인력, 원격근무 가능

현재 문제:
• 지방소멸 (60% 소멸위험)
• 인구 유출 심각
• 일자리 = 수도권 집중
```

### 서비스 모델

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  서비스명: 로컬워크 / 지역일자리                                │
│                                                                 │
│  핵심 기능:                                                     │
│  1. 원격근무 가능 일자리 매칭                                   │
│     → 고객상담, 데이터 입력, 번역                               │
│     → 콘텐츠 제작, 디자인                                       │
│     → 영상 편집, 마케팅                                         │
```
