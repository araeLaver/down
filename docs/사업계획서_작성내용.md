## 2. 주요 개요

### 아이템명

```
TravelMate - AI 여행 동행 매칭 및 위치 인증 NFT 수집 플랫폼
```

### 사업개요

```
TravelMate는 AI 기반 여행 동행 매칭과 블록체인 위치 인증 NFT 수집을 결합한 모바일 플랫폼입니다.

[문제 인식]
국내 1인 여행자 수는 매년 15% 이상 증가하고 있으나, 혼자 여행하는 MZ세대(20~30대)가 현지에서 동행자를 찾기 어렵고, 여행 추억을 체계적으로 기록·증명할 수단이 부재합니다. 기존 여행 커뮤니티(예: 네이버 카페, 에브리타임)는 사전 매칭에 그치며, 실시간 현지 매칭과 여행 기록의 디지털 자산화를 동시에 제공하는 서비스는 국내에 전무합니다.

[해결 방안]
TravelMate는 (1) AI가 여행자의 취향·일정·위치를 분석하여 최적의 동행자를 실시간 추천하고, (2) GPS 기반 위치 인증을 통해 실제 방문한 관광지를 블록체인(Polygon) NFT로 자동 발행하여 수집 가능하게 합니다. 이를 통해 "혼자 떠나도, 함께 추억을 만드는 여행" 경험을 제공합니다.

[사업 목표]
2026년 MVP 출시 후 1년 내 MAU 5만 명, NFT 누적 발행 10만 건 달성을 목표로 하며, 3년 내 연 매출 50억원 규모의 여행·블록체인 융합 플랫폼으로 성장하고자 합니다.
```

### 제품 및 서비스 소개

```
TravelMate는 다음 4가지 핵심 서비스로 구성됩니다.

1. AI 동행 매칭 시스템
   - 여행자의 관심사(맛집, 자연, 문화체험 등), 여행 스타일(활동적/여유), 일정, 현재 위치를 AI가 종합 분석하여 호환성 높은 동행자를 실시간 추천합니다.
   - 매칭 정확도 향상을 위해 협업 필터링 + 콘텐츠 기반 하이브리드 추천 알고리즘을 적용합니다.
   - 매칭 후 그룹 채팅방이 자동 생성되며, 일정 공유·실시간 위치 공유 기능을 제공합니다.

2. GPS 위치 인증 NFT 수집
   - 관광지에 실제 방문하면 GPS 좌표 인증을 통해 해당 장소의 고유 NFT가 자동 발행됩니다.
   - Polygon 블록체인(ERC-721)을 사용하여 이더리움 대비 가스비 99% 절감(건당 약 1~5원)으로 대중적 접근성을 확보합니다.
   - 관광지별 한정판 NFT, 시즌 한정 NFT, 희귀 등급 NFT 등을 통해 수집 재미를 극대화합니다.

3. NFT 마켓플레이스
   - 수집한 NFT를 다른 사용자와 거래할 수 있는 인앱 마켓플레이스를 운영합니다.
   - 거래 수수료 3~5%로 수익을 창출하며, 희귀 NFT의 경우 경매 기능도 지원합니다.

4. B2B 지자체/관광지 연계 서비스
   - 지방자치단체, 관광공사와 협업하여 지역 한정 NFT를 발행하고, 특정 NFT 보유자에게 현지 할인·혜택을 제공합니다.
   - 지자체의 관광 활성화 및 방문 데이터 분석 니즈를 충족시킵니다.

[기술 스택]
- 백엔드: Spring Boot 3.4, PostgreSQL, Redis
- 프론트엔드: React 18 + TypeScript
- 모바일: React Native (Expo) - iOS/Android 동시 개발
- 블록체인: Polygon 네트워크, ERC-721 스마트 컨트랙트 (Solidity)
- AI/ML: Python (scikit-learn, TensorFlow Lite) - 추천 엔진
- 인프라: AWS (ECS, RDS, S3), IPFS (NFT 메타데이터 저장)
```

### 시장분석

```
1. 시장 규모 (TAM-SAM-SOM)
   - TAM (전체 시장): 글로벌 여행 산업 약 $1.2조 (한화 약 1,560조원)
   - SAM (유효 시장): 아시아 1인 여행 시장 약 $50B (약 65조원)
   - SOM (목표 시장): 한국 MZ세대 1인 여행자 약 $500M (약 6,500억원)

2. 국내 시장 동향
   - 한국관광공사 통계에 따르면, 국내 1인 여행자 비율은 2023년 27%에서 2025년 35%로 증가 추세입니다.
   - 20~30대 MZ세대의 '혼행(혼자 여행)' 검색량은 연평균 40% 이상 증가하고 있습니다.
   - NFT/블록체인 시장은 국내 약 2.5조원 규모이며, 실물 연동 유틸리티 NFT에 대한 수요가 급증하고 있습니다.

3. 경쟁사 분석
   - 여행 매칭: '여행에 미치다'(네이버 카페, 180만 회원) - 사전 텍스트 기반 모집만 가능, 실시간 AI 매칭 없음
   - 동행 앱: 'TripFriend' - 외국인 대상, 국내 MZ세대 타겟팅 미흡
   - 여행 NFT: 해외에 'Travala', 'Pinata' 등 존재하나 동행 매칭 연계 없음
   → 동행 매칭 + NFT 수집을 결합한 서비스는 국내외 전무하여, 선점 효과 극대화 가능

4. 타겟 고객
   - 1차 타겟: 20~35세 MZ세대 1인 여행자 (국내 약 800만 명)
   - 2차 타겟: 여행 기록·수집 취미 보유자, NFT/블록체인 관심층
   - 3차 타겟: 지자체, 관광공사 등 B2B 파트너

5. 진입 장벽 및 경쟁 우위
   - GPS 위치 인증 기반 NFT 발행 기술 (특허 출원 예정)
   - 국내 최초 여행 + 블록체인 융합 서비스로 선점 효과
   - Polygon 네트워크 활용으로 건당 가스비 1~5원 수준의 초저비용 운영
   - AI 매칭 알고리즘의 지속적 학습을 통한 정확도 향상
```

## 3. 주요 애로사항 및 기대효과

### 주요 애로사항

```
1. 기술 개발 역량 보강 필요
   - 블록체인(Polygon/Solidity) 스마트 컨트랙트 개발과 AI 추천 알고리즘 고도화를 위한 전문 개발 인력 확보가 시급합니다. 현재 대표자가 풀스택 개발을 수행하고 있으나, MVP 수준 이상의 안정적 서비스 운영을 위해 블록체인 전문 개발자 1인, 백엔드 개발자 1인의 채용이 필요합니다.

2. 초기 고객 확보 및 네트워크 효과 구축
   - 동행 매칭 서비스의 특성상, 일정 수 이상의 사용자가 확보되어야 매칭 품질이 높아지는 '네트워크 효과'가 존재합니다. 초기 사용자 확보를 위한 마케팅 비용 및 전략(대학생 여행 동아리, 인플루언서 협업 등)에 대한 전문가 자문이 필요합니다.

3. 블록체인 규제 환경 대응
   - NFT 거래에 대한 국내 법·제도적 환경이 아직 명확하지 않아, 세금 처리(소득세, 부가세), 개인정보 보호(GPS 위치 데이터), 가상자산 관련 규제(특금법) 등에 대한 법률 자문이 필수적입니다.

4. 지자체/관광기관 B2B 파트너십 구축
   - 지역 한정 NFT 발행 및 관광 데이터 연계를 위해 지방자치단체, 한국관광공사 등과의 협업 체계 구축이 필요하나, 예비 창업자로서 공공기관 접근이 어려운 상황입니다. 경기스타트업플랫폼을 통한 네트워킹 및 멘토링 지원이 큰 도움이 될 것입니다.
```

### 예상 기대효과

```
1. 사업적 기대효과
   - 국내 최초 AI 동행 매칭 + 위치 인증 NFT 융합 플랫폼으로 시장 선점
   - 출시 1년 내 MAU 5만 명, 누적 NFT 발행 10만 건 달성
   - 3년 내 연 매출 50억원 달성 및 시리즈 A 투자 유치(100억원 목표)
   - 고용 창출: 1년차 5명 → 3년차 20명 이상 (개발, 마케팅, 운영)

2. 기술적 기대효과
   - GPS 위치 인증 기반 NFT 발행 기술 특허 확보 (1건 이상)
   - AI 여행 추천 알고리즘 고도화로 매칭 정확도 90% 이상 달성
   - Polygon 블록체인 활용 초저비용 NFT 운영 모델 실증

3. 사회적 기대효과
   - 1인 여행자의 안전한 동행 문화 조성 (신원 인증, 리뷰 시스템)
   - 지역 관광 활성화: 지자체 한정 NFT를 통해 비수도권 관광지 방문 유도
   - 블록체인 기술의 실생활 적용 사례로 기술 대중화 기여
   - MZ세대의 새로운 디지털 수집 문화 창출

4. 전문가 지원 활용 기대효과
   - 기술 멘토링을 통한 블록체인 아키텍처 최적화 및 보안 강화
   - 경영/마케팅 자문을 통한 효과적인 초기 고객 확보 전략 수립
   - 법률 자문을 통한 NFT 거래 관련 규제 대응 체계 마련
   - 경기스타트업플랫폼 네트워크를 활용한 지자체 협업 기회 확보
```
