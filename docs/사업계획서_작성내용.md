## 2. 주요 개요

### 아이템명

```
Fryndo(프린도) - 1인 여행자를 위한 신뢰 기반 동행 매칭 플랫폼
```

### 사업개요

```
Fryndo(프린도)는 혼자 여행하는 MZ세대를 위한 신뢰 기반 동행 매칭 + GPS 기반 여행 NFT 수집 플랫폼입니다.

[문제 인식]
아시아 1인 여행 검색량은 전년 대비 16% 증가하고 있으며(Agoda, 2025), 한국인의 80%가 2026년 더 많이 여행할 계획입니다(Skyscanner, 2026). 그러나 1인 여행자의 78%가 안전을 최대 걱정으로 꼽고(Booking.com, 2025), 41%가 여행에서 함께할 동반자를 원합니다(Skyscanner, 2026). 기존 여행 플랫폼(트립어드바이저, 에어비앤비)은 숙소/관광지 정보만 제공할 뿐, 신뢰할 수 있는 동행자를 찾는 기능은 전무합니다.

[해결 방안]
Fryndo는 (1) 여행 스타일·일정·관심사를 기반으로 최적의 동행자를 매칭하고, (2) 신분증/SNS 인증을 통한 신뢰 프로필 시스템으로 안전한 만남을 보장하며, (3) GPS 기반 위치 인증으로 방문한 여행지를 블록체인(Polygon) NFT로 수집할 수 있게 합니다. 전략적으로 동행 매칭에 80%의 리소스를 집중하고, NFT는 차별화 요소로 탄력 운영합니다.

[사업 목표]
초기 타겟은 제주도 여행 25-34세 여성에 집중하며, 6개월 내 MAU 5,000명, 매칭 성사율 30% 이상 달성을 목표로 합니다. 이후 국내 전역 → 일본/동남아로 확장하여 3년 내 MAU 20만 명, 연 매출 13억원 규모의 여행 동행 매칭 플랫폼으로 성장하고자 합니다.
```

### 제품 및 서비스 소개

```
Fryndo는 다음 3가지 핵심 기능으로 구성됩니다.

1. 여행 동반자 매칭 시스템 (핵심 - 80% 집중)
   - 여행 스타일(활동적/여유), MBTI, 관심사, 일정, 목적지를 종합 분석하여 호환성 높은 동행자를 추천합니다.
   - 신분증/SNS 연동 인증 뱃지로 프로필 신뢰도를 보장합니다.
   - 여행 그룹 단위 장기 동행을 지원하며, 매칭 후 그룹 채팅방이 자동 생성됩니다.
   - 후기/평점 시스템으로 매칭 품질을 지속적으로 개선합니다.

2. GPS 기반 여행 NFT 수집 (차별화 요소)
   - 관광지에 실제 방문하면 GPS 좌표 인증을 통해 해당 장소의 고유 NFT가 자동 발행됩니다.
   - Polygon 블록체인(ERC-721)을 사용하여 가스비 $0.01 이하로 대중적 접근성을 확보합니다.
   - Common(60%)/Rare(30%)/Epic(9%)/Legendary(1%) 4단계 희귀도 시스템으로 수집 재미를 극대화합니다.
   - IPFS로 이미지를 영구 보존하여 여행 기록을 디지털 자산화합니다.

3. 안전 케어 시스템 (부가 가치)
   - 그룹원 간 실시간 위치 공유 기능을 제공합니다.
   - 원터치 긴급 SOS 버튼으로 자동 위치 전송 및 긴급연락처 호출이 가능합니다.
   - 신뢰 점수 시스템(인증 뱃지 + 후기 평점 + 여행 횟수)으로 안전한 매칭 환경을 조성합니다.

[기술 스택 - 창업자가 직접 구현 완료]
- 백엔드: Flask/Python, REST API
- 프론트엔드: React + Tailwind CSS (반응형 웹)
- 실시간 통신: WebSocket, STOMP 기반 그룹 채팅
- 블록체인: Polygon Network, ERC-721, Web3.js, IPFS
- 인증: Spring Security, JWT, OAuth 2.0 (카카오/구글)
- 데이터베이스: PostgreSQL + Redis
- 인프라: Koyeb (PaaS) + GitHub Actions CI/CD + Docker
- 서비스 URL: https://various-belva-untab-1a59bee2.koyeb.app

※ 창업자의 9년 풀스택 개발 경력으로 MVP 100% 자체 개발 완료
※ 크리에이티브힐에서 Polygon NFT 연동 실무 경험으로 블록체인 기능 검증 완료
```

### 시장분석

```
1. 시장 규모 (TAM-SAM-SOM)
   - TAM (전체 시장): 글로벌 1인 여행 시장 $4,823억 → 2030년 $1.07조 (CAGR 14.3%, Grand View Research 2025)
   - SAM (유효 시장): 아시아태평양 1인 여행 시장 $580억 (약 75조원), CAGR 16.1%로 가장 빠른 성장 지역
   - SOM (목표 시장): 한국+일본+동남아 시장 점유율 0.1% 기준 $5.8억 (약 7,500억원)

2. 시장 성장 트렌드 (2025-2026년 기준)
   - 아시아 1인 여행 검색량 전년 대비 +16% (Agoda, 2025)
   - 인도네시아 1인 여행 관심 +72%, 태국 +37% (Agoda, 2025)
   - 2026년 더 많이 여행 계획인 한국인 80% (Skyscanner, 2026)
   - 한국인 41%가 여행에서 친구/로맨스 추구 (Skyscanner, 2026)
   - 여성 1인 여행자 78%가 "안전이 최대 걱정" (Booking.com, 2025)
   - 아시아 1인 여행 인기 목적지: 도쿄 > 방콕 > 서울 > 오사카 > 쿠알라룸푸르

3. 타겟 고객 분석
   - 아시아태평양 1인 여행자 연령 분포: 25-34세 30%(Primary), 35-44세 23%(Secondary)
   - 초기 집중 타겟: 제주도 여행 25-34세 여성
     · 제주도 = 국내 1인 여행 1위 목적지
     · 여성 = 안전 니즈 높음 → 동행 수요 강함 (78%)
     · 25-34세 = 가장 활발한 여행 연령대 (30%)
   - 확장 순서: 제주도 → 국내 전역 → 일본 → 동남아(태국/베트남)

4. 경쟁사 분석 및 차별점
   - 트립어드바이저: 여행 정보 중심, 동반자 매칭 기능 없음
   - 에어비앤비 익스피리언스: 일회성 투어, 지속적 관계 형성 불가
   - Couchsurfing Hangouts: 현지인 만남 중심, 신뢰도 이슈
   - 틴더 (여행 모드): 로맨스 중심, 여행 동행 특화 아님
   → Fryndo의 차별점: 여행 목적 동행 매칭 + 신뢰 프로필 검증 + GPS NFT 수집을 결합한 서비스는 국내외 전무

5. 진입 장벽 및 경쟁 우위
   - 신뢰 기반 매칭 시스템 (인증 뱃지 + 후기 + 평점)으로 네트워크 효과 구축
   - 창업자의 매칭 알고리즘 개발 경험 (에너지마켓플레이스)과 블록체인 실무 경험 (크리에이티브힐)
   - MVP 100% 완성 상태로 즉시 시장 검증 가능
   - Polygon 활용 초저비용 NFT 운영 (가스비 $0.01 이하)
```

## 3. 주요 애로사항 및 기대효과

### 주요 애로사항

```
1. 초기 고객 확보 및 네트워크 효과 구축
   - 동행 매칭 서비스의 특성상, 일정 수 이상의 사용자가 확보되어야 매칭 품질이 높아지는 '네트워크 효과'가 존재합니다. 초기 타겟을 제주도 + 25-34세 여성으로 극도로 좁혀 집중하는 전략을 수립했으나, 초기 사용자 확보를 위한 인플루언서 마케팅, 제주 현지 커뮤니티 침투 등에 대한 전문가 자문이 필요합니다.

2. 마케팅/영업 역량 보강 필요
   - 창업자가 9년차 풀스택 개발자로서 기술 역량은 충분하나, 마케팅/영업 경험이 부족합니다. SNS 마케팅, 콘텐츠 전략, 인플루언서 협업 등 고객 획득 전략 수립에 대한 전문가 멘토링이 큰 도움이 될 것입니다.

3. 1인 창업의 한계 극복
   - MVP 개발은 1인으로 완성했으나, 서비스 운영/마케팅/고객응대를 1인이 동시에 수행하기 어렵습니다. 경기스타트업플랫폼의 전문가 지원을 통해 체계적인 1인 린스타트업 운영 방법론과 우선순위 설정에 대한 자문을 받고자 합니다.

4. 지자체/관광기관 B2B 파트너십 구축
   - 제주도 관광청, 한국관광공사 등과의 협업을 통해 지역 한정 NFT 발행 및 관광 데이터 연계를 계획하고 있으나, 예비 창업자로서 공공기관 접근이 어려운 상황입니다. 경기스타트업플랫폼의 네트워킹 및 멘토링 지원이 큰 도움이 될 것입니다.
```
