### 1. 기획성

□ 아이템 개요

입주해는 세입자가 자신의 신뢰도를 증명할 수 있는 AI 기반 프로필 플랫폼입니다. 재직·소득·신용 인증과 이전 집주인 레퍼런스를 통해 신뢰점수(Trust Score)를 산출하고, AI가 자기소개를 생성하여 집주인에게 검증된 디지털 프로필을 제시합니다. 전세사기 이후 무너진 임대차 시장의 신뢰를 기술로 복원합니다.

  ○ 핵심 문제 인식
    - 전세사기 피해자 3만 명 돌파(국토부, 2025.7), 피해액 2.5조원 이상. 피해자의 75%가 2030 청년층
    - 전세사기 이후 월세 전환 가속화 — 서울 빌라 임대차 중 월세 비중 60%(2025). 임대인·임차인 모두 상호 불신 심화
    - 2026년 전세 수요 23.5만 호 vs 입주 물량 8.9만 호 — 14만 호 수급 불균형. 세입자 간 경쟁 심화 예상
    - 세입자가 자신의 신뢰도를 체계적으로 증명할 수단이 전무. 집주인은 세입자 평가 기준이 "느낌"에 의존

  ○ 해결 방법 (3대 핵심 기능)
    - **신뢰점수(Trust Score)**: 프로필 완성도(+20), 재직인증(+25), 소득인증(+25), 신용등급(+10~20), 집주인 레퍼런스(+30) 등 100점 만점 정량 평가. 세입자 신뢰를 숫자로 증명
    - **공인 인증 시스템**: CODEF API 연동 재직·소득·신용 인증 + NICE 본인인증. 위변조 불가능한 공인 데이터 기반
    - **AI 자기소개 생성**: OpenAI 기반으로 프로필 데이터(라이프스타일·거주 습관·가족 구성)를 분석해 집주인에게 호감을 주는 자기소개 자동 생성

□ 차별성 및 혁신성

  ○ 기존 서비스와의 차별점
    - 직방/다방/네이버 부동산: 매물 정보 중심 → 세입자 프로필 기능 없음
    - 전세사기 방지 앱 (안심전세): 집/건물의 위험도 체크 → 세입자 관점 신뢰 증명 없음
    - 부동산 중개 플랫폼: 매물 매칭에 집중 → 세입자-집주인 간 신뢰 구축 미지원
    - **핵심 차별점**: "매물"이 아닌 "사람"에 집중. 세입자의 신뢰를 정량화하여 임대차 거래의 정보 비대칭 해소

  ○ 기술 혁신
    - AI: OpenAI API 기반 자기소개 자동 생성 — 프로필 데이터를 분석하여 집주인 관점에서 최적화된 소개 문구 작성
    - 공인 인증 연동: CODEF(재직/소득/신용) + NICE(본인인증) API 실시간 연동 — 서류 제출 없이 원클릭 인증
    - 레퍼런스 시스템: SMS 기반 이전 집주인 설문(월세 납부, 시설 관리, 이웃 관계, 퇴거 상태, 추천 여부 5항목) → 자동 신뢰점수 반영
    - 신뢰점수 알고리즘: 다차원 인증 데이터를 가중 합산하여 0~100+ 점수 산출. 등급별 시각화(Excellent/Good/Fair/Low)

### 2. 사업성

□ 시장 현황 및 목표 고객

  ○ 시장 규모
    - TAM(전체 시장): 국내 연간 임대차 거래 약 250만 건, 전월세 시장 규모 약 540조원
    - SAM(유효 시장): 수도권 임대차 거래 약 150만 건 × 디지털 활용 세입자 60% = 90만 건
    - SOM(초기 목표): 수도권 2030 MZ 세입자 타겟, 초기 점유율 0.5% = 4,500건/년 (월 375건)
    - 글로벌 프롭테크 시장: 2024년 $182억 → 2034년 $865억 (CAGR 16.9%)

  ○ 목표 고객
    - 1차 타겟(세입자): 수도권 25~35세 직장인 세입자 — 전세사기 불안, 첫 독립, 이직 잦은 청년층. "좋은 집 구하려면 좋은 세입자임을 증명"해야 하는 니즈
    - 2차 타겟(집주인): 다세대/오피스텔 임대인 — 세입자 검증 기준 부재, 공실 리스크 최소화 니즈
    - 3차 타겟(중개업소/관리업체): 부동산 중개사, 임대관리 업체 — 세입자 스크리닝 자동화 니즈

□ 수익 모델 (BM)

  ○ 매출 구조
    - **B2C 프리미엄 구독 (50%)**: ₩4,900/월 또는 ₩39,000/년
      · 무료: 기본 프로필 + 신뢰점수 조회 1회 + AI 자기소개 1회
      · 프리미엄: 무제한 인증 갱신, AI 자기소개 무제한, 프로필 공유 링크, 집주인 레퍼런스 요청 무제한, 프로필 조회 통계
    - **B2B SaaS (35%)**: 부동산 중개업소·임대관리 업체 대상 월 구독
      · 세입자 프로필 검색/열람, 신뢰점수 기반 필터링, 대시보드·통계, API 연동
      · 월 ₩49,000 (소규모) ~ ₩199,000 (대형)
    - **B2G 데이터/솔루션 (15%)**: 공공기관 대상
      · 임대차 신뢰 데이터 제공 (익명화), 전세사기 예방 시스템 연계, LH/SH 공공임대 세입자 스크리닝

  ○ 단위 경제 (Unit Economics)
    - CAC: ₩15,000 (온라인 마케팅 + 바이럴)
    - LTV: ₩88,200 (월 ₩4,900 × 평균 18개월 유지)

## 신청 절차

1. 경기스타트업플랫폼(gsp.or.kr) 회원가입 및 로그인
2. 마이스페이스 → 나의 정보 기본정보 확인
3. '2026년 경기 창업 공모(G스타 오디션)' 공고 클릭
4. '참여신청' 클릭
5. 정보 입력 + 파일 업로드 (5개 250MB 제한)
6. **'제출' 클릭** (임시저장만 하면 접수 불인정!)

## 주의사항

- 양식 항목 수정 불가, 내용 많으면 추가 가능 (신청서 제외 **10장 이내**)

- 예비·초기리그 입상 시 하반기 도약리그 평가 대상에서 자동 제외
- 예선(서류) 가산점: 청년대표 0.5점 (최대 4점 중)
