### Ⅰ. 기업 현황 및 입주 공간 필요성

#### □ 기업 현황

**창업배경**

전세사기 피해자 3만 명 돌파(국토부, 2025.7), 피해액 2.5조원 이상. 피해자의 75%가 2030 청년층입니다. 전세사기 이후 임대인·임차인 모두 상호 불신이 심화되었으나, 세입자가 자신의 신뢰도를 체계적으로 증명할 수단은 전무합니다. 집주인은 세입자 평가 기준이 "느낌"에 의존하고, 세입자는 좋은 거주 이력이 있어도 다음 계약에 활용할 수 없습니다.

현재 지미디어에서 개발자로 재직 중이며, 퇴근 후와 주말을 활용해 약 10년간 축적한 풀스택 개발 역량으로 MVP를 단독 개발했습니다. 특히 금융 인증 시스템(IBK 기업은행 전자약정, 헥토데이터 CODEF API)과 매칭 알고리즘(에너지마켓플레이스)을 직접 설계·구현한 경험이 이 문제를 기술로 해결할 수 있다는 확신을 주었습니다. "좋은 세입자도 증명할 방법이 없다"는 구조적 문제를 풀어, 세입자와 집주인 모두 안심할 수 있는 임대차 거래 환경을 만들고자 창업을 준비하고 있습니다.

현재 MVP는 99% 완성 상태(프로덕션 레디)이며, 창업혁신공간 입주를 기점으로 퇴사 후 풀타임 창업에 전념할 계획입니다.

**제품(서비스) 개요**

입주해(입주해)는 세입자가 자신의 신뢰도를 증명할 수 있는 AI 기반 프로필 플랫폼입니다.

◦ 개발배경: 전세사기 이후 무너진 임대차 시장의 신뢰를 기술로 복원. 기존 부동산 플랫폼(직방/다방/네이버부동산)은 "매물" 중심이지만, 입주해는 "사람"에 집중하여 세입자 신뢰를 정량화합니다.

◦ 3대 핵심 기능:

1) 신뢰점수(Trust Score) — 0~100+ 정량 평가
  - 프로필 완성도(+20점), 재직인증(+25점, CODEF API), 소득인증(+25점, CODEF API), 신용등급(+10~20점), 집주인 레퍼런스(+30점, SMS 설문)
  - 등급 시각화: Excellent(80+) / Good(60~79) / Fair(40~59) / Low(~39)

2) 공인 인증 시스템
  - CODEF API: 금융결제원 공인 데이터 기반 재직·소득·신용 인증
  - NICE API: 본인인증 (휴대폰 인증)
  - 위변조 불가능, 서류 제출 없이 원클릭 인증

3) AI 자기소개 생성
  - OpenAI API 기반, 프로필 데이터(라이프스타일·거주 습관·가족 구성) 분석
  - 집주인 관점에서 호감을 주는 자기소개 자동 생성

◦ 차별성:
  - 기존 서비스(직방/다방)는 매물 중심 → 세입자 프로필 기능 없음
  - 안심전세 앱은 건물 위험도 체크 → 세입자 관점 신뢰 증명 없음
  - 입주해는 "사람 중심 신뢰 플랫폼" — 한국 최초 임차인 레퍼런스 시스템

◦ MVP 구현 현황 (99% 완성):
  - 세입자 온보딩(3단계), 신뢰점수 시스템, 재직/소득/신용 인증(CODEF API), 집주인 레퍼런스(SMS), AI 자기소개(OpenAI), 집주인 대시보드, 메시징 시스템, 매물 관리, 소셜 로그인(카카오/네이버/구글), 보안(Rate Limiting/CSRF/XSS), 테스트(Vitest/Playwright) 전 기능 구현 완료

◦ 기술 스택:
  - Frontend: Next.js 14, React 18, TypeScript, Tailwind CSS
  - Backend: Next.js App Router, PostgreSQL, JWT 인증
  - AI: OpenAI API / 인증: CODEF, NICE API
  - 인프라: Vercel, AWS S3, Cloudflare R2, Sentry

**대표자 소개**

◦ 최종학력: 한국방송통신대학교 컴퓨터과학과 재학
◦ 주요 근무경력:
  - 지미디어 (2024~현재) — 풀스택 개발 (재직 중, 창업 시 퇴사 예정)
  - 헥토데이터(구 쿠콘) 파트장 (2021~2024, 3년) — CODEF·금융결제원 등 공인 인증 API 실무, 팀 관리
  - IBK 기업은행 파견 (2020~2021) — 전자약정 다중인증(MFA) 시스템 구현
  - 에너지마켓플레이스 (2019~2020) — 수요-공급 매칭 알고리즘 설계/개발
  - 크리에이티브힐 (2018~2019) — NFT B2C 플랫폼 기획·개발·운영
  - 지자체 웹서비스 (2016~2018) — 서귀포시/고양시 등 B2G 프로젝트
◦ 전문분야: 풀스택 개발(10년), 금융 인증 시스템, 매칭 알고리즘, AI 연동
◦ 특기사항: 기획→설계→개발→배포→운영 1인 수행 가능, 재직 중 퇴근·주말 활용하여 MVP 99% 단독 개발 완료

#### □ 입주 공간 필요성

◦ 입주 필요성:
  - 현재 예비창업 단계로 사업자등록을 위한 사업장 주소가 필요합니다. 하남 창업혁신공간 입주 시 즉시 법인 설립 및 서비스 런칭이 가능합니다.
  - 자택(경기도 광주시)에서는 안정적인 개발·운영 환경 확보가 어려우며, 전문 창업 공간이 사업 초기 집중도를 높이는 데 필수적입니다.
  - 경기도 동부권(하남·광주·성남) 부동산 중개업소 B2B 영업 시 지리적 접근성이 우수합니다.

◦ 공간 활용계획:
  - 제품 개발 및 운영: 서비스 고도화, 모바일 앱 개발, 버그 수정 등 상시 개발 업무 수행
  - 사업장 등록: 입주 후 3개월 이내 법인 설립 및 사업자등록증 본사 주소 등록
  - B2B 미팅: 동부권역 부동산 중개업소, 임대관리 업체 대상 영업 미팅 거점으로 활용
  - 투자 IR: 센터 내 회의실 활용하여 VC/AC 투자 미팅 진행

◦ 기대효과:
  - 무상 입주를 통한 초기 고정비 절감 → 사업화 자금(마케팅·API 비용)에 집중 투자
  - 센터 입주기업 간 네트워킹·오픈이노베이션 참여 → 기술 교류 및 협업 기회
  - 경영 멘토링·투자 상담 프로그램 → BM 고도화 및 Seed 투자 유치 준비
  - 경기도 창업 지원사업·전세피해지원센터 등 지역 자원 연계 활용

### Ⅱ. 주요 사업계획

**사업 목표**

◦ 기술개발 목표: 현재 99% 완성된 MVP를 정식 서비스로 전환하고, 모바일 앱(React Native) 개발 및 AI 자기소개 생성 모델 고도화를 추진합니다.
◦ 사업화 목표: 2026년 하반기 정식 출시 → 수도권 2030 직장인 세입자 MAU 3,000명 확보 → B2C 프리미엄 구독 출시로 월 매출 발생
◦ 기본 경영전략:
  - 입주 확정 후 퇴사 → 풀타임 창업 전환. 1인 린스타트업으로 시작, 매출 발생 후 팀 확장
  - "세입자 풀 확보 → 집주인 자연 유입" 순서로 양면 플랫폼 구축
  - B2C(프리미엄 구독) → B2B(중개업소 SaaS) → B2G(공공기관 데이터) 단계적 확장

**기술성**

◦ 국내외 관련 기술/산업현황:
  - 글로벌 프롭테크 시장: 2024년 $182억 → 2034년 $865억 (CAGR 16.9%)
  - 해외 유사 서비스: Naborly(캐나다, 세입자 스크리닝), RentProfile(호주) — 한국에는 동일 서비스 없음
  - 국내 부동산 플랫폼(직방/다방/네이버부동산)은 매물 중심, 세입자 신뢰 프로필 기능 전무

◦ 경쟁사 비교:

| 경쟁 서비스 | 기술/품질 | 가격 | 입주해 차별점 |
|:-----------|:---------|:-----|:-------------|
| 직방/다방 | 매물 검색·지도 | 무료/광고모델 | 세입자 프로필 없음 → 입주해는 "사람" 중심 |
| 네이버 부동산 | 시세·매물 정보 | 무료 | 매칭 없음 → 입주해는 신뢰 기반 매칭 |
| 안심전세 앱 | 건물 위험도 | 무료 | 세입자 관점 없음 → 입주해는 세입자 신뢰 정량화 |
| 피터팬의좋은방 | 직거래 커뮤니티 | 무료 | 인증 없음 → 입주해는 CODEF 공인 인증 |

◦ 핵심 기술 혁신:
  - AI 자기소개 생성: OpenAI API 기반, 프로필 데이터 분석 → 집주인 관점 최적화된 소개 문구 자동 작성
  - 공인 인증 연동: CODEF API(재직/소득/신용) + NICE(본인인증) 실시간 연동, 서류 없이 원클릭 인증
  - 레퍼런스 시스템: SMS 기반 이전 집주인 설문(월세 납부/시설 관리/이웃 관계/퇴거 상태/추천 여부 5항목) → 한국 최초
  - 신뢰점수 알고리즘: 다차원 인증 데이터 가중 합산 → 0~100+ 점수 산출, 등급별 시각화

◦ 지식재산권 보유 현황: 출원 예정 (서비스명 "입주해" 상표 출원 예정)

**사업성**

◦ 아이템 독창성: 국내 최초 "세입자 신뢰 프로필" 플랫폼. 기존 부동산 플랫폼은 매물 중심이나, 입주해는 세입자 개인의 신뢰를 정량화하여 임대차 거래의 정보 비대칭을 해소합니다.

◦ 시장규모(전망):

  - SAM: 수도권 임대차 거래 약 150만 건 × 디지털 활용 세입자 60% = 90만 건
  - SOM: 초기 점유율 0.5% = 4,500건/년 (월 375건)

◦ 주요 수요처:

  - 2차: 다세대/오피스텔 임대인 — 세입자 검증 기준 부재, 공실 리스크 최소화
  - 3차: 부동산 중개사·임대관리 업체 — 세입자 스크리닝 자동화

◦ 수익 모델:
  - B2C 프리미엄 구독 (50%): ₩4,900/월 또는 ₩39,000/년
  - B2B SaaS (35%): 부동산 중개업소·관리업체 ₩49,000~₩199,000/월
  - B2G 데이터/솔루션 (15%): 공공기관(LH/SH) 대상 맞춤 계약
  - 단위 경제: CAC ₩15,000 / LTV ₩88,200 / LTV:CAC = 5.9:1

◦ 경쟁사 분석: 직방(매물 중심, 세입자 프로필 없음), 다방(유사), 안심전세(건물 위험도만), 피터팬(인증 없음). 세입자 "신뢰 프로필" 분야에 경쟁자 부재 → 선점 기회.

**성장성**

◦ 제품 경쟁력: MVP 99% 완성으로 즉시 서비스 런칭 가능. 공인 인증(CODEF/NICE) 기반 위변조 불가능한 신뢰 데이터 축적 → 후발 주자 진입 장벽 형성.

◦ 영업 전략:
  - 온라인: 부동산 커뮤니티(40%), 콘텐츠 마케팅(30%), SNS(20%), PR(10%)
  - 바이럴 루프: 세입자 가입 → 프로필 → 집주인 공유 → 집주인 가입 → 세입자 확대
  - B2B: 경기도 동부권 부동산 중개업소 우선 공략 → 전국 확대
  - 경기도 연계: 청년 월세 지원사업, 전세피해지원센터 연계 홍보

◦ 미래 성장성:
  - Phase 1 (2026 하반기): 정식 출시, MAU 3,000명, 매출 500만원
  - Phase 2 (2027): B2B SaaS 출시, 100곳 제휴, Seed 투자 2~3억, 매출 1.5억원
  - Phase 3 (2028~): 공공 연계(LH/SH), 양방향 신뢰 플랫폼, Series A, 매출 8억원(BEP)
