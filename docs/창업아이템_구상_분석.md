## 아이템 1: AI 시니어 케어 플랫폼

### 개요

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  서비스명: 실버메이트 (SilverMate)                               │
│                                                                 │
│  한 줄 소개:                                                     │
│  "AI 기반 독거 노인 안전 모니터링 및 돌봄 매칭 플랫폼"           │
│                                                                 │
│  핵심 기능:                                                      │
│  1. AI 음성 기반 안부 확인 (매일 전화)                           │
│  2. 이상 징후 감지 및 보호자 알림                                │
│  3. 돌봄 서비스 매칭 (요양보호사, 자원봉사)                      │
│  4. 건강 데이터 수집 및 분석                                     │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 왜 유리한가?

| 요소 | 분석 | 점수 |
|------|------|------|
| **정책 부합** | 고령화 대응, 돌봄 서비스 = 정부 최우선 과제 | ⭐⭐⭐⭐⭐ |
| **사회 문제 해결** | 독거 노인 고독사 예방 = 명확한 임팩트 | ⭐⭐⭐⭐⭐ |
| **시장 성장** | 시니어 케어 시장 연 20% 성장 | ⭐⭐⭐⭐ |
| **기술 차별성** | AI 음성 인식 + 감정 분석 | ⭐⭐⭐⭐ |
| **B2G 확장** | 지자체/복지관 연계 용이 | ⭐⭐⭐⭐⭐ |

### 타겟 지원사업

```
✅ 예비창업패키지 (일반)
✅ 소셜벤처 육성사업 (복지 분야)
✅ AI 바우처 (AI 기술)
✅ 디지털 헬스케어 지원사업
✅ 지역 사회적경제 지원
```

### 예상 난이도

| 항목 | 난이도 | 이유 |
|------|--------|------|
| 개발 | 중 | AI 음성 = OpenAI/Clova API 활용 |
| 인허가 | 중 | 돌봄 서비스 = 신고 필요 |
| 경쟁 | 중 | 대기업 진출 (KT, SKT) |
