# TravelMate 사업계획서

> **여행 동반자 매칭 + 위치 기반 NFT 수집 플랫폼**

---

## 1. 사업 개요 (Executive Summary)

### 1.1 회사 정보
| 항목 | 내용 |
|------|------|
| 서비스명 | TravelMate |
| 사업 형태 | IT 플랫폼 서비스 (여행 소셜 + NFT) |
| 설립 예정 | 2026년 |
| 서비스 URL | https://various-belva-untab-1a59bee2.koyeb.app |
| GitHub | https://github.com/araeLaver/travle_mate |

### 1.2 비전 & 미션
- **비전**: 전 세계 여행자들이 신뢰할 수 있는 동반자를 찾고, 여행의 추억을 디지털 자산으로 영구 보존하는 글로벌 플랫폼
- **미션**: 혼자 여행하기 망설이는 사람들에게 안전하고 즐거운 동행 경험을 제공하고, 여행지 방문을 게이미피케이션으로 동기부여

### 1.3 핵심 가치 제안 (Value Proposition)
1. **안전한 동행 매칭**: 프로필 검증, 리뷰 시스템, 실시간 채팅으로 신뢰 구축
2. **여행의 게이미피케이션**: GPS 기반 NFT 수집으로 여행 동기 부여
3. **디지털 여행 기록**: 블록체인에 영구 저장되는 여행 인증서

## 2. 서비스 상세

### 2.1 핵심 기능

#### A. 여행 동반자 매칭
```
[사용자 프로필] → [여행 그룹 생성/참여] → [실시간 채팅] → [동행 확정]
```
- 여행 스타일, 관심사, 일정 기반 매칭 알고리즘
- 실시간 WebSocket 채팅 (이미지/위치 공유)
- 팔로우/팔로워 소셜 네트워크
- 동행 후기 및 평점 시스템

#### B. 위치 기반 NFT 수집
```
[GPS 위치 확인] → [특정 장소 도착] → [NFT 민팅] → [컬렉션 저장]
```
- Polygon 네트워크 기반 (가스비 저렴)
- 희귀도 등급: Common → Rare → Epic → Legendary
- 장소별 한정 NFT (예: 에펠탑 방문 인증 NFT)
- 리더보드 경쟁 시스템

#### C. 커뮤니티 기능
- 여행 후기 공유
- 장소 리뷰 및 평점
- 여행 팁 게시판
- 실시간 여행자 위치 공유 (옵션)

## 3. 시장 분석

### 3.1 타겟 시장 규모

#### TAM (Total Addressable Market)
- 글로벌 여행 시장: **$1.9조** (2025년 예상)
- 1인 여행자 비율: 약 25%
- **TAM**: ~$4,750억

#### SAM (Serviceable Addressable Market)
- 아시아 태평양 1인 여행 시장: ~$950억
- 디지털 플랫폼 이용 비율: 40%
- **SAM**: ~$380억

#### SOM (Serviceable Obtainable Market)
- 초기 타겟: 한국 + 일본 + 동남아
- 시장 점유 목표 (3년): 0.1%
- **SOM**: ~$3.8억 (약 5,000억원)

### 3.2 타겟 고객

#### Primary Target: MZ세대 1인 여행자 (20-35세)
- 혼자 여행은 원하지만 외로움/안전 우려
- SNS/디지털에 익숙
- 새로운 경험에 적극적
- NFT/블록체인에 관심

#### Secondary Target: 시니어 여행자 (50-65세)
- 은퇴 후 여행 욕구 증가
- 동반자 필요성 높음
- 프리미엄 서비스 지불 의향

### 3.3 시장 트렌드
1. **1인 여행 증가**: 코로나 이후 55% 증가 (Booking.com 조사)
2. **여행 + 디지털 자산**: NFT 여행 인증 시장 성장
3. **안전한 동행 수요**: 특히 여성 여행자 중심
4. **경험 공유 욕구**: 인스타그래머블 여행 트렌드

- **타겟**: 일본, 동남아 여행자
