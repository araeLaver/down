### 2. 문제 정의 (45초)

```
[슬라이드 내용]
제목: 해외여행, 이런 경험 있으시죠?

문제 1: 동행자와 말이 안 통해요
- 다국적 그룹 여행 시 소통 단절
- 번역 앱은 1:1만 지원

문제 2: 만나기로 한 장소에서 못 찾아요
- "하치코 동상 앞"인데 사람이 너무 많음
- 전화해도 설명이 어려움

문제 3: 낯선 도시에서 헤매요
- 지도 보느라 여행을 못 즐김

[말할 것]
"해외여행 중 외국인 친구와 만났는데 대화가 안 돼서
어색했던 경험, 다들 있으시죠?
기존 번역 앱은 1:1만 되고, 그룹 대화는 불가능합니다.
만남 장소에서 서로 못 찾아 헤맨 경험도 많으시죠."
```

### 3. 솔루션 (60초)

```
[슬라이드 내용]
제목: Fryndo AR Companion

기능 1: 실시간 그룹 번역
- 3명 이상 다국어 동시 번역
- 음성 → 자막 실시간 표시

기능 2: AR 동행자 찾기
- 카메라로 주변 스캔
- "3시 방향 15m" AR 표시

기능 3: AR 길안내
- 화면에 화살표 오버레이
- 지도 안 봐도 됨

[이미지]
- 앱 목업 또는 실제 화면 캡처

[말할 것]
"Fryndo AR Companion은 3가지로 이 문제를 해결합니다.
첫째, 그룹 번역. 한국인, 일본인, 미국인이 같이 있어도
각자 모국어로 실시간 자막이 뜹니다.
둘째, AR로 동행자 찾기. 카메라 켜면 '저기 15미터'라고 보여줍니다.
셋째, AR 길안내. 지도 안 보고 화면만 보면 화살표가 안내합니다."
```

### 4. 데모/시연 (60초)

```
[슬라이드 내용]
- 30초~1분 데모 영상 삽입
- 또는 실제 앱 라이브 시연

[데모 시나리오]
1. 앱 실행
2. 동행자 그룹 화면
3. 번역 기능 시연 (음성 → 자막)
4. AR 카메라 화면 (동행자 위치 표시)

[말할 것]
"실제로 어떻게 동작하는지 보여드리겠습니다.
(영상 재생 또는 라이브 시연)
이렇게 말하면 바로 번역되고,
카메라로 보면 동행자 위치가 AR로 표시됩니다."

[참고]
- MVP 없으면 목업 영상도 OK
- "현재 개발 중이며, 이렇게 구현될 예정입니다"
```

### 5. 시장 분석 (45초)

```
[슬라이드 내용]
제목: 시장 기회

TAM (전체 시장)
- 글로벌 여행자: 연간 15억 명

SAM (유효 시장)
- 한국 출국자 + 방한 외국인: 4,500만 명

SOM (목표 시장)
- 다국적 그룹 여행자: 100만 명
- 1년차 목표: 1,000명 (0.1%)

[그래프]
- 해외여행 회복 추이 (코로나 이후 급증)

[말할 것]
"해외여행 시장은 코로나 이후 폭발적으로 회복 중입니다.
한국인 출국자만 연간 2,800만 명.
이 중 다국적 그룹 여행을 하는 100만 명이 저희 타겟이고,
1년차에 0.1%인 1,000명 확보가 목표입니다."
```

## 발표 팁

### Do (해야 할 것)
- 문제 → 솔루션 흐름을 명확하게
- 데모/영상으로 시각적 임팩트
- 자신감 있게, 천천히 말하기
- 심사위원 눈 마주치기

### Don't (하지 말 것)
- 슬라이드 글자 빽빽하게
- 기술 용어 남발 (AR, API 등 최소화)
- 시간 초과 (7분 넘기면 감점)
- 과장된 시장 규모/매출 예측

### Q&A 예상 질문

| 질문 | 답변 방향 |
|------|-----------|
| AR 개발 경험 있나요? | "외주 협업 예정, 본인은 앱/백엔드 담당" |
| 경쟁 앱과 차이점? | "그룹 번역 + 동행 매칭 통합" |
| 1년 후 목표 현실적인가? | "기존 Fryndo 사용자 전환 + 타겟 마케팅" |
| 번역 정확도는? | "DeepL API 사용, 업계 최고 수준" |
