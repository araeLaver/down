## 검증된 한국 Pain Point

### 1. 소상공인 (570만 명)

```
문제:
• 디지털 전환 어려움 (키오스크, 예약, 마케팅)
• 배달앱 수수료 부담 (15~30%)
• 인건비 상승
• 고객 관리 수동

정부 지원:
• 스마트 기술 보급 지원
• 디지털 전환 바우처
• 소상공인 SaaS 구독 지원
```

### 2. 프리랜서/1인 기업 (230만 명)

```
문제:
• 일감 확보 어려움
• 계약/대금 관리 불안
• 세금/회계 처리 복잡
• 법적 보호 부재

시장:
• 월 80만 명 → 140만 명 (1.5년간 74% 증가)
• 개발자 평균 575만원 vs 일반 100만원 미만
```

### 3. 중소기업 HR/업무

```
문제:
• 채용 비용 높음
• 직원 교육 어려움
• 업무 자동화 필요
• 리포트/문서 작성 시간

성공 사례:
• 플렉스 (HR SaaS) - 대규모 투자 유치
• 채널톡 (고객 상담) - 3.7만 고객
```

## 아이템 1: AI 세금/회계 자동화 (프리랜서 특화)

### 개요

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  서비스명: 프리택스 (FreeTax) / 세금메이트                       │
│                                                                 │
│  한 줄 소개:                                                     │
│  "프리랜서 세금 신고를 AI가 자동으로"                            │
│                                                                 │
│  타겟:                                                           │
│  • 프리랜서 230만 명                                             │
│  • 1인 사업자                                                    │
│  • 부업러 (직장인 투잡)                                          │
│                                                                 │
│  핵심 Pain Point:                                                │
│  • 종합소득세 신고 = 매년 5월 스트레스                           │
│  • 세무사 비용 = 30~100만원                                      │
│  • 뭘 경비로 넣어야 하는지 모름                                  │
│  • 3.3% 원천징수 환급 모름                                       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 핵심 기능

```
1. 카카오페이/토스 내역 자동 연동
2. 경비 자동 분류 (AI)
3. 절세 항목 자동 추천
4. 종합소득세 신고서 자동 생성
5. 홈택스 연동 신고
6. 부가세 신고 자동화
```

### 왜 현실적인가?

| 항목 | 분석 |
|------|------|
| **시장 크기** | 프리랜서 230만 + 부업러 수백만 |
| **지불 의사** | 세무사비 30만원 vs 월 9,900원 |
| **경쟁** | 삼쩜삼 있지만 기능 제한적 |
| **기술** | GPT + 홈택스 API (공개) |
| **정부 연계** | 1인 창조기업 지원사업 |

### 수익 모델

| 플랜 | 가격 | 기능 |
|------|------|------|
| Free | 무료 | 예상 세금 계산 |
| Basic | ₩9,900/월 | 자동 분류 + 신고서 |
| Pro | ₩19,900/월 | 절세 컨설팅 + 부가세 |
| 신고 대행 | ₩49,000/회 | 직접 신고 |

### 투자 가능성

```
[검증된 시장]
• 삼쩜삼: 누적 사용자 1,900만 명

[차별화]
• 삼쩜삼: 환급만 집중 → 전체 세금 관리

[투자 시나리오]
```

### 왜 1순위인가?

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  🏆 추천 1순위: AI 세금 자동화 (프리랜서 특화)                   │
│                                                                 │
│  [검증된 시장]                                                   │
│  • 삼쩜삼: 1,900만 사용자 = 시장 존재 증명                       │
│  • 자비스: 시리즈B 투자 유치 = 투자자 관심                       │
│                                                                 │
│  [명확한 Pain Point]                                             │
│  • 종합소득세 = 프리랜서 최대 스트레스                           │
│  • 세무사 비용 30만원 vs SaaS 10만원/년                          │
│                                                                 │
│  [큰 시장]                                                       │
│  • 프리랜서 230만 명 (급증 중)                                   │
│  • 부업러 (직장인 N잡) 수백만 명                                 │
│  • 1인 사업자 100만+                                             │
│                                                                 │
│  [정부 지원 연계]                                                │
│  • 1인 창조기업 지원사업                                         │
│  • 핀테크 지원사업                                               │
│  • 예비창업패키지                                                │
│                                                                 │
│  [차별화 가능]                                                   │
│  • 삼쩜삼: 환급만 → 전체 세금 관리                               │
│  • 자비스: 기업용 → 프리랜서 특화                                │
│  • AI 절세 컨설팅 = 추가 가치                                    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

```
[MVP 개발] (6~8주)

• 스타트업 개발자 커뮤니티

[수익화]
• 5월 종소세 시즌 집중 마케팅
• 환급 성공 시 수수료 모델 병행
```
