# 세컨드 아이템: 시니어 생활동행 서비스

> **서비스명 (가칭)**: 효도메이트 / 동행플러스 / 어르신동행
> **핵심**: 요양등급 못 받는 경증 노인(899만명)을 위한 생활 동행 서비스

---

## 1. 문제 정의: 왜 이 서비스가 필요한가?

### 대한민국 노인 돌봄의 현실

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  65세 이상 노인: 1,012만명                                      │
│                                                                 │
│  ├── 장기요양보험 수급자: 113만명 (11%)                         │
│  │   → 케어링, 케어닥 등이 서비스 중                            │
│  │                                                              │
│  └── 공적 돌봄 못 받는 노인: 899만명 (89%)  ← 틈새 시장         │
│      → 요양등급 못 받음                                         │
│      → 간병까지는 필요 없음                                     │
│      → 하지만 생활 지원이 필요함                                │
│                                                                 │
│  "간병사는 필요 없는데, 혼자 병원 가기 어려워요"                │
│  "장 보러 나가기 힘들어요"                                      │
│  "자녀는 바빠서 자주 못 와요"                                   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2025 노인돌봄공백지수

| 지표 | 수치 | 의미 |
|------|------|------|
| 종합 돌봄공백지수 | 197 | 2008년 대비 2배 |
| 장기요양 공백지수 | 189 | 89%가 공백 상태 |
| 시니어하우징 공백지수 | 205 | 97%가 입소 불가 |
| 간병비 물가지수 | 210 | 소득 초과 |

### 경증 노인의 실제 니즈

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  "간병사까지는 필요 없는데..."                                  │
│                                                                 │
│  [병원 동행]                                                    │
│  • 혼자 병원 가기 어려움 (이동, 접수, 대기)                     │
│  • 의사 설명 이해 어려움                                        │
│  • 약 타기, 수납 복잡함                                         │
│                                                                 │
│  [외출 동행]                                                    │
│  • 마트 장보기 (무거운 짐)                                      │
│  • 은행, 관공서 (복잡한 절차)                                   │
│  • 경조사 참석                                                  │
│                                                                 │
│  [말벗/안부]                                                    │
│  • 외로움 (독거노인)                                            │
│  • 자녀에게 안부 전달                                           │
│  • 정기적인 방문/대화                                           │
│                                                                 │
│  [생활 지원]                                                    │
│  • 간단한 집안일                                                │
│  • 스마트폰 사용법                                              │
│  • 각종 신청/서류 대행                                          │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 2. 시장 분석

### 타겟 시장 규모

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  TAM (전체 시장)                                                │
│  • 65세 이상 노인: 1,012만명                                    │
│  • 시니어 돌봄 시장: 2030년 168조원                             │
│                                                                 │
│  SAM (유효 시장)                                                │
│  • 요양등급 없는 경증 노인: 899만명                             │
│  • 독거노인: 약 200만명                                         │
│  • 노인 1인가구 + 노인 부부가구: 500만+ 가구                    │
│                                                                 │
│  SOM (목표 시장) - 초기 1년                                     │
│  • 수도권 거주 + 자녀 있는 노인 가구                            │
│  • 월 1회 이상 병원 방문 필요                                   │
│  • 추정: 50만 가구                                              │
│  • 점유율 1%: 5,000가구                                         │
│  • 월 ₩50,000 × 5,000 = ₩2.5억/월                               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 경쟁사 분석

| 경쟁사 | 타겟 | 서비스 | 투자유치 | 차별점 |
|--------|------|--------|---------|--------|
| **케어링** | 중증 (요양등급) | 방문요양, 주간보호 | 750억원 | 장기요양보험 |
| **케어닥** | 중증 (요양등급) | 간병매칭, 시니어하우징 | 315억원 | 간병인 매칭 |
| **케어네이션** | 중증~경증 | 간병, 동행, 가사 | 305억원 | 종합 돌봄 |
| **청연** | 일반 가구 | 가사 서비스 | - | 가사 중심 |
| **서울시 동행서비스** | 취약계층 | 병원 동행 | 공공 | 저소득 한정 |

### 틈새 시장 (경쟁 공백)

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  기존 서비스의 한계:                                            │
│                                                                 │
│  • 케어링/케어닥: 요양등급 있어야 함 (장기요양보험)             │
│    → 경증 노인은 이용 불가                                      │
│                                                                 │
│  • 간병인: 월 432만원 (너무 비쌈)                               │
│    → 가벼운 동행만 필요한데 과잉 서비스                         │
│                                                                 │
│  • 서울시 병원동행: 저소득층 한정                               │
│    → 일반 가구는 이용 불가                                      │
│    → 월 10회, 연 200시간 제한                                   │
│                                                                 │
│  틈새:                                                          │
│  "요양등급 없는 경증 노인 + 일반 가구"를 위한                   │
│  "합리적 가격의 생활 동행 서비스"                               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 5. 실행 전략

### MVP (최소 기능 제품)

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  Phase 1: MVP (3개월)                                           │
│                                                                 │
│  [기능]                                                         │
│  • 카카오톡 채널 기반 예약 (앱 개발 전)                         │
│  • 수동 매칭 (운영자가 직접)                                    │
│  • 병원 동행 서비스만 제공                                      │
│  • 서울 강남/서초/송파 지역 한정                                │
│                                                                 │
│  [목표]                                                         │
│  • 월 50건 서비스 완료                                          │
│  • 동행메이트 20명 확보                                         │
│  • 서비스 만족도 90% 이상                                       │
│  • PMF (Product-Market Fit) 검증                                │
│                                                                 │
│  [비용]                                                         │
│  • 개발비: ₩0 (카카오톡 채널)                                   │
│  • 마케팅: ₩500,000/월                                          │
│  • 운영비: ₩1,000,000/월                                        │
│  • 총: ₩4,500,000 (3개월)                                       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 확장 로드맵

```
Phase 1 (0~3개월): MVP
• 카카오톡 기반 수동 운영
• 서울 3개구 한정
• 병원 동행만

Phase 2 (4~6개월): 앱 출시
• 보호자용 앱 개발
• 동행메이트용 앱 개발
• 자동 매칭 시스템
• 서울 전역 확장

Phase 3 (7~12개월): 서비스 확장
• 외출 동행, 말벗 서비스 추가
• 정기 구독 모델 도입
• 수도권 확장 (경기/인천)

Phase 4 (13~24개월): 전국 확장
• 광역시 확장
• B2B (기업 복지, 보험사 연계)
• 지자체 연계 사업
```
