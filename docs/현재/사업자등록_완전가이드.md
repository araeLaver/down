## 사전 준비 (등록 전에 미리 해둘 것)

사업자등록은 하루면 끝나지만, **사전 준비 안 하면 뒤에서 시간 잡아먹힘**.
아래 항목은 등록 전에 미리 결정/준비해두기.

### 결정해야 할 것

| 항목 | 선택지 | 추천 | 이유 |
|------|--------|:----:|------|
| **사업자 유형** | 개인 vs 법인 | 예창패 선정 → 법인 / 탈락 → 개인 | 법인은 정부사업 필수, 개인은 비용 0원 |
| **과세 유형** | 간이 vs 일반 | **간이과세자** | 연매출 8천만 이하, 부가세 감면. B2B 세금계산서 필요하면 일반 |
| **업종코드** | 아래 참조 | 복수 등록 | 메인+부업종 같이 등록 |
| **사업장 주소** | 자택 vs 공유오피스 | **자택** (초기) | 비용 0원. 혁신공간 입주되면 이전 |
| **상호명** | 다운랩 / 프린도 / 렌트미 | **다운랩** | 포괄적 상호. 서비스명은 브랜드로 사용 |

### 업종코드 (미리 정해두기)

| 업종 | 코드 | 용도 |
|------|------|------|
| **응용 소프트웨어 개발 및 공급업** | 722000 | 메인 (Fryndo, RentMe) |
| **포털 및 기타 인터넷 정보매개 서비스업** | 631210 | 플랫폼 서비스 |
| **컴퓨터 프로그래밍 서비스업** | 620100 | 외주/프리랜서 수입 |

→ **메인 1개 + 부업종 1~2개** 등록 가능. 세무서에서 바로 추가됨.

### 준비물 체크리스트

| # | 준비물 | 상태 | 비고 |
|:-:|--------|:----:|------|
| 1 | 신분증 (주민등록증/운전면허증) | ⬜ | 원본 필수 |
| 2 | 임대차계약서 (사업장 주소 증빙) | ⬜ | 자택이면 본인 명의 계약서 or 무상임대확인서 |
| 3 | 업종코드 메모 | ⬜ | 위 표에서 선택 |
| 4 | 도장 | ⬜ | 없어도 서명으로 대체 가능 |
| 5 | 핸드폰 | ⬜ | 본인인증용 |

## DAY 1: 사업자등록

### 방법 1: 세무서 방문 (추천)

```
관할 세무서 확인: 홈택스 → "관할세무서 찾기" 또는 국세청 126
영업시간: 평일 09:00~18:00 (점심 12:00~13:00)
소요시간: 30분~1시간
```

**방문 순서:**

1. **민원실 방문** → 사업자등록 신청서 작성 (비치된 양식)
2. **신분증 + 임대차계약서** 제출
3. 업종코드 말하기 → 담당자가 등록
4. **사업자등록증 즉시 발급** (빳빳한 종이)
5. **보안카드 수령** (전자세금계산서 발행용)

> 💡 당일 발급이 원칙. 간혹 현장 확인이 필요하면 2~3일 소요.

### 방법 2: 홈택스 온라인 (대안)

```
홈택스 → 신청/제출 → 사업자등록 신청
소요시간: 입력 10분, 처리 1~3영업일
```

- 임대차계약서 스캔본 첨부
- 처리 완료 후 등록증 PDF 다운로드
- **빳빳한 종이 원본은 세무서 방문해서 수령해야 함**

### 방법 3: 정부24 (대안)

```
정부24 → 사업자등록 신청
동일 프로세스
```

## DAY 1~2: 사업자등록과 동시에 할 것

사업자등록증 받자마자 **같은 날** 처리할 수 있는 것들.

### 은행: 사업용 계좌 개설

```
준비물: 신분증 + 사업자등록증 원본
추천 은행: 가까운 곳 아무데나 (국민/신한/하나/우리)
소요시간: 30분
```

- [ ] **사업용 계좌 개설** (입출금 통장)
- [ ] **인터넷뱅킹 신청** (OTP 발급)
- [ ] 공동인증서 발급 여부 확인
  - 전자세금계산서 발행용 공동인증서: **4,400원**
  - 소프트웨어업은 세금계산서 빈도 낮음 → 필요 시 나중에 해도 됨
  - 홈택스 전자세금계산서는 **보안카드**로도 발행 가능
- [ ] **체크카드 발급** (사업용 카드로 사용)

> 💡 사업용 계좌 ≠ 개인 계좌. **반드시 분리**해야 경비처리 깔끔.

### 홈택스: 필수 등록 (은행 다녀와서 바로)

```
홈택스 (hometax.go.kr) 로그인
→ 공동인증서 or 간편인증(카카오/네이버)
```

**① 사업용 계좌 등록**
- [ ] 홈택스 → 조회/발급 → 사업용 계좌 개설 신고
- 용도 선택: 매출/매입/임대료/급여 (전부 체크)
- 계좌번호 입력 → 등록

**② 사업용 신용카드 등록**
- [ ] 홈택스 → 조회/발급 → 사업용 신용카드 등록
- **기존 개인 카드도 등록 가능** (사업 경비 자동 집계)
- 여러 장 등록 가능
- 등록하면 카드 사용 내역이 홈택스에 자동으로 잡힘 → **부가세 신고 시 매입세액 공제**

**③ 현금영수증 발급사업자 가맹**
- [ ] 홈택스 → 조회/발급 → 현금영수증 → 현금영수증 발급사업자 가맹
- 연매출 2,400만원 이상이면 의무, 이하여도 미리 해두기
- 가맹하면 현금 결제 시 현금영수증 발행 가능

**④ 공동인증서 등록** (있으면)
- [ ] 홈택스 → 인증서 등록
- 보안카드는 세무서에서 받은 것 등록

### 세무기장 (선택, 추천)

- [ ] 세무사 계약 or 온라인 서비스 가입
  - **삼쩜삼 비즈** — 월 5.5만원, 기장+신고 자동화
  - **자비스** — 월 8.8만원, 전담 세무사
  - **동네 세무사** — 월 8~15만원, 대면 상담
- 간이과세자 + 초기 매출 적으면 직접 해도 되지만, 실수 방지용으로 추천
- **계약 시점**: 사업자등록 직후 or 첫 매출 발생 시

## DAY 2~3: 온라인 서비스 사업자 전환

사업자등록번호가 나오면 **기존에 쓰던 서비스들을 사업자로 전환**.
비용처리 가능 + 일부 서비스 10% 할인.

### 결제 수단 (비용처리용)

| 서비스 | 할 일 | 효과 |
|--------|-------|------|
| **네이버페이** | 설정 → 사업자등록번호 입력 | 사업 경비 자동 분류 |
| **카카오페이** | 설정 → 사업자등록번호 입력 | 사업 경비 자동 분류 |
| **쿠팡 로켓배송** | 비즈 계정 전환 | 세금계산서 발행 가능 |
| **토스** | 사업자 인증 | 사업용 머니 관리 |

### AI/개발 서비스 (10% 부가세 환급)

| 서비스 | 할 일 | 효과 |
|--------|-------|------|
| **ChatGPT Plus/API** | Settings → Billing → 사업자번호 입력 | **10% 할인** (부가세 역발행) |
| **Claude API** | Console → Billing → Tax ID 입력 | **10% 할인** |
| **Google (Gemini/GCP)** | 결제 프로필 → 사업자 정보 | 세금계산서 발행 |
| **Meta (Facebook/Instagram 광고)** | 비즈니스 관리자 → 사업자 정보 | 세금계산서 + 광고비 경비처리 |
| **AWS** | 계정 설정 → Tax Settings → KR 사업자번호 | 세금계산서 발행 |
| **Vercel** | Team → Billing → Tax ID | 세금계산서 |
| **GitHub (Pro/Team)** | Billing → Tax ID | 세금계산서 |

> 💡 해외 서비스(OpenAI, AWS 등)는 **역발행 세금계산서**로 처리됨.
> 부가세 신고 시 매입세액공제 가능 (일반과세자).

### 기타 경비처리

| 항목 | 할 일 | 비고 |
|------|-------|------|
| **핸드폰 요금** | 자동이체 → 사업용카드 결제로 변경, 또는 통신사에 세금계산서 발행 신청 | 매월 경비처리 |
| **인터넷 요금** | 동일 | 사업장 주소 일치 시 |
| **전기/가스** | 사업장 자택이면 일부 경비 인정 | 비율 안분 |
| **교통비** | 사업용카드로 결제 | 업무 관련 이동 |
| **도서/교육** | 사업용카드로 결제 | 업무 관련 도서, 온라인 강의 |
| **커피/식사** | 사업용카드로 결제 | 1인 식대 한도 있음 (접대비 vs 복리후생비) |

## DAY 3~7: 추가 신고/등록

### 통신판매업 신고 (필수 — 유료 서비스 판매 전)

```
온라인: 정부24 → "통신판매업 신고"
오프라인: 관할 구청 방문
소요시간: 온라인 3~5영업일 / 방문 즉시~2일
수수료: 무료
```

**준비물:**
- [ ] 사업자등록증 사본
- [ ] 신분증
- [ ] 구매안전서비스 이용확인증 (PG사 가입 전이면 나중에 보완 가능)

**결과물:** 통신판매업 신고번호 → **PG사 계약에 필수**

### PG사 가입 (토스페이먼츠)

```
토스페이먼츠 개발자센터: developers.tosspayments.com
소요시간: 신청 10분, 심사 3~5영업일
```

**제출 서류:**
- [ ] 사업자등록증
- [ ] 통신판매업 신고증
- [ ] 대표자 신분증
- [ ] 사업용 통장 사본

**순서:**
1. 개발자센터 가입 → 테스트 키 즉시 발급 (사업자 없이도 가능)
2. 사업자 인증 → 상용 키 발급 신청
3. 심사 완료 → 실결제 가능

> 💡 테스트 키로 개발은 **지금 당장** 가능.
> 상용 전환만 사업자등록 후.

### 상표 출원 (선택, 추천)

```
키프리스(kipris.or.kr)에서 선행상표 검색 → 특허로(patent.go.kr)에서 출원
소요시간: 출원 30분, 심사 10~14개월
비용: 온라인 출원 1건당 약 62,000원 (1분류 기준)
```

- [ ] "프린도" / "Fryndo" 선행 검색
- [ ] "렌트미" / "RentMe" 선행 검색
- [ ] 해당 분류(42류: 소프트웨어) 출원
- 직접 출원 가능, 대행 시 건당 20~30만원 추가

## 주의사항

### 간이과세자 vs 일반과세자

| | 간이과세자 | 일반과세자 |
|:--|:----------|:----------|
| 기준 | 연매출 8,000만원 이하 | 그 외 |
| 부가세 | 매출의 1.5~4% | 매출의 10% - 매입세액 |
| 세금계산서 발행 | 연매출 4,800만 이상만 가능 | 자유롭게 가능 |
| 부가세 신고 | 연 1회 (1월) | 연 2회 (1월, 7월) |
