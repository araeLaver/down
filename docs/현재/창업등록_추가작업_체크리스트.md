> **작성일**: 2026-03-11
> **현재 단계**: 예비창업자 (사업자 미등록)
> **핵심 원칙**: 예창패 결과(4~5월)까지 사업자등록 보류, 서류 준비만 선행

---

## ⚠️ 사업자등록 타이밍 주의사항

**지금 사업자등록을 하면 안 되는 이유:**

| 프로그램 | 자격 요건 | 상태 |
|----------|-----------|------|
| 예비창업패키지 | 사업자등록 **전** 예비창업자 | 심사 진행중 |
| 관광벤처 (예비부문) | 사업자등록 **전** 예비창업자 | 제출 완료 |
| 경기창업혁신공간 | 공고일 기준 사업자등록 **미등록** | 제출 완료 |
| G스타 오디션 | 예비창업자 가능 | 제출 완료 |

→ 사업자등록 시 위 프로그램 **"예비창업자" 자격 상실 위험**

### 올바른 타이밍

```
예창패 선정 (5월) → 협약 → 법인설립 + 사업자등록 (M1)
관광벤처 선정 시  → 협약 후 사업자등록
전부 탈락 시     → 그때 개인사업자 등록 후 부트스트랩
```

### 예외: 디딤돌 2차 (4/9 마감)

- 개인사업자 등록이 신청 자격 → 예창패 자격과 트레이드오프
- 예창패가 더 중요하면 디딤돌은 포기하고 예비창업자 자격 유지 권장

## 1단계: 지금 바로 할 것 (우선순위순)

### 1순위: 겸업동의서 (이번주)

사업자등록 전이라도 **모든 후속 작업의 전제조건**. 이게 없으면 아무것도 못 함.

- [ ] G-Media 대표에게 겸업동의서 서명 요청
- [ ] 서명 완료본 스캔/보관

### 2순위: 입주해 MVP 완성 (3월 내)

선정 여부와 관계없이 **제품이 있어야 사업**. 외부 유료 API 의존 없이 자체 스크래핑으로 구현.

- [ ] 임대인 레퍼런스 페이지 (P0) 완성
- [ ] 재직/소득 증빙 — 자체 스크래핑 모듈 개발 (홈택스, 건강보험 등)
- [ ] 신용정보 대안 — 사용자 직접 업로드 방식 (CB 스크래핑 or PDF 파싱)
- [ ] 전체 플로우 테스트 (회원가입 → 프로필 생성 → 신뢰점수 산출)

### 3순위: Fryndo 수익화 준비 (3~4월)

MVP는 100%인데 **돈 버는 구조가 아직 없음**. 선정 전에 유료 전환 테스트해야 심사에서도 유리.

- [ ] 유료 플랜 베타 오픈 (제주 타겟)
- [ ] 결제 플로우 구현 (토스페이먼츠 테스트모드 — 사업자등록 전 가능)
- [ ] 실사용자 피드백 수집 (최소 5명)

### 4순위: 발표심사 대비 (4월)

예창패·관광벤처 선정되면 **발표심사**가 온다. 미리 준비 안 하면 급함.

- [ ] Fryndo 데모 영상 제작 (1~2분, 핵심 플로우만)
- [ ] 입주해 데모 영상 제작
- [ ] 발표 스크립트 초안 (3분 / 5분 버전)
- [ ] 핵심 지표 정리 (MAU, 전환율, 사용자 피드백)

### 5순위: 서류 사전 준비 (틈틈이)

급하진 않지만 선정 후 바로 움직이려면 미리 해둬야 함.

- [ ] 법인 정관 최종 검토 (`legal/정관_주식회사프린도.md`)
- [ ] 업종코드 선정 (정보통신업 / 소프트웨어 개발)
- [ ] 법인설립 대행 서비스 비교 (헬프미, 법인설립119 등)
- [ ] 사업장 주소 확보 방안 확인 (창업혁신공간 입주 or 공유오피스)
- [ ] 입주해 이용약관 / 개인정보처리방침 작성
- [ ] 도메인 확보 (fryndo.com, fryndo.kr, ipjuhae.com)
- [ ] 상표 검색 (출원은 사업자등록 후)

### 6순위: 대시보드 최신화

- [ ] 사업_대시보드.md 3월 현황 반영 (마지막 업데이트 2/20)
- [ ] 지원사업 제출 결과 반영 (관광벤처, G스타, 혁신공간)
