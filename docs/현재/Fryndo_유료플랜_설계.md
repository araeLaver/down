```

타겟: 25-34세 여성, 여행 지출 의향 높음
→ 월 3,900원은 심리적 저항 최소
→ 연간 결제 시 23% 할인으로 장기 유도
```

## 2. 전환 유도 전략

### 2-1. 무료 → Plus 전환 트리거

| 트리거 | 시점 | 메시지 |
|--------|------|--------|
| **매칭 3회 소진** | 매칭 시도 시 | "이번 달 무료 매칭을 모두 사용했어요. Plus로 무제한 매칭!" |
| **Rare NFT 발견** | 여행지 도착 시 | "🎉 Rare NFT를 발견했어요! Plus 회원만 수집할 수 있어요" |
| **프로필 조회수** | 프로필 페이지 | "이번 주 3명이 프로필을 봤어요. 부스트로 노출을 10배 높이세요" |
| **채팅 제한** | 채팅 시도 시 | "무료 회원은 매칭된 그룹 채팅만 가능해요" |
| **7일 무료 체험** | 가입 3일 후 | "Plus 7일 무료 체험해보세요. 자동 결제 없이 체험 후 결정!" |

### 2-2. 전환 퍼널

```
가입 → 프로필 작성 → 첫 매칭 (무료) → 성공 경험
  → 2회차 매칭 (무료) → 관계 형성
  → 3회차 매칭 (무료) → 습관화
  → 4회차 시도 → "매칭 소진" 알림 → Plus 전환 유도
                    ↓
              7일 무료 체험 → 체험 종료 → 유료 전환 or 무료 복귀
```

### 2-3. 리텐션 전략

| 전략 | 설명 |
|------|------|
| **NFT 컬렉션** | 수집 욕구 자극, 월간 한정 NFT |
| **커뮤니티** | 여행 후기, 동행 스토리 공유 |
| **여행 캘린더** | 다가오는 여행에 대한 동행 추천 푸시 |
| **시즌 이벤트** | 제주 벚꽃, 부산 불꽃축제 등 시즌별 매칭 이벤트 |
| **레벨 시스템** | 매칭 횟수/후기에 따른 신뢰 레벨 (브론즈→실버→골드) |

## 3. 결제 플로우

### 3-1. 결제 수단

| 수단 | 우선순위 | 비고 |
|------|:--------:|------|
| 토스페이먼츠 (카드/간편결제) | 1순위 | 테스트모드 사업자등록 전 가능 |
| 인앱결제 (앱스토어/구글플레이) | 2순위 | 앱 출시 후 |
| 계좌이체 | 3순위 | 수동 처리, 초기에만 |

### 3-2. 결제 화면 플로우

```
[요금제 선택 페이지]
  │
  ├─ Free 선택 → 바로 서비스 이용
  │
  ├─ Plus 선택
  │   ├─ 월간 / 연간 토글
  │   ├─ "7일 무료 체험" 버튼 (첫 가입자)
  │   └─ "3,900원 결제하기" 버튼
  │       → 토스페이먼츠 결제 위젯
  │       → 결제 완료 → 구독 활성화 → 마이페이지
  │
  └─ Pro 선택
      ├─ 월간 / 연간 토글
      └─ "9,900원 결제하기" 버튼
          → 동일 플로우
```

### 3-3. 구독 관리

```
[마이페이지 > 구독 관리]
  │
  ├─ 현재 플랜: Plus (월간)
  ├─ 다음 결제일: 2026-04-11
  ├─ 결제 수단: 카카오페이 ****1234
  │
  ├─ [플랜 변경] → 업그레이드/다운그레이드
  ├─ [결제 수단 변경] → 카드/간편결제 변경
  └─ [구독 해지] → 확인 팝업 → 해지 사유 선택 → 해지 완료
                    (남은 기간 동안 서비스 계속 이용 가능)
```

### 3-4. 환불 정책

| 구분 | 정책 |
|------|------|
| 구매 7일 이내 | 전액 환불 (미이용 시) |
| 구매 7일 이내 (이용) | 이용일수 비례 공제 후 환불 |
| 구매 7일 이후 | 환불 불가, 구독 해지만 가능 |
| 연간 구독 | 미사용 월 비례 환불 (월 정가 기준 공제) |
