## 2. 확인 사항

본인은 아래 사항을 확인하고 준수할 것을 서약합니다.

1. 겸업 활동은 **근무시간 외(퇴근 후, 휴일)**에만 수행하며, 소속 회사의 업무 수행에 지장을 주지 않겠습니다.

2. 소속 회사의 **영업비밀, 기밀정보, 고객정보** 등을 겸업 활동에 이용하지 않겠습니다.

3. 소속 회사의 **거래처, 고객, 파트너사**를 대상으로 겸업 활동의 영업 행위를 하지 않겠습니다.

4. 소속 회사의 **자산, 장비, 시스템** 등을 겸업 활동에 사용하지 않겠습니다.

5. 겸업 내용에 **중대한 변경**이 있을 경우 사전에 소속 회사에 통보하겠습니다.

6. 소속 회사가 겸업으로 인한 업무 지장이 있다고 판단하여 **겸업 중단을 요청**할 경우 이에 따르겠습니다.
