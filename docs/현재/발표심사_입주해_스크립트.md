## 3분 스크립트

### [슬라이드 1] 표지 (10초)

안녕하세요. AI 기반 세입자 신뢰 프로필 플랫폼 입주해, 대표 김다운입니다.

### [슬라이드 2] 문제 (30초)

전세사기 피해자 3만 명, 피해액 2조 5천억 원.
이 사건 이후 한국 임대차 시장의 신뢰는 완전히 무너졌습니다.

그런데 문제는 **집주인만 나쁜 게 아닙니다.**
집주인 입장에서도 "이 세입자가 월세를 잘 낼 사람인지" 알 방법이 없습니다.

현재 세입자가 자신의 신뢰를 증명할 수 있는 플랫폼은 **제로**입니다.
직방, 다방, 네이버부동산 — 전부 "매물" 검색이지, "사람"은 없습니다.

### [슬라이드 3] 솔루션 (40초)

입주해는 **세입자의 링크드인**입니다.

**첫째, 신뢰점수(Trust Score).**
0점부터 120점까지. 프로필 완성도, 재직 증명, 소득 수준, 신용 등급, 이전 집주인 레퍼런스.
이 5가지를 종합해서 세입자의 신뢰를 숫자로 만듭니다.

**둘째, 공인 인증.**
국민연금, 건강보험, 본인인증을 자체 스크래핑으로 자동 검증합니다.
서류 제출 없이, 원클릭으로 재직과 소득을 증명합니다.

**셋째, 시맨틱 매칭.**
OpenSearch 벡터 검색으로 세입자와 매물을 자동 매칭합니다.
"역삼역 10분, 월세 80만 이하, 반려동물 가능" — 자연어로 검색하면 끝입니다.

### [슬라이드 4] 데모 (30초)

실제 동작하는 앱입니다.
회원가입 → 본인인증 → 재직/소득 자동 검증 → 이전 집주인 레퍼런스 요청 →
신뢰점수 87점 산출 → 프로필 공개.

집주인은 이 점수를 보고 세입자를 선택합니다.
**MVP가 이미 완성되어 있습니다.** 전부 직접 개발했습니다.

### [슬라이드 5] 시장 (20초)

국내 임대차 시장 연 250만 건, 540조 원 규모입니다.
수도권 디지털 활용 세입자 90만 건이 저희 목표 시장이고,
초기 점유율 0.5%, 연 4,500건이 SOM입니다.

### [슬라이드 6] 비즈니스 모델 (20초)

세입자 프리미엄 구독 월 4,900원.
중개업소 대상 B2B SaaS 월 4만 9천~19만 9천 원.
장기적으로 LH, SH 같은 공공임대 기관에 B2G 데이터 솔루션.

LTV:CAC 비율 5.9:1. 건전한 유닛 이코노믹스입니다.

### [슬라이드 7] 경쟁우위 (20초)

직방, 다방, 네이버부동산, 안심전세앱.
이 중 세입자 프로필을 제공하는 서비스는 없습니다.
공인 인증, 집주인 레퍼런스, 시맨틱 검색, 자동 매칭 — 전부 입주해만 가능합니다.

프로필이 쌓일수록 매칭 정확도가 올라가는 **데이터 네트워크 효과**가 핵심 해자입니다.

### [슬라이드 8] 팀 (15초)

10년차 풀스택 개발자입니다.
IBK에서 다중인증, 에너지마켓플레이스에서 매칭 알고리즘을 개발했습니다.
입주해 MVP를 외주 없이 100% 직접 개발했습니다.

### [슬라이드 9] 마일스톤 (15초)

8개월 목표.
M2 베타 출시, M4 가입자 1,000명, M6 가입자 5,000명.
B2B 중개업소 10곳 계약, 월매출 목표 50만원에서 시작합니다.

### [슬라이드 10] 클로징 (10초)

전세사기 이후, 무너진 신뢰를 다시 세우는 플랫폼.
**세입자에게 링크드인을, 집주인에게 안심을.**
입주해가 만들겠습니다. 감사합니다.
