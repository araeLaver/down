## 3. 단계별 상세 전략

### 3-1. TIPS (가장 현실적인 첫 투자, 2026 Q3~Q4)

TIPS는 정부 R&D 자금 + 민간 투자가 결합된 구조로, **대표 인건비 해결 + 투자 이력 확보**를 동시에 달성하는 최적 경로.

#### TIPS 구조

```
┌─────────────────────────────────────────────────────────────────┐
│  TIPS 선정 시 자금 구조                                          │
│                                                                 │
│  정부 R&D: 최대 3억 (2년)                                       │
│    ├─ 인건비 (대표 포함): 최대 60%                               │
│    ├─ 연구장비/재료비: 최대 30%                                  │
│    └─ 위탁연구/기타: 나머지                                     │
│                                                                 │
│  운영사 투자: 1~2억 (지분 투자)                                  │
│    ├─ 보통주 or SAFE                                             │
│    └─ 밸류: 10~15억 (Pre-money)                                  │
│                                                                 │
│  추가 지원: 엔젤 매칭펀드, 해외진출, 사업화 등                    │
│                                                                 │
│  합계: 최대 5억+ 확보 가능                                       │
│  핵심: 대표 인건비 → R&D비 항목으로 공식 수령 가능                │
└─────────────────────────────────────────────────────────────────┘
```

### 3-2. 엔젤 투자 (TIPS 병행 or 대안, 2026 Q3~)

#### 타겟 엔젤 투자자 유형

| 유형 | 투자금 | 접근법 |
|------|--------|--------|
| **부동산 업계 종사자** | 500~3,000만 | 임대 시장 pain point 공감 → 전략적 엔젤 |
| **IT 업계 선배 창업자** | 1,000~5,000만 | 개발자 출신 1인 창업 공감 |
| **엔젤투자 매칭 플랫폼** | 500~2,000만 | 와디즈, 오픈트레이드, 비긴메이트 |
| **지인 네트워크** | 100~500만 | SAFE 계약, 최소 규모로 시작 |

#### 엔젤 라운드 설계

```
목표 금액: 5,000만 ~ 1억
밸류에이션: Pre-money 5~8억 (SAFE or 전환사채)
투자자 수: 3~5인
용도: 마케팅 (60%) + 서버/인프라 (20%) + 법률/지재권 (20%)
타이밍: 법인 설립 + 첫 유저 확보 후
```

#### SAFE (Simple Agreement for Future Equity) 조건

```
┌─────────────────────────────────────────────────────────────────┐
│  SAFE 핵심 조건 (투자자 친화적 + 창업자 보호)                     │
│                                                                 │
│  밸류에이션 캡: 8억원                                            │
│  할인율: 20%                                                    │
│  전환 트리거: 다음 라운드 1억원 이상 투자 유치 시                  │
│  Pro-rata: 있음 (다음 라운드 참여 권리)                          │
│                                                                 │
│  예시) 1,000만원 투자 시:                                        │
│  → 다음 라운드 밸류 15억이면 → 캡 8억 적용 → 지분 1.25%          │
│  → 다음 라운드 밸류 6억이면 → 할인 20% → 밸류 4.8억 → 지분 2.08% │
└─────────────────────────────────────────────────────────────────┘
```

→ 합계: 5~6억 (현재) → 8~10억 (정부사업 선정 + 유저)

## 5. 투자자에게 어필할 핵심 포인트

### 5-1. Why 입주해? (시장 기회)

```
1. 거대한 시장 + 명확한 타이밍
   - TAM 540조 (부동산 거래), SAM 2조 (임차 거래), SOM 30억 (3년)
   - 전세사기 이후 신뢰 인프라 니즈 폭발
   - 전세→월세 전환 = 매달 반복되는 "신뢰 검증" 수요

2. 아무도 안 하고 있다
   - 직방/다방: 매물 중심 (세입자 검증 기능 없음)
   - 피터팬/당근: 직거래 (신뢰 장치 없음)
   - 전세보증보험: 집주인만 검증 (세입자 검증 없음)
   → 세입자 관점 서비스 = 한국에 0개

3. 규제 환경이 유리해지고 있다
   - 전세사기 특별법 → 임대차 투명성 강화 기조
   - 마이데이터 확대 → 개인 신용/소득 데이터 활용 용이
   - 정부도 임대차 신뢰 인프라 필요성 인지
```

## 9. 투자 시 주의사항

### 9-1. 지분 희석 관리

```
목표: Series A까지 창업자 지분 60% 이상 유지

현재                    100%
TIPS 운영사 (1.5억)     → 85~90% (지분 10~15%)
엔젤 (5,000만)          → 80~85% (지분 5~7%)
Seed (10~15억)          → 60~70% (지분 15~20%)
────────────────────────────────────────────
Series A 전: 60~70% 유지 → 건강한 지분 구조

⚠️ 주의:
- Pre-Seed에서 지분 20% 이상 주지 말 것
- 밸류 너무 낮게 시작하면 후속 라운드에서 희석 과다
- SAFE 사용 시 밸류 캡 적정 수준 유지
```

### 9-2. 투자 계약 시 체크포인트

| 항목 | 주의사항 |
|------|----------|
| 밸류에이션 | Pre-money 기준인지 Post-money 기준인지 확인 |
| 의결권 | 보통주 vs 우선주, 이사회 구성 |
| 동반매도권 | Tag-along / Drag-along 조건 |
| 반희석 조항 | Full ratchet은 피할 것, Weighted average OK |
| 청산 우선권 | 1x non-participating이 표준 |
| 경업금지 | 범위와 기간 합리적인지 |
| SAFE 전환 조건 | 캡/할인율, 전환 트리거 명확히 |

### 9-3. Fryndo vs 입주해: 투자 시 아이템 포지셔닝

```
투자 피칭 시 → 입주해를 메인으로

이유:
1. 시장 규모가 더 큼 (540조 vs 20조)
2. 사회 문제 해결 = 정부/미디어 관심
3. B2B + B2G 수익 다각화 가능
4. 보증보험 연계 = 핀테크 밸류
5. 미국 레퍼런스 명확 (Zillow, SmartMove)

Fryndo 포지션:
- "첫 번째 제품으로 실행력 증명"
- "MAU/매출 실적 보유"
- "여행 + 부동산 = 1인가구 생활 플랫폼으로 확장 가능"
```
