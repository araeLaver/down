## 2. 시나리오 A: 예창패 선정 (5~6월)

### 가장 유리한 경로

```
예창패 선정 (5월)
  │
  ├─ 1주차: 법인 설립 (주식회사 프린도 / 입주해)
  │         ├─ 무보수 대표 등록 (4대보험 부담 제거)
  │         └─ 법인카드 발급 → 경비처리 시작
  │
  ├─ 2주차: 사업비 집행 시작
  │         ├─ 외주 디자인/마케팅 (사업비 사용)
  │         └─ 서버/인프라 비용 (사업비 사용)
  │
  ├─ 1~3개월: TIPS 운영사 피칭 시작
  │         ├─ 매쉬업벤처스, 스파크랩, 프라이머
  │         ├─ "예창패 선정 + MVP 완성 + 매출 실적" = 강력한 피칭 카드
  │         └─ TIPS 선정 시 → 대표 인건비 R&D비 항목으로 공식 지급
  │
  └─ G-Media 퇴사 타이밍
            ├─ TIPS 선정 전: 퇴사 NO (급여 보장 없음)
            ├─ TIPS 선정 후: 퇴사 OK (R&D비로 인건비 해결)
            └─ 매출 월 200만+ 달성 시: 퇴사 검토 가능
```

```
7월   [B] 초기창업패키지 하반기 신청
      [공통] 서비스 지표 리뷰

8~9월 [A] TIPS 결과 대기
      [B] 하반기 사업 결과 대기
      [공통] MAU/매출 성장

10월~ TIPS or 하반기 사업 선정 시 → 퇴사 검토
      미선정 → 2027 상반기 재도전 (더 강한 포지션)

────────────────────────────────────────────────────────────────────
핵심: 어떤 시나리오든 G-Media 재직 = 안전망
     매출 + 유저 지표 = 모든 루트의 공통 무기
────────────────────────────────────────────────────────────────────
```

## 11. 리스크 매트릭스

| 리스크 | 확률 | 영향 | 대응 |
|:-------|:----:|:----:|:-----|
| 예창패 탈락 | 중 | 중 | 하반기 재도전 + TIPS 직행 |
| 전부 탈락 | 낮 | 중 | G-Media 유지 + Bootstrap |
| 겸업 불허 | 낮 | 높 | 사전 동의서 확보로 예방 |
| 예창패 협약 시 전업 요건 | 중 | 높 | 사전 확인, 불가 시 관광벤처 선택 |
| 서비스 유저 0명 | 중 | 높 | 마케팅 전략 변경, 타겟 피봇 |
| 매출 발생 안됨 | 중 | 중 | 무료로 유저 확보 → 데이터로 투자 유치 |
| G-Media 퇴사 압박 | 낮 | 높 | 법적 보호 (겸업 동의서) |
| 체력 소진 (겸업 + 개발) | 높 | 중 | 주 10시간 이내 부업 제한, 자동화 우선 |

## 12. 핵심 원칙 3가지

```
1. 퇴사는 수입이 보장될 때만
   → TIPS 선정 or 매출 월 200만+ or 투자금 6개월 런웨이

2. 정부지원금은 보너스, 없어도 전진
   → 서비스 출시 + 유저 확보는 지원금 없이도 가능
   → 서버비 월 2~5만원, 마케팅은 SNS/커뮤니티

3. 매출과 유저가 모든 문을 연다
   → 소액이라도 매출 = 정부사업 재도전 시 가산점
   → 유저 지표 = TIPS/투자 피칭의 핵심 근거
   → 지금 당장 유료 오픈하는 게 가장 중요한 한 수
```
