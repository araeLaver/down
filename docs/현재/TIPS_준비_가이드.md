# TIPS 준비 가이드 (2026년 기준)

> **최종 업데이트**: 2026-03-16
> **핵심**: 운영사 선투자(1~2억)가 대표 생활비 해결의 열쇠. TIPS R&D비로는 대표 인건비 현금 지급 불가.

---

## 1. 예창패 현물 계상이란?

```
실제로 돈을 받는 게 아님

예창패 사업비 중 "자부담" 비율을 채우기 위해
대표가 무급으로 일한 시간을 금액으로 환산해서 서류에 적는 것

예시:
┌─────────────────────────────────────────────┐
│  대표자 인건비 현물 계상                       │
│                                               │
│  월 근무시간: 160시간                          │
│  시급 환산: ~25,000원 (경력 9년 기준)          │
│  월 계상액: 약 400만원                         │
│                                               │
│  → 통장에 입금: 0원                            │
│  → 서류상 자부담: 400만원/월 인정              │
│  → 정부 매칭비율 충족용 "숫자"일 뿐             │
└─────────────────────────────────────────────┘

즉, 장부에만 찍히고 실제 수입은 0원. 생활비 해결과는 무관.
```

---

## 2. 대표 인건비 — 예창패도 TIPS도 동일

```
⚠️ 정부지원사업 공통: 대표 인건비 현금 지급 불가

예창패: 대표 인건비 현물만 → 통장 입금 0원
TIPS:   대표 인건비 현물만 → 통장 입금 0원 (동일!)

→ "TIPS 선정 → R&D비로 대표 인건비 해결" 은 부정확
→ 정부지원금 통장에서 대표 급여 직접 지급 불가
→ 법인 자체 매출/투자금으로 급여 지급 후, 현물로 인정받는 구조

예외: 신규 채용 청년(만 39세 이하)은 현금 인건비 가능
      2026년부터 만 34세 이하 추가 인센티브
```

## 3. TIPS 신청 전제조건

| 조건 | 내용 |
|------|------|
| 운영사 선투자 | 1~2억원 필수 (2026년 수도권 2억) |
| 법인 | 필수 (개인사업자 불가) |
| 업력 | 7년 이내 |
| 접수 방식 | 2026년부터 수시 → **분기별 접수**로 변경 |
| 핵심 흐름 | 운영사 미팅 → 투자 심사 → 선투자 → TIPS 추천 |

## 4. TIPS 지원 규모

| 구분 | 기간 | 한도 | 비고 |
|------|------|------|------|
| 일반트랙 R&D | 24개월 | 최대 8억원 | 2025년 5억에서 상향 |
| 딥테크트랙 R&D | 36개월 | 최대 15억원 | 일반트랙 완료 기업만 |
| 창업사업화 (비R&D) | 10개월 | 최대 1.5억원 | 정부 70% + 자부담 30% |
| 해외마케팅 (비R&D) | 10개월 | 최대 1.5억원 | 사업화와 합산 최대 3억 |
| **총 합산** | | **최대 약 11억원** | R&D 8억 + 비R&D 3억 |

## 5. 심사 배점 (100점 + 가점 3점)

| 항목 | 배점 | 입주해 현황 |
|------|------|-------------|
| 개발 필요성 (기술 차별성) | 15점 | AI Trust Score, k-NN 검색 → OK |
| 개발 적정성 (목표 구체성) | 15점 | R&D 계획서 작성 필요 |
| 개발 실현성 (기술 검증) | 10점 | MVP 70% 완성 → 강점 |
| 사업화 전략 | 15점 | 매출 실적 필요 (지금부터) |
| 글로벌 성장 가능성 | 15점 | 글로벌 진출 전략 보강 필요 |
| 팀 역량 등 | 30점 | 1인 → 약점, 팀 보강 고려 |
| 가점 | +3점 | 창업경진대회 수상, 특허 등 |

**핵심**: 기술성(40점) + 사업성/글로벌(30점) = 전체의 70%

---

## 6. R&D 과제계획서 구성

### 요약서 (온라인 입력)
- 과제명, 개발 기간, 기술개발 개요, 기술 성숙도(TRL)

### 본문1 (HWP 업로드 — 핵심)

**입주해 기준 작성 항목:**

```
1. 기술개발 목표
   ├─ AI Trust Score 알고리즘 고도화 (0-120점 산출 모델)
   ├─ 자체 스크래핑 모듈 (홈택스/건강보험/국민연금)
   ├─ OpenSearch k-NN 기반 유사도 매칭
   └─ 성능 지표: 정확도 OO%, 처리시간 OO초

2. 기술 차별성
   ├─ CODEF API 미사용 → 자체 데이터 수집 파이프라인
   ├─ GPT-4o-mini 활용 비정형 데이터 분석
   └─ 기존 경쟁사 대비 기술적 우위 근거

3. 사업화 계획
   ├─ B2C: 세입자 프리미엄 구독 (4,900원/월)
   ├─ B2B: 중개업소 SaaS (49,000~199,000원/월)
   └─ TAM/SAM/SOM 수치 근거

4. 글로벌 진출
   ├─ 해외 임대 시장 확장 (일본/동남아 등)
   └─ 다국어 Trust Score 모델
```

### 본문2 (온라인 입력)
- 연구책임자 인적사항, 학력, 경력
- 최근 5년 R&D 실적, 논문, 특허

## 7. 진짜 돈이 들어오는 구조

```
┌─────────────────────────────────────────────────────┐
│  대표 급여가 실제로 가능한 시점                         │
│                                                       │
│  1. 운영사 선투자 (1~2억)                              │
│     → 투자금이므로 대표 급여 지급 가능!                 │
│     → 이게 TIPS의 진짜 메리트                          │
│     → 시점: 운영사 미팅 후 2~3개월                      │
│                                                       │
│  2. TIPS 비R&D (창업사업화 1.5억 + 해외마케팅 1.5억)   │
│     → 마케팅/외주 등에 사용 (대표 인건비 불가)           │
│                                                       │
│  3. TIPS R&D (최대 8억)                                │
│     → 대표 인건비: 현물만 (정부 통장에서 직접 지급 X)    │
│     → 신규 채용 청년(39세↓) 인건비: 현금 가능           │
│                                                       │
│  결론: 운영사 선투자 1~2억이 대표 생활비 해결의 열쇠     │
└─────────────────────────────────────────────────────┘
```

## 8. 예창패 → TIPS 현실적 타임라인

```
2026년

5월     예창패 선정 → 법인 설립
        ↓
6~7월   운영사 탐색 + IR 자료 준비 + 미팅
        타겟 운영사 (프롭테크/AI/데이터):
        ├─ 매쉬업벤처스 (AI/SaaS, 이택경 대표)
        ├─ 스파크랩 (B2C 플랫폼, 글로벌 네트워크)
        ├─ 카카오벤처스 (IT/플랫폼)
        ├─ 프라이머 (초기 스타트업)
        ├─ 블루포인트파트너스 (딥테크)
        └─ 퓨처플레이 (기술 중심)
        ↓
8~9월   운영사 투자 심사 → 선투자 1~2억 유치
        ★ 이 시점부터 대표 급여 지급 가능
        ↓
10월    TIPS 분기별 접수 (3분기 or 4분기)
        ↓
11~12월 서면평가 → 발표평가 → 최종 선정
        ↓
2027년
1~2월   3자 협약 체결 → R&D 사업비 집행 시작

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  법인 설립 ~ TIPS 사업비 집행까지: 약 8~10개월
  법인 설립 ~ 선투자 수령까지: 약 3~4개월 ← 핵심
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 9. 예창패 선정 시 재직 + 겸업 문제

```
문제: G-Media 재직 중인데 예창패 수행 가능한가?

├─ 예창패는 전담의무가 상대적으로 약함
├─ 단, 협약 시 전업 요건이 있을 수 있음 → 선정 후 확인 필수
├─ 겸업동의서가 있으면 법적으로 양쪽 커버 가능
└─ ⚠️ 겸업동의서 아직 G-Media 서명 미완료

해결 방법 (시나리오별):

A. 전업 요건 없음 → G-Media 유지 + 겸업
B. 전업 요건 있음 → 관광벤처로 전환 (전업 요건 약함)
C. 퇴사 필요 시 → 실업급여(월 ~180만) + 외주 개발 병행
```

---

## 10. 예창패 선정자의 TIPS 지원 시 유리한 점

- 예창패 자체가 TIPS 가점은 아님
- 단, 간접적 유리함:
  1. **실행력 증명**: 사업화 실적(매출/고객/MVP)으로 사업화 전략 15점 유리
  2. **정부사업 수행 경험**: 자금 집행/정산 경험 → 신뢰도 상승
  3. **중복 수행 가능**: 예창패와 TIPS R&D 동시 수행 가능
  4. **법인 설립 완료**: TIPS 신청 요건 충족

│                                                       │
│  ✅ 수정                                              │
│  "운영사 선투자 수령 → 투자금으로 대표 급여 → 퇴사 검토" │

│  퇴사 조건 (하나 이상):                                │

│  □ 서비스 매출 월 200만원+ (3개월 연속)                │
