# 투자유치 실행 가이드 (현실편)

> **작성일**: 2026-03-11
> **현실 기준**: 1인 예비창업자, 법인 미설립, 매출 0, 유저 0
> **핵심**: 뜬구름 말고 지금 당장 뭘 어떻게 해야 하는지

---

## 0. 냉정한 현실 진단

### 지금 투자유치가 가능한가?

```
솔직한 답: 아직 아니다.

투자자가 보는 것:
  1. 트랙션 (유저, 매출, 성장률) → ❌ 없음
  2. 팀 (2인 이상, 보완적 역량) → ❌ 1인
  3. 시장 검증 (PMF 신호) → ❌ 베타 전
  4. 법인 → ❌ 미설립

→ 지금 VC 미팅 잡아봤자 "좋은데, 트랙션 나오면 다시 오세요" 듣고 끝.
→ 투자 전에 해야 할 일이 있다.
```

### 투자 가능 시점

```
현실적 투자 가능 시점: 2026년 Q3 (7~9월)

그 전까지 확보해야 할 것:
  □ 법인 설립 완료
  □ 입주해 정식 출시
  □ 가입자 500명+
  □ 유료 유저 30명+ (월매출 15만원이라도)
  □ 월 성장률 수치 (3개월치)
  □ B2B 계약 1건+ (셰어하우스)

이것 없이 투자 미팅 = 시간 낭비
```

## 2. 루트 ① TIPS (가장 현실적)

### TIPS가 뭔지 정확히

```
TIPS = 민간투자주도형 기술창업지원

쉽게 말하면:
  1. TIPS "운영사" (= 민간 VC/액셀러레이터)가 먼저 투자 (1~2억)
  2. 정부가 매칭으로 R&D비 지원 (최대 3억, 2년)
  3. 합치면 최대 5억

핵심:
  - "운영사"가 투자하겠다고 해야 정부 지원이 따라옴
  - 즉, 운영사를 설득하는 게 전부
  - 운영사 = VC라고 생각하면 됨
```

### TIPS 신청 절차 (실제 순서)

```
STEP 1: 운영사 선택 + 접촉 (1~2주)
─────────────────────────────────
  ① tips.mss.go.kr 접속
  ② "운영사 현황" 메뉴에서 전체 리스트 확인 (약 70개사)
  ③ 프롭테크/AI/B2C 투자하는 운영사 5~10개 추림
  ④ 각 운영사 홈페이지에서 IR 접수 방법 확인
  ⑤ 콜드메일 or 홈페이지 IR 제출

STEP 2: 운영사 미팅 (2~4주)
─────────────────────────────────
  ① 서류 심사 통과 → 미팅 제안 받음
  ② 보통 1차 미팅 (30분~1시간, 줌 or 방문)
  ③ 관심 있으면 2차 미팅 (심층, 제품 데모)
  ④ 최종 투자 결정 → "투자 의향서" 받음

STEP 3: TIPS 과제 신청 (2~4주)
─────────────────────────────────
  ① 운영사가 "이 팀에 투자하겠다" → TIPS 포털에 과제 등록
  ② 창업팀이 R&D 사업계획서 작성
  ③ 기술성 + 사업성 평가 (서류 + 발표)
  ④ 선정 통보

STEP 4: 협약 + 사업 시작 (2~4주)
─────────────────────────────────
  ① 정부 R&D 협약 체결
  ② 운영사 투자 집행 (지분 투자)
  ③ R&D비 1차 지급
  ④ 사업 수행 시작

전체 소요: 약 3~6개월
```

### TIPS 접수 방법 (구체적)

```
TIPS 운영사별 접수 방법:

매쉬업엔젤스: mashupangels.com → "투자 문의" 폼
  → IR 덱 (PDF) + 서비스 URL + 대표 연락처

스파크랩: sparklabs.co.kr → "Apply" 버튼
  → 온라인 신청서 + IR 덱

프라이머: primer.kr → "지원하기"
  → 온라인 폼 (사업 소개, 팀 소개, 재무)

블루포인트파트너스: bluepoint.vc → 이메일 (ir@bluepoint.vc)
  → 1-pager + IR 덱

소풍벤처스: sopoong.net → "투자 제안" 폼
  → 사회적 임팩트 + 비즈니스 모델

퓨처플레이: futureplay.co → "Contact" 이메일
  → 기술 차별성 중심 IR

본엔젤스: bonangels.net → 이메일 (invest@bonangels.net)
  → IR 덱 + 제품 데모 영상

※ 대부분 상시 접수. 특별한 마감 없음.
※ 접수 후 1~2주 내 회신 (관심 없으면 무응답 or 정중한 거절)
```

### TIPS 선정 확률 올리는 법

```
1. "기술"이 있어야 한다
   → 입주해 기술: 자체 스크래핑 엔진 + Trust Score 알고리즘 + OpenSearch 시맨틱 매칭
   → 특허 출원하면 가산점 (출원만 해도 됨, 등록 안 돼도 OK)
   → 기술이 "왜 남들이 못 따라하는지" 설명 가능해야 함

2. 트랙션이 있으면 압도적으로 유리
   → MAU 1,000이면 상위 20%
   → 유료 전환 있으면 상위 10%
   → "유저가 돈을 냈다" = PMF 신호

3. 운영사가 좋아하는 팀
   → 1인이라도 풀스택이면 OK (비용 효율)
   → "외주비 0원으로 MVP 만들었다" = 실행력 증명
   → 도메인 전문성 (헥토데이터 3년 = 스크래핑 전문가)

4. 시장이 크고 타이밍이 맞아야
   → 전세사기 이슈 = 미디어/정부 관심 = 투자자도 관심
   → "왜 지금?" 대답이 명확하면 좋음
```

## 3. 루트 ② 액셀러레이터 (TIPS 전 단계로 활용)

### 현실적으로 지원 가능한 프로그램

| 프로그램 | 모집 시기 | 투자/지원 | 지원 방법 |
|----------|-----------|-----------|-----------|
| **프라이머** | 상/하반기 | 5,000만 / 7% | primer.kr 온라인 지원 |
| **스파크랩** | 분기별 | 5,000만~1억 / 6~8% | sparklabs.co.kr Apply |
| **신한 스퀘어브릿지** | 상반기 | 보육형 (투자 별도) | shinhan.com 공고 |
| **KB스타터스** | 상반기 | 3,000만~5,000만 | kbfg.com 공고 |
| **현대 제로원** | 상반기 | 보육+투자 | hyundai-zeroone.com |
| **네이버 D2SF** | 수시 | 1~3억 | d2startup.com 이메일 |
| **카카오벤처스** | 수시 | 개별 협의 | kakaoventures.com |

### 액셀러레이터 지원서 작성법

```
핵심: 짧고 임팩트 있게. 심사위원은 하루에 50개+ 읽음.

1. 한 줄 요약 (10초 안에 이해되게)
   "전세사기 이후, 세입자가 자신의 신뢰를 증명하는 플랫폼"

2. 문제 (2~3줄)
   "900만 임차가구가 매번 새 집주인에게 처음부터 신뢰를 쌓아야 합니다.
    결국 보증금이라는 '돈'으로만 해결. 사회초년생에게 가장 큰 부담."

3. 솔루션 (2~3줄)
   "재직/소득/신용 자동인증 + 이전 집주인 레퍼런스 = Trust Score.
    높은 점수 → 보증보험 연계 → 보증금 50% 절감."

4. Why Now (1~2줄)
   "전세→월세 전환 가속 + 전세사기 특별법 = 신뢰 인프라 수요 폭발"

5. 현재 단계 (구체적으로)
   "MVP 70% 완성, 3월 베타 출시 예정,
    예비창업패키지 심사중, 셰어하우스 B2B 미팅 준비중"

6. 팀 (왜 이 사람이 이걸 해야 하는지)
   "9년차 풀스택, 헥토데이터 3년 (금융 스크래핑),
    외주비 0원으로 3주 만에 MVP 개발"
```

## 4. 루트 ③ 엔젤 투자 (가장 빠른 소액)

### 엔젤 투자자 찾는 현실적인 방법

```
방법 1: 엔젤투자 매칭 플랫폼 (가장 접근성 좋음)
────────────────────────────────────────────────
- 비긴메이트 (beginmate.com): 엔젤-스타트업 매칭
  → 프로필 등록 → 투자자가 관심 표시 → 미팅
  → 투자금: 건당 100~1,000만원

- 오픈트레이드 (opentrade.co.kr): 비상장주식 거래
  → 법인 설립 후 등록 가능

- 와디즈 (wadiz.kr): 크라우드펀딩 + 투자형
  → "투자형" 선택 시 소액 다수 투자
  → 법인 필요, 증권신고서 제출


방법 2: 스타트업 행사/밋업 참석 (네트워킹)
────────────────────────────────────────────────
- 서울창업허브 행사: seoul-startup.kr (월 2~3회)
- 판교 스타트업 밋업: meetup.com 검색
- 디캠프 (d.camp): 프로그램 참가 → 투자자 네트워킹
- 넥스트라이즈: 6월 코엑스 (국내 최대)
- 스타트업 위크엔드: 부정기 해커톤
- 슬래시 (slash.page): 온라인 커뮤니티

※ 행사 가면 명함 돌리지 말고, 발표/데모 기회를 잡을 것
※ "투자해주세요"가 아니라 "피드백 주세요"로 접근


방법 3: 링크드인 콜드 DM
────────────────────────────────────────────────
- "엔젤투자" or "벤처투자" 검색
- 프롭테크/AI 관심 표시한 사람 필터
- 짧은 메시지 (3줄 이내) + 1-pager 링크
- 10명 보내면 1~2명 응답이 현실

※ 응답률 높이려면: 그 사람이 투자한 포트폴리오 언급
   "OO 서비스에 투자하신 걸 보고 연락드렸습니다.
    비슷한 영역에서 세입자 신뢰 플랫폼을 만들고 있습니다."


방법 4: 기존 창업자 소개 (가장 효과적)
────────────────────────────────────────────────
- 프롭테크 스타트업 대표들과 커피챗
  → "우리 투자자분 소개해줄게" = 따뜻한 소개 (warm intro)
  → 콜드메일 대비 미팅 성사율 5배+
- 어떻게 만나나?
  → 링크드인 DM: "같은 프롭테크 분야에서 창업 준비중인데 조언 부탁"
  → 디캠프/서울창업허브 행사에서 만남
  → 프롭테크 커뮤니티 (오픈카톡, 슬랙)
```

### 엔젤 투자 실제 절차

```
STEP 1: 투자자 발굴 + 접촉 (1~4주)
  → 위 방법으로 관심 투자자 5~10명 리스트업
  → 1-pager + 데모 링크로 접근
  → 미팅 제안

STEP 2: 미팅 (1~2회, 각 30분~1시간)
  → 1차: 사업 소개 + Q&A
  → 2차: 제품 데모 + 조건 논의
  → 투자 의향 확인

STEP 3: 조건 협의 (1~2주)
  → 투자금, 밸류에이션, 투자 구조 (SAFE or 전환사채 or 보통주)
  → 법무사/변호사 통해 계약서 작성
  → SAFE가 가장 간단 (계약서 2~3장)

STEP 4: 투자 집행 (1주)
  → 법인 계좌로 송금
  → 투자계약서 서명
  → 주주명부 변경 (보통주일 경우)

STEP 5: 사후 관리
  → 월간/분기 업데이트 이메일
  → 주요 마일스톤 공유
  → 후속 투자 시 우선 참여 기회
```

### SAFE 계약 실제 작성법

```
SAFE 계약서는 2~3페이지. 핵심 조항:

1. 투자금액: ____________원
2. 밸류에이션 캡: ________억원
3. 할인율: ____% (보통 10~20%)
4. 전환 트리거: 후속 투자 ____만원 이상 유치 시
5. Pro-rata 권리: 있음/없음

어디서 구하나:
- Y Combinator SAFE 한글 번역본 (무료, 구글 검색)
- 법률사무소 템플릿 (10~30만원)
- 스타트업 전문 변호사 자문 (50~100만원)
  → 법무법인 디라이트, 율촌, 태평양 스타트업팀

※ 첫 투자라면 변호사 자문 받는 것 추천 (50만원 아끼다가 나중에 큰일)
```

## 5. 루트 ④ 크라우드펀딩 (브랜딩 + 자금)

### 와디즈 투자형 vs 리워드형

```
투자형 (증권형):
  - 주식/채권 발행 → 다수 소액 투자
  - 법인 필요, 증권신고서 제출
  - 모집 한도: 연 15억원
  - 수수료: 모집액의 5~8%
  - 장점: 실제 투자금 확보
  - 단점: 서류 복잡, 소액 주주 다수 생김

리워드형:
  - "입주해 출시 전 1년 이용권" 같은 리워드 제공
  - 사업자등록증만 있으면 가능 (법인 불필요)
  - 수수료: 모집액의 10%
  - 장점: 간단, 시장 검증 + 자금 동시
  - 단점: 큰 금액 어려움 (보통 500만~3,000만)

현실 추천: 리워드형 먼저 (시장 반응 검증) → 이후 투자형
```

### 크라우드펀딩 성공 조건

```
1. 스토리가 있어야 함
   → "전세사기 피해자가 만든 서비스" (본인 경험 있으면)
   → "사회초년생의 보증금 부담을 줄이겠다"
   → 감정적 공감 = 펀딩 성공의 80%

2. 영상이 필수
   → 2~3분 소개 영상
   → 대표 얼굴 나오면 신뢰도 UP
   → 제작비: 직접 촬영 0원, 외주 50~200만원

3. 초기 3일이 승부
   → 지인 50명에게 사전 홍보
   → 오픈 첫날 목표의 30% 달성 → 와디즈 메인 노출
   → 메인 노출 → 자연 유입으로 나머지 채움

4. 리워드 설계
   → 입주해 프리미엄 1년 이용권: 39,000원 (정가 58,800원)
   → 얼리버드 500명 한정
   → B2B 패키지: 셰어하우스 1년 이용권 290,000원
```

## 7. 투자유치 전에 반드시 해야 할 것

### 7-1. 법인 설립 (모든 투자의 전제)

```
왜: 개인에게 투자하는 사람은 없음. 법인 주식에 투자함.

언제: 예창패 결과 나오면 바로 (2026년 5월)
     or 예창패 상관없이 빨리 하고 싶으면 지금

비용: 30~50만원 (등록면허세 + 법무사)
기간: 3~5일 (법무사 대행 시)

법인 설립 → 법인 통장 → 사업자등록 → 통신판매업 → 투자 유치 가능
```

### 7-2. 트랙션 만들기 (투자의 핵심)

```
투자자가 가장 좋아하는 숫자 (우선순위):

1순위: 매출 (0원이 아니기만 하면 됨)
  → "15만원이라도 유료 결제가 있다" = "시장이 돈을 낸다"
  → 월 100만원 매출이면 시드 투자 미팅 가능

2순위: 유저 수 + 성장률
  → 절대 수보다 "월 성장률"이 중요
  → 매달 30% 성장 (100→130→170→220...) = "곡선"이 보임
  → 3개월치 데이터면 충분

3순위: 리텐션 (재방문율)
  → MAU 대비 DAU 비율
  → 주간 리텐션 40%+ = 좋은 서비스

4순위: 단위경제학 (Unit Economics)
  → CAC (유저 획득 비용) vs LTV (고객 생애 가치)
  → LTV:CAC > 3:1 = 건강한 비즈니스
```

### 7-3. 지재권 확보 (밸류에이션 UP)

```
특허 출원 (등록 전이라도 "출원" 상태면 충분):

출원 가능한 기술:
  1. Trust Score 산출 알고리즘 (가중치 모델)
  2. 자체 웹 스크래핑 기반 소득/재직 인증 방법
  3. 시맨틱 매칭 기반 세입자-매물 추천 시스템
  4. 임대인 레퍼런스 수집/평가 방법

비용:
  - 직접 작성 (특허로): 관납료 약 16만원
  - 변리사 대행: 100~200만원
  - 정부 바우처 활용 시: 80% 할인 가능

상표 출원:
  - "입주해" 상표 출원: 관납료 6.2만원 (전자출원)
  - 특허로 (patent.go.kr)에서 직접 가능
  - 분류: 제9류 (소프트웨어), 제36류 (부동산 서비스), 제42류 (SaaS)
```

## 9. 투자 미팅 실전 가이드

### 미팅 전 준비

```
필수 지참:
  □ IR 피칭덱 (PDF, 12~15장)
  □ 1-pager (A4 한 장 요약)
  □ 제품 데모 (실제 서비스 or 영상)
  □ 재무 모델 (엑셀, 3년 전망)
  □ 명함 (법인 설립 전이면 개인 명함이라도)

복장: 캐주얼 OK (스타트업이니까), 깔끔하게만

장소: 보통 투자사 사무실, 요즘은 줌도 많음
시간: 30분~1시간
```

### 미팅 구조 (30분 기준)

```
0~5분: 인사 + 자기소개 (간단히)
  "입주해 대표 김다운입니다. 세입자 신뢰 프로필 플랫폼입니다."

5~15분: 피칭 (10분, 핵심만)
  문제 → 솔루션 → 시장 → 제품 데모 → 비즈니스 모델 → 트랙션 → Ask
  ※ 슬라이드 넘기면서 말하는 게 아니라, 대화하듯이

15~25분: Q&A (투자자 질문)
  자주 나오는 질문:
  - "직방/다방이 따라하면?"
  - "1인인데 리스크 아닌가?"
  - "세입자가 왜 돈을 내나?"
  - "집주인 레퍼런스를 왜 해주나?"
  - "BEP 언제?"
  - "이 돈 어디에 쓸 건가?"

25~30분: 네스트 스텝
  "관심 있으시면 다음 단계는 어떻게 되나요?"
  → 보통: "내부 논의 후 연락드리겠습니다" (1~2주)
  → 좋으면: "자료 추가로 보내주세요" or "다음 미팅 잡죠"
```

### 미팅 후 팔로우업

```
당일:
  → 감사 이메일 (미팅 요약 + 요청 자료 첨부)
  → "오늘 말씀드린 데모 링크와 재무 모델 첨부합니다."

1주 후 (답 없으면):
  → 가벼운 팔로우업
  → "지난번 미팅 이후 MAU가 20% 성장했습니다" 같은 업데이트

2주 후 (여전히 답 없으면):
  → 한 번 더 팔로우업 후 넘어가기
  → 3번 이상 follow-up은 역효과

거절 받았을 때:
  → "피드백 주실 수 있으시면 감사하겠습니다"
  → 피드백은 금. 다음 미팅에 반영.
  → 3개월 후 트랙션 업데이트로 재접촉 가능
```
