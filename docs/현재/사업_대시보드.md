### 2-4. 중복수혜 규정

```
┌─────────────────────────────────────────────────────────────────┐
│  관광벤처 ↔ 예비창업패키지  = 동시 신청 O / 동시 수행 X         │
│  관광벤처 ↔ 경기도 예비창업 = 동시 신청 O / 동시 수행 O (지자체)│
│  예비창업패키지 ↔ 경기도    = 동시 신청 O / 동시 수행 O (지자체)│
│                                                                 │
│  → 관광벤처 + 예창패 둘 다 붙으면 1개만 선택                    │
│  → 경기도는 뭘 선택하든 추가 수령 가능                           │
└─────────────────────────────────────────────────────────────────┘
```
