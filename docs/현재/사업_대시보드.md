### 2-3. 중복수혜 규정 (중요!)

```
┌─────────────────────────────────────────────────────────────────┐
│  [중복 신청/수행 규정]                                           │
│                                                                 │
│  관광벤처 ↔ 예비창업패키지  = 동시 신청 O / 동시 수행 X         │
│  관광벤처 ↔ 경기도 예비창업 = 동시 신청 O / 동시 수행 O (지자체)│
│  예비창업패키지 ↔ 경기도    = 동시 신청 O / 동시 수행 O (지자체)│
│                                                                 │
│  → 관광벤처 + 예창패 둘 다 붙으면 1개만 선택해야 함             │
│  → 경기도는 뭘 선택하든 추가 수령 가능                           │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2-4. 예창패 vs 관광벤처 비교

| | **예비창업패키지** | **관광벤처 예비 (17회)** |
|:--|:--|:--|
| 지원금 | 최대 **1억** (평균 5천만) | 최대 **7천만** (평균 4,400만) |
| 선정규모 | **~748명** | **20명** |
| 사업기간 | 8개월 | 7개월 |
| 대표 인건비 | 불가 | 불가 |
| 전업 요건 | 협약 전 퇴사 필수 | 상대적 유연 (종료 2개월 내 사업자등록) |
| 부가혜택 | 멘토링/교육 | **TIPS 연계, 관광기업 육성펀드, 관광벤처 자격** |
| Fryndo 적합도 | 범용 IT | **관광 특화 = 딱 맞음** |

```
┌─────────────────────────────────────────────────────────────────┐
│  [전략: 3개 모두 신청 → 결과 보고 선택]                          │
│                                                                 │
│  🔴 관광벤처 예비 신청   (3/4 마감, 지금!)                       │

│                                                                 │
│  [선정 결과별 선택]                                              │
│  ├─ 둘 다 선정 → 예창패 선택 (금액 큼) + 경기도                 │
│  ├─ 관광벤처만  → 관광벤처 + 경기도 (최대 8,500만)              │
│  ├─ 예창패만    → 예창패 + 경기도 (최대 1.15억)                 │
│  └─ 경기도만    → 1,500만으로 린하게 시작                       │
│                                                                 │
│  ※ 선정 전까지 재직 유지 → 리스크 최소화                        │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 2-6. 리스크 & 대응

| 리스크 | 심각도 | 대응 |
|:-------|:------:|:-----|
| 대표 인건비 불가 (둘 다) | 🔴 | 재직 유지 or 자부담 현물로 인건비 인정 |

| 사업비 오용 → 형사처벌 | 🔴 | 사용기준 철저 준수, 세무기장료도 사업비 OK |
