## 사업계획서 작성 가이드

> 반드시 **touraz.kr 제공 양식**을 사용할 것 (임의양식 평가 제외)
> 아래는 양식에 채울 내용 초안

### 1. 사업개요 (서류 25% / 발표 20%)

**서비스명**: Fryndo (프린도)

**한 줄 소개**: 혼자 여행하는 관광객을 위한 AI 기반 동반자 매칭 플랫폼

**모집유형**: 관광체험서비스
- 공고 예시와 일치: "AI기반 여행 큐레이션 서비스"

**문제인식의 구체성**:

```
[Problem 1] 1인 관광객 증가 vs 동행 인프라 부재
- 1인 여행 비율 25% 이상 (2024 한국관광공사)
- 혼자 여행 시 불편함 경험 67%
- "동행 있으면 좋겠다" 응답 72%
- 현실: 식당 1인석 부족, 사진 촬영 어려움, 안전 우려(여성)

[Problem 2] 수도권 집중 관광 vs 지역 관광 침체
- 서울/경기 관광객 집중 65%
- 비수도권 관광 수입 지속 감소
- 지역 관광 콘텐츠 부족

[Problem 3] 관광 경험의 휘발성
- 여행 후 기록 남기는 비율 23%
- 방문 인증 체계 부재
```

### 2. 시장성 (서류 25% / 발표 20%)

**시장현황 및 목표고객**:

```
TAM: 국내 관광시장 × 1인 여행자 = 28조 × 25% = 7조원
SAM: 디지털 플랫폼 이용 1인 여행자 = 7조 × 40% = 2.8조원
SOM: 초기 타겟 (MZ세대 + 주요 관광지) = 시장 0.1% = 28억원

1차 타겟: 25~34세 여성 1인 여행자 (제주도)
2차 타겟: MZ세대 전체 (부산, 강릉 등 확대)
```

**사업모델 (BM)**:

```
[B2C] 프리미엄 구독 ₩9,900/월 (무제한 매칭, 프리미엄 추천)
[B2B] 지자체/관광공사 제휴 (지역 관광 데이터, NFT 발행 대행)
[B2G] 관광 데이터 공공 제공, 관광 안전 시스템 연계
```

**잠재리스크 및 대응방안**:

| 리스크 | 대응 |
|:-------|:-----|
| 안전사고 (동행 간 분쟁) | 실명 인증, 후기 시스템, 긴급 SOS, 실시간 위치 공유 |
| NFT 시장 침체 | 투기 아닌 "수집/인증" 목적, 구독+B2B로 매출 80% 유지 |
| 사용자 확보 어려움 | 제주도 1지역 집중 → 검증 후 확대 |
