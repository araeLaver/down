| 항목 | 내용 |
|------|------|
| **주관** | 중소벤처기업부 / 창업진흥원 |
| **지원금** | 최대 1억원 (평균 5천만원) |

**재직자 신청:**
- 신청 시: ✅ 재직 중 가능

## PART 8. 대표자 인건비/생활비 현실

### 핵심: 대표자 본인 급여 지급 불가

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   예비창업패키지 사업비 5천만원 받으면...                     │
│                                                             │
│   ✅ 사용 가능          ❌ 사용 불가                         │
│   ─────────────        ─────────────                        │
│   • 직원 인건비         • 대표자 본인 급여                   │
│   • 외주용역비          • 대표자 4대보험                     │
│   • 재료비/장비                                              │
│   • 사무실 임대료                                            │
│   • 마케팅비                                                 │
│   • 특허/법인설립비                                          │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 생활비 해결 방법 3가지

| 방법 | 설명 | 추천도 |
|------|------|--------|
| **① 투잡 유지** | 퇴사 안 하고 직장 병행 | ⭐⭐⭐⭐⭐ |
| **② 개인 저축** | 8개월 생활비 미리 확보 | ⭐⭐⭐ |
| **③ 법인 무보수** | 대표 무급 등록, 직장 유지 | ⭐⭐⭐⭐ |

### 추천 전략: 투잡 유지

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│  [재직 유지]     [예비창업패키지]      [퇴사 시점]           │
│                                                             │
│   직장 다니면서  →  선정 & 협약  →  MVP 완성 후             │
│   신청/준비         사업 수행       투자유치 확정 시         │
│                                                             │
│   • 필수 교육/행사만 연차 사용                               │
│   • 법인 설립 시 "무보수 대표" 등록                          │
│   • 생활비 걱정 없이 사업 집중                               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**선배 창업자 조언:**
> "직장인일 경우 바로 퇴사하지 말고 최대한 투잡으로 병행하세요.
> 시제품 완성하고 초기 투자가 어느 정도 논의된 후에 퇴사해도 늦지 않습니다."
