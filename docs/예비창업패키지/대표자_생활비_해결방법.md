# 예비창업패키지 대표자 생활비 해결 방법 총정리

> 예비창업패키지는 대표자 인건비를 현금으로 지급하지 않음 (현물 계상만 가능)
> 아래는 대표 생활비를 확보할 수 있는 현실적인 방법들을 정리한 문서

---

## 1. 직접 수입 확보

### 외주 개발 병행
- **위시켓** (wishket.com) - IT 외주 최대 플랫폼, 프로젝트/상주 모두 가능 (수수료 ~12%)
- **크몽** (kmong.com) - 소규모 단건 작업 적합 (수수료 20%)
- **숨고** (soomgo.com) - 견적 기반 매칭
- **원티드 긱스** - 단기 상주 인력
- 예창패는 전담의무가 약해 겸업 가능 (단, 심사 시 실행력 평가)

### 매출 발생
- 시제품 완성 즉시 얼리 매출 만들기
- 매출에서 대표 급여 지급 가능 (지원금이 아닌 자체 매출)

### 개발자 부수입
- Micro SaaS: 작은 구독형 서비스 만들어 월 고정 수익
- 앱스토어 유료앱/인앱결제
- 개발 강의 (탈잉, 클래스101, 유데미)
- GitHub Sponsors 오픈소스 후원

---

## 2. 투자 유치 → 대표 급여 지급

### SAFE 투자 (조건부지분인수계약)
- 지분가치 평가 없이 빠르게 투자금 수령
- 이자/만기 부담 없음
- 투자금으로 대표 급여 지급 가능
- 예창패 선정 이력 있으면 투자자 설득 용이
- 참고: zuzu.network/resource/guide/safe/

### 엔젤투자
- 개인 투자자에게 1억 이하 소액 투자 유치
- 한국엔젤투자협회 매칭 가능 (kban.or.kr)
- 엔젤투자 매칭펀드: 엔젤투자 시 정부가 매칭 추가 투자

### 액셀러레이터 (AC)
- 매쉬업벤처스, 스파크랩, 프라이머 등
- 투자금 (보통 1~3천만원) + 사무공간 + 멘토링
- 투자금으로 대표 급여 충당 가능

### 크라우드펀딩 (사전판매 = 즉시 매출)
- **와디즈** (wadiz.kr) - 시제품 리워드 펀딩, 출시 전 사전판매
- **텀블벅** (tumblbug.com) - 창작/콘텐츠 중심
- 증권형 크라우드펀딩 - 지분 투자 형태 (1인당 최대 500만원)

## 3. 대출 (생활비 버퍼 확보)

### 정책자금
- **청년전용 창업자금**: 최대 1억원, 연 2.5% 고정 (만 39세 이하)
- **소상공인 정책자금**: 사업자등록 후 신청, 저금리 변동
- **중진공 정책자금**: 최대 1억원, 저금리 (kosmes.or.kr)

### 보증서 대출
- **기술보증기금 사전보증**: 예비창업자도 기술평가 후 보증서 발급 → 은행 대출 (kibo.or.kr)
- **신용보증기금**: 청년창업특례보증, 혁신형 창업기업보증 (kodit.co.kr)
- **카카오뱅크 보증서대출**: 개인사업자 비대면 신청 가능
- 전략: 신보로 먼저 운영자금 확보 → 기보로 전환

### 기타 금융
- **서민금융진흥원 긴급소액생계비**: 100만원 당일 지급 (1397 전화)

---

## 4. 비용 절감

### 법인 운영
- **법인 무보수 대표 등록**: 건강보험공단에 무보수 신청서 제출 → 4대보험 부담 제거
- **대표 인건비 현물 처리**: 자부담 비율 충족용 서류상 계상 (현금 지출 X)

### 법인카드 경비처리
- 식비, 교통비, 통신비, 소모품 등 업무 관련 비용 → 법인 경비 처리
- 복리후생비: 식대 월 20만원 비과세, 교통비 등
- 출장비/여비: 업무 관련 이동 경비 처리
- 체감 월 수십만원 생활비 절감 효과

### 공간 비용 절감
- 창업보육센터/창업공간 입주 (경기창업혁신공간 등)

## 7. 가족/지인 자금

- **가족 차용**: 차용증 작성 후 무이자/저이자 (증여세 주의: 10년간 5천만원 비과세)
- **지인 투자**: SAFE 계약서로 깔끔하게 처리

---

## 현실적 추천 조합

1. **위시켓 외주 개발**로 당장 생활비 확보 (월 300~500 가능)
2. **법인 무보수 등록 + 법인카드 경비처리**로 지출 최소화
3. **기보 사전보증 or 청년전용 창업자금** 대출로 버퍼 확보
4. 예창패 선정 후 **SAFE 엔젤투자** 유치하여 대표 급여 체계화
5. 시제품 완성 시 **와디즈 크라우드펀딩**으로 얼리 매출
