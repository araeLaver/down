### 1.2 비전 & 미션
- **비전**: 전 세계 여행자들이 신뢰할 수 있는 동반자를 찾고, 여행의 추억을 디지털 자산으로 영구 보존하는 글로벌 플랫폼
- **미션**: 혼자 여행하기 망설이는 사람들에게 안전하고 즐거운 동행 경험을 제공하고, 여행지 방문을 게이미피케이션으로 동기부여

1. **🛡️ 안전한 동행 매칭**: 신분증 인증, 리뷰 시스템, 긴급 SOS로 신뢰 구축
2. **👥 검증된 동행자**: 여행 스타일/관심사 기반 AI 매칭 알고리즘
3. **📍 여행 기록 자산화**: GPS 기반 방문 인증 (NFT는 선택적 부가 기능)

## 2. 서비스 상세

### 2.1 핵심 기능

#### A. 여행 동반자 매칭
```
[사용자 프로필] → [여행 그룹 생성/참여] → [실시간 채팅] → [동행 확정]
```
- 여행 스타일, 관심사, 일정 기반 매칭 알고리즘
- 실시간 WebSocket 채팅 (이미지/위치 공유)
- 팔로우/팔로워 소셜 네트워크
- 동행 후기 및 평점 시스템

#### B. 위치 기반 NFT 수집
```
[GPS 위치 확인] → [특정 장소 도착] → [NFT 민팅] → [컬렉션 저장]
```
- Polygon 네트워크 기반 (가스비 저렴)
- 희귀도 등급: Common → Rare → Epic → Legendary
- 장소별 한정 NFT (예: 에펠탑 방문 인증 NFT)
- 리더보드 경쟁 시스템

#### C. 커뮤니티 기능
- 여행 후기 공유
- 장소 리뷰 및 평점
- 여행 팁 게시판
- 실시간 여행자 위치 공유 (옵션)

## 3. 시장 분석

### 3.1 타겟 시장 규모

#### TAM (Total Addressable Market)
- 글로벌 여행 시장: **$1.9조** (2025년 예상)
- 1인 여행자 비율: 약 25%
- **TAM**: ~$4,750억

#### SAM (Serviceable Addressable Market)
- 아시아 태평양 1인 여행 시장: ~$950억
- 디지털 플랫폼 이용 비율: 40%
- **SAM**: ~$380억

#### SOM (Serviceable Obtainable Market)
- 초기 타겟: 한국 + 일본 + 동남아
- 시장 점유 목표 (3년): 0.1%
- **SOM**: ~$3.8억 (약 5,000억원)

### 3.2 타겟 고객

#### Primary Target: MZ세대 1인 여행자 (20-35세)
- 혼자 여행은 원하지만 외로움/안전 우려
- SNS/디지털에 익숙
- 새로운 경험에 적극적
- NFT/블록체인에 관심

#### Secondary Target: 시니어 여행자 (50-65세)
- 은퇴 후 여행 욕구 증가
- 동반자 필요성 높음
- 프리미엄 서비스 지불 의향

### 3.3 시장 트렌드
1. **1인 여행 증가**: 코로나 이후 55% 증가 (Booking.com 조사)
2. **여행 + 디지털 자산**: NFT 여행 인증 시장 성장
3. **안전한 동행 수요**: 특히 여성 여행자 중심
4. **경험 공유 욕구**: 인스타그래머블 여행 트렌드

### 8.2 Fryndo 안전 시스템 (5단계)

#### Level 1: 가입 단계 - 신원 검증
| 인증 유형 | 방식 | 필수 여부 |
|-----------|------|-----------|
| **휴대폰 인증** | 본인 명의 휴대폰 | 필수 |
| **신분증 인증** | AI 신분증 OCR + 얼굴 매칭 | Pro 필수, Plus 선택 |
| **SNS 연동** | 카카오/네이버/인스타그램 | 선택 (신뢰도 점수 반영) |
| **직장/학교 인증** | 이메일 도메인 인증 | 선택 (신뢰도 점수 반영) |

**인증 뱃지 시스템**
```
🔵 기본 인증 (휴대폰)
🟢 신분증 인증 완료
🟡 SNS 연동 (팔로워 100+)
🟣 직장/학교 인증
⭐ 슈퍼 인증 (모든 인증 완료)
```

#### Level 2: 매칭 단계 - 신뢰도 점수
```
신뢰도 점수 = 인증 점수(40%) + 리뷰 점수(40%) + 활동 점수(20%)

• 인증 점수: 신분증(+30), SNS(+5), 직장(+5)
• 리뷰 점수: 동행 후기 평균 (5점 만점 → 40점 환산)
• 활동 점수: 가입 기간, 매칭 횟수, 신고 이력

최소 매칭 조건: 신뢰도 점수 60점 이상
```

#### Level 3: 동행 중 - 실시간 안전
| 기능 | 설명 | 제공 플랜 |
|------|------|-----------|
| **실시간 위치 공유** | 그룹원 간 위치 공유 (동의 시) | Plus+ |
| **긴급 SOS 버튼** | 원터치 긴급 신고 + 자동 위치 전송 | 전체 |
| **지정인 알림** | 가족/친구에게 여행 경로 공유 | Pro |
| **체크인 알림** | 정기적 안전 확인 푸시 | Pro |

**긴급 SOS 작동 방식**
```
[SOS 버튼 3초 누름]
       ↓
[자동 녹음 시작 + GPS 전송]
       ↓
[지정 연락처 + 112 자동 연결]
       ↓
[Fryndo 안전팀 알림]
```

#### Level 4: 동행 후 - 리뷰 시스템
- **필수 리뷰**: 동행 완료 후 72시간 내 리뷰 작성 필수
- **상호 리뷰**: 양측 모두 작성해야 공개
- **리뷰 항목**: 매너, 시간 준수, 소통, 재동행 의사
- **악성 리뷰 방지**: AI 감정 분석 + 신고 시스템

#### Level 5: 사후 관리 - 신고/제재
| 위반 유형 | 제재 수준 |
|-----------|-----------|
| 노쇼 (연락 두절) | 경고 1회 → 7일 정지 → 영구 정지 |
| 부적절한 행동 | 즉시 조사 → 30일 정지 → 영구 정지 |
| 범죄 행위 | 즉시 영구 정지 + 수사기관 협조 |
| 허위 프로필 | 즉시 영구 정지 |
