## Phase 1: 법인 설립 (예상 기간: 2주)

### 1.1 법인 형태 선택

| 형태 | 최소 자본금 | 장점 | 단점 |
|------|------------|------|------|
| **주식회사** | 제한 없음 | 투자 유치 용이, 신뢰도 높음 | 설립 비용 높음 |
| 유한회사 | 제한 없음 | 설립 간단 | 투자 유치 어려움 |
| 개인사업자 | 없음 | 즉시 시작 | 무한 책임, 투자 불가 |

**권장: 주식회사** (투자 유치 및 확장 고려)

### 1.2 법인 설립 절차

```
[정관 작성] → [발기인 총회] → [주금 납입] → [설립등기] → [사업자등록]
    1일          1일           1일          3-5일         1-2일
```

#### Step 3: 정관 작성

필수 기재 사항:
1. 목적
2. 상호
3. 발행할 주식의 총수
4. 1주의 금액

7. 회사가 공고하는 방법

#### Step 4: 자본금 납입

1. **개인 통장**에 자본금 입금 (본인 명의)
2. 은행에서 **잔액증명서** 발급
3. 등기 후 법인 통장 개설 시 이체

**온라인 등기 (권장)**

**필요 서류**
- [ ] 정관 1부
- [ ] 발기인 총회 의사록 1부
- [ ] 주식발행사항 동의서 1부
- [ ] 취임승낙서 (대표이사) 1부
- [ ] 인감신고서 1부

#### NFT 관련 특별 조항 (중요)

```markdown
제13조 (NFT 서비스 특별 조항)

1. NFT의 성격
   - 본 서비스에서 발행되는 NFT는 여행 인증 목적의 디지털 수집품입니다.
   - NFT는 투자 상품이 아니며, 회사는 NFT의 가치 상승을 보장하지 않습니다.

2. 소유권
   - 사용자는 NFT에 대한 소유권을 가지나, 기초 콘텐츠의 저작권은 회사에 귀속됩니다.
   - NFT의 상업적 이용은 제한됩니다.

3. 거래
   - NFT 거래 시 발생하는 블록체인 네트워크 수수료(가스비)는 사용자 부담입니다.
   - 회사는 2차 거래 금액의 5%를 로열티로 수취합니다.

4. 환불
   - 민팅 완료된 NFT는 블록체인의 특성상 환불이 불가합니다.
   - 민팅 실패 시 지불한 금액은 전액 환불됩니다.

5. 서비스 종료
   - 서비스 종료 시에도 발행된 NFT는 사용자 지갑에 보존됩니다.
   - 단, NFT 관련 부가 기능(리더보드 등)은 제공되지 않습니다.
```
