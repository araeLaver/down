## 대표 월급 관련 중요사항

**정부 창업지원사업(예비창업패키지 등)에서 대표자 본인 급여 지급 불가**

### 권장 대응 전략:
1. **투잡 유지** - 현 직장 유지하면서 창업 병행 (가장 추천)
2. **8개월분 생활비 준비** - 개인 저축으로 충당
3. **무급 대표 등록** - 직장 유지하면서 무급 대표로 참여
