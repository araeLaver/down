## PART 1: 현실 점검

### 기존 계획의 문제점

| 문제 | 현실 | 해결책 |
|------|------|--------|
| NFT 의존도 높음 | NFT 시장 불확실, MZ세대도 관심 감소 | **동반자 매칭이 핵심**, NFT는 부가기능 |
| 투자 유치 전제 | 초기 스타트업 투자 받기 매우 어려움 | **부트스트래핑으로 시작** |
| 팀 빌딩 가정 | 좋은 팀원 구하기 어려움 | **1인 창업으로 MVP 검증** |
| 정부 지원 의존 | 예비창업패키지 경쟁률 10:1 | **지원 없이도 시작 가능한 구조** |
| 비용 과다 | 월 170만원 고정비 | **월 30만원 이하로 시작** |

### 현실적 시나리오

```
[최악의 경우를 가정]

- 투자: 0원
- 정부 지원: 탈락
- 팀원: 없음
- 초기 자금: 본인 자금 500만원

→ 이 상황에서도 창업 가능해야 함
```

## PART 2: MVP 재정의

### 핵심 가치 우선순위 재조정

```
[기존]                              [수정]
1. 동반자 매칭         →            1. 동반자 매칭 (핵심)
2. NFT 수집            →            2. 커뮤니티/후기 (신뢰)
3. 안전 기능           →            3. 실시간 채팅 (소통)
```

## PART 4: 자금 조달 다각화

### 투자 없이 시작하는 방법

#### Option 1: 부트스트래핑 (권장)

```
[필요 자금: 최소 300만원]

초기 비용:
- 법인 설립: ₩200,000
- 도메인/인프라: ₩50,000
- 마케팅 테스트: ₩200,000
- 예비비: ₩350,000
─────────────────
합계: ₩800,000

6개월 운영비:
- 월 ₩50,000 × 6 = ₩300,000

총 필요: ₩1,100,000 (약 110만원)
```

#### Option 2: 소액 정부 지원

| 지원 사업 | 금액 | 경쟁률 | 신청 시기 |
|-----------|------|--------|-----------|
| 소상공인 창업 | 최대 2천만원 | 낮음 | 수시 |
| 청년창업사관학교 | 최대 1억 | 중 | 연 2회 |

### 유료 전환 유도 전략

#### Free vs Plus 차별화

| 기능 | Free | Plus (₩9,900/월) |
|------|------|------------------|
| 여행 그룹 참여 | 월 3개 | **무제한** |
| 채팅 | 월 20회 | **무제한** |
| 프로필 노출 | 기본 | **상단 노출** |
| 매칭 알림 | 없음 | **실시간 알림** |
| 후기 열람 | 일부 | **전체 열람** |

#### 전환 유도 포인트

```
1. 채팅 제한 도달 시
   → "이번 달 채팅을 다 사용했어요. Plus로 무제한 채팅하세요!"

2. 인기 그룹 참여 시
   → "이 그룹은 Plus 회원 우선입니다"

3. 프로필 조회수 알림
   → "당신의 프로필을 10명이 봤어요! Plus로 누가 봤는지 확인하세요"
```

## PART 7: 리스크 최소화

### 최악의 시나리오 대비

| 시나리오 | 대응 |
|----------|------|
| 사용자가 안 모임 | 타겟 좁히기 (제주도 여행자만) |
| 매칭이 안 됨 | 운영자가 직접 매칭 지원 |
| 유료 전환 0% | 가격 낮추기 (₩4,900) |
| 6개월 후 실패 | 피봇 or 정리 (손실 100만원 이하) |

### 피봇 가능성

```
[원래 아이템]
여행 동반자 매칭 + NFT

[피봇 옵션 A]
여행 동반자 매칭만 (NFT 포기)
→ 더 단순하고 명확한 가치

[피봇 옵션 B]
소모임/취미 동반자 매칭
→ 여행 → 등산, 러닝, 맛집 등 확장

[피봇 옵션 C]
여행 후기/정보 커뮤니티
→ 매칭 실패 시 콘텐츠 플랫폼으로
```

### Exit 기준 설정

```
3개월 후 점검:
├── MAU 100명 미달 → 마케팅 전략 수정
├── 매칭 성사 0건 → 기능/UX 전면 재검토
└── 유료 전환 0명 → 가격/가치 재설계

6개월 후 점검:
├── MAU 300명 미달 → 피봇 검토
├── MRR ₩10만원 미달 → 수익모델 재검토
└── NPS 0 미만 → 서비스 정리 검토
```
