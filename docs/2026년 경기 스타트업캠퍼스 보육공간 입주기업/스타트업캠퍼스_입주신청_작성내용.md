## [지정양식] 사업계획서 작성 내용

### I. 기업 현황 및 입주공간 필요성

#### 1. 창업 배경 및 동기

804만 1인 가구 시대, 혼자 여행하는 사람이 급증하고 있지만 기존 플랫폼(직방, 에어비앤비, 트립어드바이저)은 "예약"에만 집중하고 "동행"은 제공하지 않습니다.

**3대 문제:**
1. **동행 부재** — 혼자 여행한 사람 54%가 "외로웠다" 응답
2. **안전 불안** — 여성 1인 여행자 78%가 "안전이 최대 걱정"
3. **기록 휘발** — 여행 사진은 SNS에 묻히고, 방문 인증 수단 없음

10년간 풀스택 개발자로 매칭 알고리즘(에너지마켓플레이스), 블록체인/NFT(크리에이티브힐), 금융 인증(IBK)을 직접 개발한 경험을 바탕으로, 이 세 가지 문제를 하나의 플랫폼에서 해결하는 Fryndo를 개발했습니다.

**MVP는 100% 자체 개발 완료, 외주비 0원.**

#### 2. 제품/서비스 개요

**서비스명**: Fryndo (프린도) — "혼자 떠나도 외롭지 않은 여행"

**3대 핵심 기능:**

| 기능 | 설명 | 리소스 비중 |
|------|------|:----------:|
| **신뢰 기반 동행 매칭** | 프로필 인증 + 여행 스타일/일정/관심사 AI 복합 매칭 + 상호 평점/후기 | 80% |
| **GPS 기반 여행 NFT** | 여행지 도착 시 Polygon 블록체인 NFT 자동 발행, 4단계 희귀도 | 15% |
| **실시간 안전 케어** | 위치 공유, 긴급 SOS 버튼, 신뢰 점수 필터링 | 5% |

**기술 스택:**
- Frontend: React, TypeScript
- Backend: Flask/Python, PostgreSQL, Redis
- 실시간: WebSocket (STOMP)
- 블록체인: Web3.js, Polygon Network, IPFS
- 인프라: Koyeb(PaaS), Docker, GitHub Actions CI/CD
- AI: Python ML (매칭 알고리즘), 자연어 처리

**서비스 URL**: https://various-belva-untab-1a59bee2.koyeb.app (MVP 배포 완료)

- 예비창업자로 사업자등록을 위한 사업장 주소가 필요
- 입주 확정 시 법인 설립 + 사업자등록을 판교 주소로 진행

**② 판교 IT 생태계 접근**
- 네이버, 카카오, NHN 등 **B2B 제휴 대상 기업이 모두 판교에 밀집**
- 투자자(VC/AC) 미팅 접근성 최상 (TIPS 운영사 다수 소재)
- 스타트업 네트워킹, 멘토링 프로그램 활용

**③ 단계별 성장 지원**
- 예비(3층) 입주 → 초기(5층) 이동 가능 → 성장(8층)까지 연속 보육
- 교육장, 멘토룸, 회의실 등 인프라 활용으로 초기 비용 최소화

**④ 현재 재직 중이지만 전업 준비 완료**
- 정부 지원사업 선정(예창패/관광벤처) 시 퇴사 → 판교 상주 전업
- 겸업동의서 준비 완료, 전업 전환 조건 명확

**⑤ 경기도 거주 + 창업**
- 경기도 광주시 거주 → 경기도 내 창업으로 지역 경제 기여

#### 2. 기술성 (30점 — 가장 비중 높음)

**핵심 기술 4가지:**

**① AI 매칭 알고리즘**
- 여행 스타일(MBTI형) + 일정 + 관심사 + 위치를 복합 분석
- 협업 필터링 + 콘텐츠 기반 필터링 하이브리드
- 매칭 후 상호 평점/후기로 알고리즘 지속 학습
- 경력: 에너지마켓플레이스 매칭 알고리즘 실무 개발 경험

**② 블록체인 NFT (Polygon)**
- GPS 위치 도달 시 Polygon 블록체인에 NFT 자동 민팅
- ERC-721 기반, IPFS에 메타데이터 저장
- 가스비 거의 0원 (Polygon PoS)
- 4단계 희귀도: Common → Rare → Epic → Legendary
- 경력: 크리에이티브힐에서 Polygon ERC-3525 NFT 플랫폼 개발

**③ 실시간 통신 (WebSocket)**
- STOMP 프로토콜 기반 실시간 채팅
- 동행자 간 실시간 위치 공유
- 긴급 SOS 버튼 → 즉시 알림 전송

**④ 보안/인증 시스템**
- JWT 기반 인증 + 소셜 로그인 (카카오/구글)

- 경력: IBK 기업은행 다중인증(MFA) 구현 경험
