# Fryndo AR Companion 사업계획서

> AR 안경 기반 여행 동행 보조 서비스
> 작성일: 2026-01-21

---

## 1. 개요

### 서비스명
**Fryndo AR Companion** - AR 안경 연동 여행 동행 보조 앱

### 비전
"AR 안경을 통해 언어 장벽 없이, 전 세계 여행자와 자연스럽게 연결되는 경험"

### 핵심 가치
- 기존 Fryndo(여행 동행 매칭) 서비스의 AR 확장
- 다국적 동행자 간 실시간 번역
- 여행지 AR 가이드 및 내비게이션
- NFT 수집 경험 강화

## 3. 서비스 상세

### 3.1 핵심 기능

#### 기능 1: 동행자 AR 태그
```
- 만남 장소에서 동행자 위치/프로필 오버레이
- 얼굴 인식으로 Fryndo 프로필 연동
- "저 사람이 내 동행자!" 즉시 확인
```

#### 기능 2: 실시간 그룹 번역
```
- 다국적 동행자 간 대화 자동 번역
- AR 안경 디스플레이에 자막 표시
- 지원 언어: 120+ (Google/DeepL API)
```

#### 기능 3: 여행지 AR 가이드
```
- 관광지 정보 오버레이
- 맛집/카페 리뷰 실시간 표시
- 역사적 장소 AR 복원 (선택)
```

#### 기능 4: AR 내비게이션
```
- 도보 방향 화살표 오버레이
- 동행자 위치 실시간 공유
- "여기서 만나요" 핀 공유
```

#### 기능 5: NFT 민팅 트리거
```
- 특정 장소 도착 시 자동 민팅 알림
- AR로 NFT 수집 애니메이션
- 희귀 NFT 위치 힌트 표시
```

### 3.2 사용자 시나리오

```
[시나리오: 도쿄 여행]

1. 한국인 A, 미국인 B, 일본인 C가 Fryndo로 매칭됨
2. 시부야역 하치코 동상 앞에서 만남 약속
3. A가 AR 안경 착용 후 도착
4. AR에 B, C의 프로필 사진과 "3m 앞" 표시
5. 만남 후 대화 시 실시간 번역 자막 표시
   - C(일본어): "라멘 맛집 추천할게요"
   - → A 안경에 한국어 자막 표시
   - → B 안경에 영어 자막 표시
6. 라멘집 도착 시 NFT 민팅 알림
7. 셋이 함께 "시부야 라멘 NFT" 획득
```

### 4.1 시스템 구성도

```
┌─────────────────────────────────────────────────────────┐
│                    Fryndo AR Companion                  │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  ┌─────────────┐    Bluetooth    ┌─────────────────┐   │
│  │  AR Glasses │ ◄─────────────► │   Mobile App    │   │
│  │  (Meta SDK) │                 │ (React Native)  │   │
│  └─────────────┘                 └────────┬────────┘   │
│        │                                  │            │
│        │ Camera Stream                    │ API        │
│        │ Audio I/O                        │            │
│        ▼                                  ▼            │
│  ┌─────────────┐                 ┌─────────────────┐   │
│  │  On-Device  │                 │  Fryndo Server  │   │
│  │  Processing │                 │    (Koyeb)      │   │
│  └─────────────┘                 └────────┬────────┘   │
│                                           │            │
│                                           ▼            │
│                              ┌────────────────────┐    │
│                              │   External APIs    │    │
│                              │  - Translation     │    │
│                              │  - Maps/POI        │    │
│                              │  - Blockchain      │    │
│                              └────────────────────┘    │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 4.2 기술 스택

| 영역 | 기술 | 선정 이유 |
|------|------|-----------|
| **모바일 앱** | React Native | 크로스 플랫폼, Fryndo 기존 스택 |
| **AR SDK** | Meta Wearables SDK | 가장 큰 시장, 공식 지원 |
| **번역 API** | DeepL API | 높은 품질, 합리적 가격 |
| **음성인식** | Whisper API | 다국어 지원, 정확도 |
| **AR 렌더링** | ARCore/ARKit | 네이티브 성능 |
| **백엔드** | Node.js (Koyeb) | 기존 인프라 활용 |
| **실시간 통신** | WebSocket | 저지연 동기화 |
| **블록체인** | Polygon | NFT 민팅 (기존 연동) |

### 4.3 데이터 플로우

```
[실시간 번역 플로우]

1. AR 안경 마이크 → 음성 캡처
2. Bluetooth → 모바일 앱 전송
3. Whisper API → 음성 → 텍스트
4. DeepL API → 번역
5. WebSocket → 동행자 앱 전송
6. Bluetooth → AR 안경 디스플레이
7. 자막 오버레이 표시

지연시간 목표: < 500ms
```
