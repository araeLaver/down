# TravelMate 창업 실행 체크리스트

> **목표**: 법인 설립부터 서비스 런칭까지 A to Z 체크리스트

---

## WEEK 1-2: 법인 설립

### Step 1: 사전 준비 (Day 1-2)
- [ ] **대표자 인감도장** 제작 (동네 도장집, ₩30,000~)
- [ ] **법인 인감도장** 제작 (동네 도장집, ₩50,000~)
- [ ] **인감증명서** 발급 (주민센터, 2통)
- [ ] **주민등록등본** 발급 (주민센터, 1통)
- [ ] **자본금** 준비 (권장: 1,000만원)

### Step 2: 정관 작성 (Day 2-3)
- [ ] 정관 내용 검토 (`docs/legal/정관_주식회사트래블메이트.md`)
- [ ] 회사명 확정: **주식회사 트래블메이트**
- [ ] 본점 소재지 확정: _______________
- [ ] 자본금 확정: ₩_______________
- [ ] 발기인 정보 기입

### Step 3: 자본금 납입 (Day 3-4)
- [ ] 개인 통장에 자본금 입금
- [ ] **잔액증명서** 발급 (은행 방문 또는 인터넷뱅킹)

### Step 4: 법인 설립 등기 (Day 4-7)

**옵션 A: 직접 등기 (저렴)**
- [ ] 인터넷등기소 접속: https://www.iros.go.kr
- [ ] 설립등기 신청서 작성
- [ ] 필요 서류 업로드
- [ ] 등록면허세 납부 (약 ₩135,000)
- [ ] 등기 완료 확인 (3-5 영업일)

**옵션 B: 대행 서비스 (편리)**
- [ ] 헬프미 또는 법인설립119 가입
- [ ] 정보 입력 및 서류 제출
- [ ] 결제 (₩330,000~)
- [ ] 등기 완료 확인

### Step 5: 사업자 등록 (Day 7-10)
- [ ] 홈택스 접속: https://www.hometax.go.kr
- [ ] 법인 사업자등록 신청
- [ ] 업종코드 입력:
  - 주업종: `639909` (기타 정보서비스업)
  - 부업종: `620200`, `702190`, `479119`
- [ ] 사업자등록증 발급 완료

## WEEK 2-3: 인허가 및 신고

### Step 6: 법인 통장 개설 (Day 10-12)
