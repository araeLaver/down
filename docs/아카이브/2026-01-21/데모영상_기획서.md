## 2. 제작 방식 선택

### 옵션 A: 목업 영상 (추천)

```
장점:
- AR 기능 없이도 제작 가능
- 완성도 높은 비주얼
- Figma/After Effects로 제작

단점:
- 디자인 작업 필요
- 외주 시 비용 발생 (30~50만원)

도구:
- Figma (화면 디자인)
- Protopie / Principle (인터랙션)
- After Effects (최종 편집)
```

### 옵션 B: 실제 앱 + AR 시뮬레이션

```
장점:
- 기존 Fryndo MVP 활용 가능
- 현실감 있음

단점:
- AR 부분은 합성 필요
- 편집 스킬 필요

도구:
- 스마트폰 화면 녹화
- CapCut / Premiere (편집)
- Canva (AR 오버레이 합성)
```

### 옵션 C: 간단 슬라이드 영상

```
장점:
- 가장 빠르게 제작 가능
- 직접 제작 가능

단점:
- 임팩트 낮음

도구:
- Canva 또는 PowerPoint
- 화면 전환 애니메이션
```

## 3. 스토리보드 (45초)

### Scene 1: 인트로 (5초)

```
[화면]
검은 배경 → 로고 페이드인
"Fryndo AR Companion"
"언어가 달라도 함께 여행하는"

[자막]
없음

[음악]
밝은 BGM 시작
```

---

### Scene 2: 상황 설정 (8초)

```
[화면]
도쿄 시부야 거리 영상 (스톡 영상)
→ 스마트폰 들고 있는 손 클로즈업

[자막]
"도쿄에서 만난 외국인 친구"
"언어가 달라서 대화가 어려웠던 적 있으시죠?"

[음악]
BGM 계속
```

### Scene 3: 기능 1 - 그룹 번역 (12초)

```
[화면]

- 미국인 "Hey, let's grab some ramen!"
- → 자동 번역: "라멘 먹으러 가자!"
- 한국인 음성 입력 → 영어로 번역

[자막]
"실시간 그룹 번역"
"3명 이상도 OK"

[효과]
번역되는 순간 하이라이트 효과

[음악]
BGM 계속
```

## 5. 직접 제작 가이드 (Canva 활용)

### Step 1: Canva 프로젝트 생성

```
1. canva.com 접속
2. "동영상" → "모바일 동영상" 선택
3. 9:16 비율 (스마트폰 화면)
```

### Step 2: 장면별 제작

```
[Scene 1: 인트로]
- 배경: 단색 또는 그라데이션
- 텍스트: 로고 + 슬로건
- 애니메이션: 페이드인

[Scene 2: 상황 설정]
- 스톡 영상 삽입
- 텍스트 오버레이

[Scene 3: 그룹 번역]
- 앱 UI 목업 이미지 제작
- 채팅 버블 애니메이션

[Scene 4-5: AR 기능]
- 실제 거리 사진/영상 배경
- AR 요소 (화살표, 마커) 오버레이
- 애니메이션 효과
```

### Step 3: 편집 & 내보내기

```
1. 장면 연결 (전환 효과 추가)
2. BGM 삽입
3. 자막 추가
4. MP4로 내보내기 (1080p)
```
