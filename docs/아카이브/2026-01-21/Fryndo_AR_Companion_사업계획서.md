# Fryndo AR Companion 사업계획서

> AR 기반 여행 동행 보조 서비스 (기초창업지원용)
> 작성일: 2026-01-21

---

## 1. 개요

### 서비스명
**Fryndo AR Companion** - AR 기반 여행 동행 보조 앱

### 비전
"언어 장벽 없이, 전 세계 여행자와 자연스럽게 연결되는 경험"

### 핵심 가치
- 기존 Fryndo(여행 동행 매칭) 서비스의 AR 확장
- 다국적 동행자 간 실시간 번역
- 여행지 AR 가이드 및 내비게이션

---

## 2. 문제 정의

### 해외여행자가 겪는 문제

| 문제 | 현황 |
|------|------|
| 언어 장벽 | 다국적 동행자 간 소통 어려움 |
| 길 찾기 | 낯선 도시에서 헤매는 시간 낭비 |
| 정보 부족 | 현지 맛집/관광지 정보 실시간 확인 어려움 |
| 동행자 찾기 | 만남 장소에서 서로 못 찾는 경우 |

### 기존 솔루션의 한계

- 번역 앱: 1:1만 지원, 그룹 대화 불가
- 지도 앱: 화면을 계속 봐야 함
- 여행 앱: 분리된 기능, 통합 경험 부재

---

## 3. 솔루션

### MVP 핵심 기능 (스마트폰 AR)

#### 기능 1: 실시간 그룹 번역
```
- 다국적 동행자 간 대화 자동 번역
- 음성 인식 → 텍스트 → 번역 → 음성 출력
- 지원 언어: 주요 20개 언어 (MVP)
```

#### 기능 2: AR 동행자 찾기
```
- 만남 장소에서 카메라로 주변 스캔
- 동행자 위치/방향 AR 오버레이
- "저기 3시 방향 15m" 표시
```

#### 기능 3: AR 내비게이션
```
- 도보 방향 화살표 AR 표시
- 화면을 들고 걸으면 길 안내
- 동행자 위치 실시간 공유
```

### 사용자 시나리오

```
[시나리오: 도쿄 여행]

1. 한국인 A, 미국인 B가 Fryndo로 매칭
2. 시부야역 하치코 동상 앞 만남 약속
3. A가 앱 켜고 카메라로 주변 스캔
4. AR로 B의 위치 표시 → 쉽게 찾음
5. 대화 시 실시간 번역
   - B(영어): "I know a good ramen place"
   - → A 화면에 한국어 자막 표시
6. AR 내비게이션으로 함께 이동
```

## 5. 기술 구현

### MVP 기술 스택

| 영역 | 기술 | 비고 |
|------|------|------|
| 모바일 앱 | React Native | 기존 Fryndo 스택 |
| AR | ARCore/ARKit | 스마트폰 기본 지원 |
| 번역 | DeepL API | 높은 품질 |
| 음성인식 | Whisper API | 다국어 정확도 |
| 백엔드 | Node.js (Koyeb) | 기존 인프라 |

- AR Companion은 **Fryndo 확장** 아이템으로 신청
- 기존 Fryndo MVP가 있으므로 "검증된 팀" 어필 가능
- AR 기능은 "향후 개발 계획"으로 제시, 당장 완성 필요 없음
- 예창패는 **아이디어와 실행력** 평가, 완성된 제품 아님
