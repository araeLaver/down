## 필수 스크린샷 (5~7장)

### 1. 메인/랜딩 페이지
```
캡처 포인트:
- 서비스 로고 및 슬로건
- 핵심 가치 제안 (Value Proposition)
- 회원가입/로그인 버튼
```
**파일명**: `01_landing.png`

---

### 2. 회원가입/로그인 화면
```
캡처 포인트:
- 소셜 로그인 버튼 (카카오, 구글)
- 일반 회원가입 폼
- 개인정보 동의 체크박스
```
**파일명**: `02_login.png`

---

### 3. 프로필 설정 화면
```
캡처 포인트:
- 여행 스타일 선택
- MBTI / 관심사 입력
- 프로필 사진 업로드
```
**파일명**: `03_profile.png`

---

### 4. 여행 그룹 목록/검색 화면
```
캡처 포인트:
- 여행지별 그룹 리스트
- 필터/검색 기능
- 그룹 카드 UI (목적지, 일정, 인원)
```
**파일명**: `04_groups.png`

---

### 5. 채팅/매칭 화면
```
캡처 포인트:
- 실시간 채팅 UI
- 그룹 멤버 목록
- 메시지 입력창
```
**파일명**: `05_chat.png`

---

### 6. NFT 수집/지갑 화면

캡처 포인트:

- 블록체인 연동 표시

**파일명**: `06_nft.png`

---

### 7. GPS 위치 인증 화면 (선택)

캡처 포인트:

## 스크린샷 촬영 팁

### 해상도
- **권장**: 1920x1080 또는 모바일 비율 (375x812)
- **형식**: PNG (고화질)

### 촬영 도구
- **Chrome DevTools**: F12 → 모바일 뷰 → 캡처
- **Windows**: Win + Shift + S
- **Mac**: Cmd + Shift + 4

### 주의사항
```
✅ 실제 데이터가 보이도록 (빈 화면 X)
✅ 테스트 계정으로 로그인 후 캡처
✅ 개인정보 모자이크 (실제 사용자 정보)
✅ 에러/로딩 화면 X
```

## 데모 영상 제작 (선택)
