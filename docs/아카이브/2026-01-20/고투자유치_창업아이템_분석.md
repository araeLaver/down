## 아이템 2: AI 문서/계약 분석 솔루션 (Document AI)

### 개요

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  서비스명: 닥스AI (DocsAI)                                       │
│                                                                 │
│  한 줄 소개:                                                     │
│  "AI가 계약서/문서를 분석하고 리스크를 자동 검출"                │
│                                                                 │
│  핵심 기능:                                                      │
│  1. 계약서 자동 분석 (불리 조항, 누락 조항 검출)                 │
│  2. 문서 간 비교 분석 (수정본 vs 원본)                           │
│  3. 표준 계약서 템플릿 자동 생성                                 │
│  4. 규정 준수 체크 (내부 정책, 법률)                             │
│  5. 계약서 데이터베이스 관리                                     │
│                                                                 │
│  기술 스택:                                                      │
│  - LLM (GPT-4/Claude) + Fine-tuning                              │
│  - OCR (문서 스캔)                                               │
│  - RAG (Retrieval Augmented Generation)                          │
│  - React + FastAPI                                               │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 투자 매력도 분석

| 항목 | 점수 | 근거 |
|------|------|------|
| **VC 관심도** | ⭐⭐⭐⭐⭐ | 업스테이지 모델 검증 (1,000억 투자) |
| **정부 지원** | ⭐⭐⭐⭐⭐ | AI + 리걸테크 |
| **시장 규모** | ⭐⭐⭐⭐ | 법률 서비스 시장 + 기업 법무 |
| **수익 모델** | ⭐⭐⭐⭐⭐ | 고단가 B2B SaaS |
| **경쟁 우위** | ⭐⭐⭐⭐⭐ | 한국 법률/계약 특화 |
| **확장성** | ⭐⭐⭐⭐ | 금융, 부동산, 인사 등 확장 |

### 왜 투자 유치에 유리한가?

```
1. 검증된 모델
   - 업스테이지: 문서 AI로 1,000억 시리즈B
   - 글로벌: Ironclad, DocuSign AI

2. 명확한 ROI
   - 법무팀 1명 연봉 = 6,000만원+
   - 계약 검토 시간 90% 절감
   - 기업 즉시 비용 절감 체감

3. 높은 ACV (Annual Contract Value)
   - 대기업 계약: 연 5,000만원~1억원
   - 중견기업: 연 1,000만원~3,000만원

4. 낮은 이탈률
   - 계약서 데이터 축적 → Lock-in
   - 업무 프로세스 의존성
```

### 예상 투자 유치

```
Pre-Seed: 5~10억원 (법률 도메인 전문성 필요)
Seed: 20~30억원 (대기업 고객 확보)
Series A: 100억원+ (스케일업)
```

### 타겟 고객

| 세그먼트 | 규모 | 결정권자 | 예상 ACV |
|----------|------|----------|----------|
| 로펌 | 1,000+ | 변호사 | ₩30M |
| 대기업 법무팀 | 5,000+ | CLO/법무팀장 | ₩50M |
| 중견기업 | 10,000+ | 대표/CFO | ₩12M |
| 스타트업 | 50,000+ | 대표 | ₩3M |
