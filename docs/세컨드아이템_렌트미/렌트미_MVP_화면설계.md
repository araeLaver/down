### 인증 항목 상세

| 인증 항목 | 연동 방식 | 표시 정보 | 신뢰점수 가중치 |
|----------|----------|----------|----------------|
| 재직 인증 | 건강보험 API | 회사명, 재직기간 | +25점 |
| 소득 인증 | 홈택스 API | 연소득 구간 | +25점 |
| 신용 등급 | NICE/올크레딧 | 등급 (1~10) | +20점 |
| 집주인 레퍼런스 | 수동 확인 | 확인 코멘트 | +30점 |

### 이전 집주인 레퍼런스 플로우

```
1. 세입자: 이전 집주인 전화번호 입력
2. 시스템: 집주인에게 SMS 발송 (인증 링크)
3. 집주인: 링크 클릭 → 간단 설문 (5문항)
   - 월세 연체 여부
   - 시설 상태
   - 이웃 민원
   - 퇴실 시 상태
   - 추천 여부
4. 세입자 프로필에 "레퍼런스 확인됨" 배지
```

### AI 자기소개서 생성 로직

```
입력: 기본정보 + 인증정보 + 라이프스타일 답변
처리: GPT API (프롬프트: 집주인 관점 최적화)
출력: 150자 내외 자기소개서

예시:
"IT 회사에서 일하는 28세 직장인입니다.
평일 저녁과 주말에 주로 집에서 시간을 보내며,
조용하게 생활합니다. 비흡연자이고,
2년 이상 거주를 희망합니다.
이전 집에서 2년간 문제 없이 거주했습니다."
```

### 신뢰점수 계산

```
기본 점수: 20점

+ 재직 인증: +25점
+ 소득 인증: +25점
+ 신용 인증: +20점 (1-3등급: +20, 4-6등급: +15, 7등급 이하: +10)
+ 집주인 레퍼런스: +30점 (긍정 평가 시)
- 부정 레퍼런스: -20점

예시: 20 + 25 + 25 + 17(3등급) = 87점
```

## 기술 스택 (MVP)

| 영역 | 기술 | 선정 이유 |
|------|------|----------|
| 프론트엔드 | Next.js + TailwindCSS | 빠른 개발, SEO |
| 백엔드 | Supabase | 인증, DB, 스토리지 통합 |
| AI | OpenAI API | 자기소개서 생성 |
| 인증 API | 공공데이터포털 | 건강보험, 홈택스 |
