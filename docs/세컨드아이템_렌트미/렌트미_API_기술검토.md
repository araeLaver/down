# 렌트미 API 연동 기술 검토서

> 세입자 신뢰 인증을 위한 API 연동 가능성 및 기술 상세 분석

작성일: 2026-01-25
버전: v1.0

---

## 1. 개요

렌트미 서비스의 핵심은 **세입자 신뢰 인증**입니다. 이를 위해 다음 API들의 연동이 필요합니다:

| 인증 항목 | 필요 API | 우선순위 |
|----------|----------|----------|
| 재직 인증 | 건강보험공단 / 4대보험 | P0 |
| 소득 인증 | 홈택스 소득금액증명 | P1 |
| 신용 인증 | NICE / KCB 신용정보 | P2 |
| 본인 인증 | 카카오/PASS 간편인증 | P0 |
| 대학 인증 | 이메일 도메인 검증 | P0 (외국인) |
| 보증보험 | SGI서울보증 | P2 |

## 2. 재직 인증 API

### 2.1 건강보험공단 자격득실확인서

| 항목 | 내용 |
|------|------|
| **API 명칭** | 건강보험 자격득실확인서 조회 |
| **제공처** | 국민건강보험공단 (NHIS) |
| **접근 경로** | 공공데이터포털 / 스크래핑 |
| **인증 방식** | 본인 인증 필수 (공동인증서/간편인증) |
| **조회 정보** | 직장가입자 여부, 사업장명, 자격취득일 |

#### API 응답 예시

```json
{
  "resultCode": "00",
  "data": {
    "qualification": "직장가입자",
    "companyName": "주식회사 OOO",
    "acquisitionDate": "2024-03-01",
    "lossDate": null
  }
}
```

#### 연동 방식 비교

| 방식 | 장점 | 단점 | 비용 |
|------|------|------|------|
| 공공데이터포털 | 무료 | 본인인증 복잡, 기업 전용 | 무료 |
| Codef 스크래핑 | 간편, 실시간 | 월 기본료 | 월 30만원 + 건당 100원 |
| 쿠콘 스크래핑 | 간편 | 비용 높음 | 월 50만원+ |
| 직접 스크래핑 | 저비용 | 유지보수 부담, 법적 리스크 | 개발비만 |

#### MVP 권장: Codef 스크래핑

```
[Codef 연동 흐름]

1. 사용자 → 건강보험공단 로그인 정보 입력
2. Codef → 건강보험공단 스크래핑
3. Codef → 렌트미 API 응답 (자격득실 정보)
4. 렌트미 → 인증 결과 저장 (원본 폐기)
```

**비용**: 기본료 30만원/월 + 건당 100원
**연동 난이도**: ★★★☆☆

## 6. 대학 재학 인증 (외국인 버전)

### 6.1 이메일 도메인 인증

```javascript
// 대학 도메인 화이트리스트
const universityDomains = {
  'yonsei.ac.kr': '연세대학교',
  'korea.ac.kr': '고려대학교',
  'snu.ac.kr': '서울대학교',
  'sogang.ac.kr': '서강대학교',
  'ewha.ac.kr': '이화여자대학교',
  'hanyang.ac.kr': '한양대학교',
  'skku.edu': '성균관대학교',
  'cau.ac.kr': '중앙대학교',
  'hufs.ac.kr': '한국외국어대학교',
  'khu.ac.kr': '경희대학교',
  // ... 200+ 대학
};

// 인증 흐름
async function verifyUniversityEmail(email) {
  const domain = email.split('@')[1];

  if (!universityDomains[domain]) {
    throw new Error('지원하지 않는 대학입니다');
  }

  const code = generateVerificationCode();
  await sendEmail(email, code);

  return {
    university: universityDomains[domain],
    pendingVerification: true
  };
}
```

**연동 난이도**: ★☆☆☆☆
**비용**: 거의 무료 (이메일 발송 비용만)

## 7. 보증보험 연계 API

### 7.1 SGI서울보증

| 항목 | 내용 |
|------|------|
| **서비스명** | 임차보증금 반환보증보험 |
| **제공사** | SGI서울보증 |
| **보험료** | 보증금의 1~3% (신용도 따라) |
| **연동 방식** | B2B 제휴 계약 |

#### 연동 프로세스

```
[보증보험 연계 흐름]

1. 렌트미 ↔ SGI서울보증 제휴 계약 체결
2. 사용자 → 보험 가입 신청
3. 렌트미 → SGI에 신용정보 + 신뢰점수 전달
4. SGI → 보험료 산정 (신뢰점수 반영 할인)
5. 사용자 → 보험료 결제
6. SGI → 보증서 발급
7. 집주인 → 보증서 확인 후 계약
```

#### 렌트미 신뢰점수 연계

| 신뢰점수 | 보험료 할인 | 예시 (보증금 500만원) |
|----------|------------|---------------------|
| 90점 이상 | 30% 할인 | 1% = 5만원 → 3.5만원 |
| 80점 이상 | 20% 할인 | 1.5% = 7.5만원 → 6만원 |
| 70점 이상 | 10% 할인 | 2% = 10만원 → 9만원 |
| 70점 미만 | 할인 없음 | 3% = 15만원 |

**연동 난이도**: ★★★★★
**시기**: Phase 2 (M4 이후)

## 11. 보안 요구사항

| 항목 | 요구사항 | 구현 방안 |
|------|----------|----------|
| **개인정보 암호화** | 민감정보 암호화 저장 | AES-256, 별도 키관리 |

| **전송 암호화** | 통신 암호화 | HTTPS (TLS 1.3) |
