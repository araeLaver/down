## 1. Snug (호주) - 가장 유사한 모델

### 기본 정보

| 항목 | 내용 |
|------|------|
| 설립 | 2017년, 시드니 |
| 창업자 | Justin Butterworth |
| 이전 Exit | Rentahome → Stayz 매각 ($29M, 2011) |
| 사용자 | 150만+ 세입자 지원 |
| 직원 | 20명 |
| 창업자 지분 | 80% 보유 |

### 투자 현황

| 라운드 | 금액 | 투자자 |
|--------|------|--------|
| 자체 투자 | $1M | 창업자 |
| 벤처 | 비공개 | Firemark Ventures, Folklore Ventures |
| 부채 | 비공개 | Tractor Ventures |

### 핵심 서비스

#### 1. Match Score (매칭 점수)
- 집주인 선호도 + 세입자 프로필 + 시장 상황
- AI 기반 최적 매칭
- 양측 만족도 향상

#### 2. BondCover (보증금 대체) ⭐
```
기존: 보증금 4주치 현금 납부 필요
Snug: 보증금의 5~9%만 내면 보증서 발급

예시:
- 보증금 $2,000 필요
- BondCover: $100~180만 납부
- 집주인에게 보증 지급 보장

→ 한국 전세/월세 보증금 문제 해결 힌트!
```

#### 3. Snug Pay
- 월세, 공과금 결제 통합
- 렌탈 히스토리 자동 생성
- 다음 임대 시 신뢰 자산화

### 수익 모델

| 수익원 | 대상 | 비고 |
|--------|------|------|
| SaaS 구독 | 부동산 중개사 | 월 정액 |
| BondCover | 세입자 | 보증금 5~9% |
| Snug Pay | 세입자/집주인 | 결제 수수료 |

### 성장 지표

- ROI per salesperson: 500~900%
- SaaS 수익 사이클: 30일 이내
- 호주 임차 가구 1/3 대상 시장

### 벤치마킹 포인트

1. **BondCover**: 보증금 대체 상품 → 한국 SGI서울보증 연계 가능
2. **렌탈 히스토리**: 결제 기록 → 신뢰 자산 전환
3. **Match Score**: AI 매칭 → 양측 만족도

Sources: [Snug 공식](https://snug.com/), [Tractor Ventures](https://www.tractorventures.com/post/snugs-two-pronged-approach-to-growth-and-transforming-property-sector)

### 기본 정보

| 항목 | 내용 |
|------|------|
| 설립 | 2016년, LA |
| 창업자 | Michael Lucarelli (CEO), Paul Sirisuphang (COO) |

| 서비스 지역 | 미국 50개 주 |

### 투자 현황

| 라운드 | 시기 | 금액 | 주요 투자자 |
|--------|------|------|------------|
| Seed | - | $2.3M | - |
| Series A | 2021.07 | $8M | 645 Ventures, Green Visor Capital |
| Series B | 2022.08 | $17.3M | Green Visor, C.A.R. (캘리포니아 부동산협회) |
| **총 투자** | - | **$28M** | - |
| **기업가치** | 2022 | **$110M** | - |

### 핵심 서비스

#### 세입자 스크리닝
```
신용 조회 (Credit Check)
  └─ TransUnion 데이터
  └─ 신용점수, 부채, 연체 이력

범죄 이력 조회 (Criminal Background)
  └─ 전국 범죄 데이터베이스
  └─ 성범죄자 등록 여부

퇴거 이력 조회 (Eviction History)
  └─ 법원 기록
  └─ 강제 퇴거 여부
```

#### 가격 정책

| 항목 | 가격 | 부담 주체 |
|------|------|----------|
| 플랫폼 사용 | 무료 | 집주인/중개사 |
| 스크리닝 리포트 | $39.99 | 세입자 |

### 차별화 전략

1. **부동산협회 투자**: C.A.R. 투자 유치 → 신뢰도 + 채널 확보
2. **API 연동**: 기존 부동산 플랫폼과 통합
3. **무료 정책**: 집주인 무료 → 세입자 과금

### 벤치마킹 포인트

1. **부동산협회 협력**: 한국부동산협회 또는 대형 중개사 제휴
2. **API First**: 직방/다방 연동 전략
3. **세입자 과금 모델**: 프로필 비용은 세입자가 부담

Sources: [RentSpree](https://www.rentspree.com/), [PYMNTS](https://www.pymnts.com/news/investment-tracker/2022/rentspree-secures-17-3m-to-expand-rental-management-tools/)

### 기본 정보

| 항목 | 내용 |
|------|------|
| 설립 | 2012년 |
| 사용자 | 820만+ |

### 가격 정책 (집주인 중심)

| 서비스 | 가격 |
|--------|------|
| 기본 매물 등록 | 무료 |
| 프리미엄 등록 | £29 |
| 풀 패키지 (테넌트 파인드) | £59 |
| 세입자 레퍼런싱 | £30/인 |

### 세입자 레퍼런싱 상세

```
포함 항목:
- 신용 조회 (Credit Check)
- CCJ 조회 (County Court Judgments)
- 이전 집주인 레퍼런스
- 소득/재직 확인
- 지불 능력 평가 (Affordability)
- 사기 정보 확인
```

### 차별화 포인트

1. **중개 수수료 0%**: 집주인-세입자 직거래
2. **투명한 가격**: 숨은 비용 없음
3. **종합 레퍼런싱**: £30에 모든 조회 포함

### 벤치마킹 포인트

1. **직거래 모델**: 복비 절감 메시지 (피터팬과 유사)
2. **종합 레퍼런싱 패키지**: 한 번에 모든 인증
3. **저가 정책**: 진입장벽 낮춤

Sources: [OpenRent](https://www.openrent.co.uk/), [UK Startup Blog](https://www.ukstartupblog.co.uk/openrent/)

## 5. 일본 시장 참고

### 외국인 임대 현황

| 지표 | 수치 |
|------|------|
| 외국인 임대 거절률 | 약 40% |
| 보증회사 필수 | 월세 30~100% 수수료 |
| 주요 거절 사유 | 언어, 비자, 문화 차이 |

### 외국인 전문 플랫폼

| 서비스 | 특징 |
|--------|------|
| Wagaya Japan | 다국어 (영/중/한/베/필) 지원 |
| Real Estate Japan | 외국인 친화 매물 전문 |
| E-Housing | 외국인 스크리닝 통과 지원 |
| Apts.jp | 사전 스크리닝으로 통과율 높임 |

### 보증회사 (保証会社) 시스템

```
기능: 집주인 대신 세입자 보증
비용: 월세 30~100% (계약 시) + 연간 갱신료
심사: 보증회사 자체 스크리닝

→ 한국 보증보험과 유사하나 더 보편화됨
```

### 벤치마킹 포인트

1. **외국인 전용 시장**: 한국 체류 외국인 250만+ 대상
2. **보증회사 연계**: SGI서울보증 등 협력
3. **다국어 지원**: 영어 필수, 중국어/베트남어 추가

Sources: [Wagaya Japan](https://wagaya-japan.com/en/), [E-Housing](https://e-housing.jp/)
