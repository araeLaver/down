## 2. 외국인 임차 Pain Point

| 문제 | 현황 | 임팩트 |
|------|------|--------|
| **입국 전 계약 불가** | 현지에서만 집보기 가능 | ★★★★★ |
| **보증인 필수** | 한국인 보증인 구하기 어려움 | ★★★★★ |
| **언어 장벽** | 계약서/소통 모두 한국어 | ★★★★☆ |
| **신용기록 無** | 한국 신용점수 없음 | ★★★★☆ |
| **차별** | "외국인 불가" 매물 40%+ | ★★★★☆ |
| **문화 차이** | 전세/월세 개념 혼란 | ★★★☆☆ |
| **사기 피해** | 보증금 미반환, 부당 공제 | ★★★☆☆ |

### 핵심 Pain Point

> **"한국인 보증인이 없어서 집을 구할 수 없다"**

외국인이 한국에서 집을 구할 때 가장 큰 장벽:
1. 한국인 보증인 요구 (법적 필수는 아니지만 관행)
2. 한국 신용기록 없음 → 신뢰 증명 불가
3. 언어 장벽 → 집주인과 소통 어려움
4. 문화 차이 → 전세/월세 개념 혼란

### 핵심 수익원: Deposit Insurance (보증인 대체 서비스)

외국인의 가장 큰 Pain Point = 한국인 보증인 없음
→ 보증보험으로 보증인 대체 서비스 제공
→ 보험사 수수료 2-3% + 우리 마진 2-5% = 총 5-8%

**예시**:
- 보증금 500만원 → 보증인 대체 수수료 25~40만원
- 외국인 입장: 보증인 없이 계약 가능 = 높은 지불의향

## 8. GTM 전략 (Go-to-Market)

### Phase 1: 대학 제휴 (M1-M6)

**타겟**: 서울권 대학교 국제처
- 연세대, 고려대, 서강대, 이화여대, 경희대
- 서울대, 한양대, 성균관대, 중앙대, 외대

**접근법**:
1. 국제처 MOU 체결 → 유학생 필수 앱 등록
2. 학교 이메일 = 자동 인증 (xxx@yonsei.ac.kr)
3. 대학별 검증된 집주인 DB 구축
4. 오리엔테이션 발표 기회 확보

## 9. 한국인 버전과의 시너지

```
┌─────────────────────────────────────────────────────────────────┐
│                    렌트미 통합 플랫폼                            │
├──────────────────────────┬──────────────────────────────────────┤
│    렌트미 (한국인)        │     렌트미 글로벌 (외국인)            │
├──────────────────────────┼──────────────────────────────────────┤
│ • 건강보험/홈택스 인증    │ • 여권/비자/대학 인증                │
│ • 한국어 AI 소개서        │ • 다국어 AI 소개서                   │
│ • SGI 보증보험            │ • 보증인 대체 보험                   │
├──────────────────────────┴──────────────────────────────────────┤
│                       공통 인프라                               │
│  • 집주인 DB (통합)  • 매칭 알고리즘  • 신뢰점수 시스템         │
│  • 집주인 앱 (하나로 양쪽 세입자 관리)                          │
└─────────────────────────────────────────────────────────────────┘
```

### 시너지 포인트

1. **집주인 DB 공유**: 한 번 확보한 집주인 = 양쪽에서 활용
2. **인프라 비용 분산**: 서버, AI, 보험 연계 비용 공유
3. **크로스 마케팅**: 외국인 친화 집주인 ↔ 한국인 우수 세입자
4. **신뢰점수 통합**: 동일한 기준으로 신뢰도 평가
