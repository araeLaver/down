# 렌트미 (RentMe) 사업계획서

> **세입자 신뢰 프로필 플랫폼**
> "좋은 세입자임을 증명하면 보증금이 달라집니다"

작성일: 2026-01-25
버전: v1.0

---

## 1. Problem (문제)

### 1.1 세입자의 문제

| 문제 | 현황 |
|------|------|
| 보증금 부담 | 서울 원룸 전세 평균 2억 1,700만원 |
| 월세 보증금 | 500~1,000만원 (사회초년생에게 큰 부담) |
| 신뢰 증명 불가 | 이전 임대 기록 활용 방법 없음 |
| 청년 무주택 | 73.2% (2023 기준) |

**핵심 Pain Point**: "나는 좋은 세입자인데 증명할 방법이 없다"
- 이전에 문제 없이 살았던 기록 → 활용 불가
- 안정적인 직장/소득 → 집주인이 확인할 방법 없음
- 결국 "돈(보증금)"으로만 신뢰를 담보

### 1.2 집주인의 문제

| 문제 | 현황 |
|------|------|
| 세입자 선별 어려움 | 전세사기 이후 임대인도 신중 |
| 사전 확인 불가 | 월세 연체, 시설 파손, 민원 이력 확인 불가 |
| 리스크 증가 | 무보증/소액보증금 월세 증가 추세 |
| 현재 확인 가능 정보 | 신분증, 재직증명서 정도 |

### 1.3 시장 환경 변화

| 지표 | 수치 | 출처 |
|------|------|------|
| 1인 가구 | 804만 (전체 36.1%) | 통계청 2024 |
| 월세 비중 | 60.2% | 국토부 2025 |
| 무보증 월세 | 서울 200건+/년 | 헤럴드경제 |
| 전세사기 피해자 | 2.5만명+ 공식 인정 | 정부 발표 |

**시장 변화 요약**: 전세 → 월세 전환 가속, 신뢰 기반 거래 니즈 증가

## 2. Solution (솔루션)

### 2.1 핵심 컨셉

> **"세입자의 LinkedIn"** - 신뢰를 자산으로 만드는 플랫폼

### 2.2 서비스 구조

```
┌─────────────────────────────────────────────────────────────┐
│                        렌트미                               │
├────────────────────────┬────────────────────────────────────┤
│      세입자            │           집주인                   │
├────────────────────────┼────────────────────────────────────┤
│ • 신뢰 프로필 생성     │ • 검증된 세입자 탐색               │
│ • 인증 (재직/소득/신용)│ • 조건 매칭 (지역/가격/선호)       │
│ • AI 자기소개서        │ • 프로필 열람 & 연락               │
│ • 이전 집주인 레퍼런스 │ • 리스크 사전 파악                 │
├────────────────────────┴────────────────────────────────────┤
│                    보증보험 연계                            │
│         신뢰점수 기반 → 보증금 50% 절감 가능                │
└─────────────────────────────────────────────────────────────┘
```

### 2.3 핵심 기능

| 기능 | 설명 | 차별화 포인트 |
|------|------|--------------|
| **신뢰 인증** | 재직/소득/신용 API 연동 | 자동화, 위변조 불가 |
| **집주인 레퍼런스** | 이전 집주인 확인 요청 | 한국 최초 |
| **AI 자기소개서** | 라이프스타일 기반 자동 생성 | 집주인 관점 최적화 |
| **매칭 알고리즘** | 조건 + 신뢰점수 기반 | 양측 만족도 향상 |
| **보증보험 연계** | 신뢰점수 → 보증금 절감 | SGI서울보증 등 |

### 2.4 핵심 가치 제안

**세입자 입장**
- "내가 좋은 세입자임을 증명하면 보증금을 낮출 수 있다"
- 신뢰 프로필 → 협상력 상승
- 이전 집주인 레퍼런스 → 차별화
- 보증보험 연계 → 보증금 50% 절감

**집주인 입장**
- "검증된 세입자를 빠르게 찾고 싶다"
- 직장/소득 인증 확인
- 이전 임대 이력 (문제 없었는지)
- 라이프스타일 매칭 (반려동물, 흡연 등)

## 3. Scale (시장 규모)

### 3.1 TAM-SAM-SOM

| 구분 | 규모 | 산출 근거 |
|------|------|----------|
| **TAM** | 약 2조원 | 한국 임차 가구 900만, 연 200만 거래 |
| **SAM** | 약 6,000억원 | 1인 가구 임차 550만, 월세 120만 건/년 |
| **SOM** | 약 30억원 | 수도권 1인 월세 50만 건, 점유율 5% (3년) |

### 3.2 타겟 고객

**1차 타겟: 세입자 (B2C)**

| 세그먼트 | 특징 | Pain Point |
|----------|------|------------|
| 사회초년생 (25-30) | 첫 독립, 보증금 부족 | "보증금 500도 없어" |
| 이직러 (28-35) | 직장 이동, 단기 계약 | "매번 새로 계약 귀찮아" |
| 프리랜서 (25-40) | 소득 불규칙 | "소득 증빙이 어려워" |

**2차 타겟: 집주인 (B2B)**

| 세그먼트 | 특징 | Pain Point |
|----------|------|------------|
| 은퇴 집주인 (55-70) | 월세 수입 의존 | "조용하고 월세 잘 내는 사람" |
| 투자형 집주인 (40-55) | 다주택, 관리 어려움 | "공실 최소화, 문제 세입자 회피" |
| 신축 오피스텔 (30-50) | 직접 관리 | "깨끗하게 쓸 사람" |

## 4. Team (팀)

### 4.1 창업자

| 항목 | 내용 |
|------|------|
| 역할 | 기획 / 개발 / 운영 (1인 린스타트업) |
| 강점 | 풀스택 개발, MVP 빠른 구현 가능 |

### 4.2 1인 창업 전략

**Phase 1: MVP (0-6개월)**
- 대표 1인 개발 + 운영
- 노코드/로우코드 활용 (Supabase, Vercel)
- 외주: 디자인 (크몽)

**Phase 2: PMF 검증 (6-12개월)**
- 마케터 1명 채용 (또는 외주)
- 고객 피드백 기반 개선

**Phase 3: 스케일업 (12개월+)**
- 투자 유치 후 팀 확장
- 개발자 2명, 영업 1명

### 4.3 외부 네트워크

| 분야 | 협력 대상 |
|------|----------|
| 법률 | 부동산 전문 변호사 자문 |
| 금융 | SGI서울보증 보증보험 연계 |
| 마케팅 | 부동산 인플루언서 협업 |
| 제휴 | 셰어하우스 (우주, 맹그로브 등) |

## 5. 비즈니스 모델

### 5.1 수익 구조

| 수익원 | 대상 | 가격 | 예상 비중 |
|--------|------|------|----------|
| 프리미엄 프로필 | 세입자 | 9,900원/월 | 30% |
| 프로필 열람 | 집주인 | 3,900원/건 | 25% |
| 집주인 구독 | 집주인 | 29,900원/월 | 25% |
| 보증보험 수수료 | 보험사 | 3-5% | 20% |

### 5.2 3년 재무 계획

| 항목 | 1년차 | 2년차 | 3년차 |
|------|-------|-------|-------|
| 세입자 가입 | 10,000명 | 50,000명 | 150,000명 |
| 유료 전환율 | 10% | 15% | 20% |
| 월 매출 | 500만원 | 3,000만원 | 1.5억원 |
| 연 매출 | 6,000만원 | 3.6억원 | 18억원 |

### 5.3 Unit Economics

```
[세입자]
CAC: 5,000원 (SNS 마케팅)
LTV: 59,400원 (9,900원 × 6개월 평균)
LTV:CAC = 약 12:1 ✅

[집주인]
CAC: 20,000원 (타겟 광고)
LTV: 179,400원 (29,900원 × 6개월)
LTV:CAC = 약 9:1 ✅
```

## 6. 실행 로드맵

### 6.1 6개월 마일스톤

| 기간 | 목표 | 세부 내용 |
|------|------|----------|
| M1-2 | MVP 개발 | 프로필 생성, API 연동, AI 자기소개서 |
| M3-4 | 베타 테스트 | 셰어하우스 B2B 제휴, 500명 확보 |
| M5-6 | 정식 런칭 | 집주인 기능, 보증보험 연계, 마케팅 |

### 6.2 KPI

| 지표 | 6개월 | 12개월 |
|------|-------|--------|
| 세입자 가입 | 3,000명 | 10,000명 |
| 프로필 완성률 | 60% | 70% |
| 유료 전환율 | 8% | 12% |
| 집주인 가입 | 200명 | 1,000명 |
| 매칭 성사 | 50건 | 300건 |

### 6.3 진입 전략

```
[1단계] 셰어하우스/코리빙
   - 자기소개 문화 이미 존재
   - B2B 제휴 (우주, 맹그로브 등)

[2단계] 대학가 원룸
   - 학생 = 집주인 불안 요소
   - 대학 인증 프로필

[3단계] 신축 오피스텔
   - 집주인 직접 관리 많음
   - 프리미엄 세입자 니즈

[4단계] 일반 월세 시장
   - 문화 변화 확산
   - 직방/다방 연동 or 경쟁
```

## 7. 경쟁 분석

### 7.1 경쟁 구도

| 서비스 | 포지션 | 강점 | 약점 |
|--------|--------|------|------|
| 직방/다방 | 매물 검색 | 트래픽, 인지도 | 세입자 프로필 無 |
| 피터팬 | 직거래 | 복비 절감 | 신뢰 인증 無 |
| 네이버부동산 | 정보 제공 | 트래픽 | 매칭 기능 無 |
| **렌트미** | **세입자 신뢰** | **인증 시스템** | 인지도 부족 |

### 7.2 차별화 포인트

1. **세입자 관점 최초**: 기존은 매물 중심, 렌트미는 세입자 프로필 중심
2. **신뢰 인증 시스템**: API 연동 자동 인증, 이전 집주인 레퍼런스 (한국 최초)
3. **보증금 절감 연계**: 신뢰점수 → 보증보험 → 보증금 50% 절감
