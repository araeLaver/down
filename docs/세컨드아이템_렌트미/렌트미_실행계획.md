## 5. Phase 3: 시장 진입 (7~9월)

### 5.1 진입 전략: 셰어하우스 먼저

**왜 셰어하우스인가?**
```
1. 자기소개 문화 이미 존재 → 서비스 수용도 높음
2. 운영사가 명확 → B2B 협의 가능
3. 타겟 고객 집중 → 마케팅 효율
4. 성공 사례 → 일반 시장 확장 근거
```

**타겟 셰어하우스 업체**
| 업체 | 규모 | 접근 방법 |
|------|------|----------|
| 우주 (WOOZOO) | 국내 1위 | 콜드메일 + 네트워킹 |
| 맹그로브 | 코리빙 | 콜드메일 |
| 커먼타운 | 프리미엄 | 콜드메일 |
| 로컬스티치 | 중소형 | 직접 방문 |

**제안 내용**
```
[셰어하우스 운영사 제안]

• 문제: 입주자 선별에 시간 소요, 문제 입주자 리스크
• 솔루션: 렌트미 프로필로 사전 검증
• 혜택:
  - 첫 6개월 무료 사용
  - 입주자 프로필 열람 무제한
  - 매칭 성사 시에만 소정의 수수료
• 요청: 입주 희망자에게 렌트미 프로필 생성 권유
```
