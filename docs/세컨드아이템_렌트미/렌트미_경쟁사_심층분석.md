## 2. 직방 (Zigbang) 분석

### 2.1 기업 개요

| 항목 | 내용 |
|------|------|
| **설립** | 2010년 |
| **누적 투자** | 약 2,000억원+ |
| **기업가치** | 약 3조원 (2021년 기준) |
| **MAU** | 약 500만명 |
| **직원 수** | 약 600명 |
| **주요 인수** | 호갱노노(2020), 삼성SDS 홈IoT(2021) |

### 2.2 서비스 구조

```
[핵심 서비스]
├── 원룸/투룸 매물 검색
├── 아파트 매물 검색 (호갱노노 흡수)
├── 오피스텔/빌라 검색
└── 신축분양 정보

[부가 서비스]
├── 3D 홈투어
├── 시세 정보
├── 안심중개사
└── 직방 Pay (보증금 결제)

[B2B 서비스]
├── 중개사 광고 플랫폼
└── 건물주/분양사 광고
```

### 2.3 수익 모델

| 수익원 | 비중 | 상세 |
|--------|------|------|
| **중개사 광고** | 70% | 매물 상위 노출, 프리미엄 광고 |
| **분양 광고** | 20% | 신축 아파트 광고 |
| **기타** | 10% | 시세 API, 기업 서비스 |

**월 광고비 (중개사 기준)**:
- 기본: 5만원~
- 프리미엄: 30만원~
- 지역 1위: 100만원+

### 2.4 세입자 관련 기능

| 기능 | 유무 | 상세 |
|------|------|------|
| 세입자 프로필 | ❌ | 없음 |
| 세입자 인증 | ❌ | 없음 |
| 집주인 레퍼런스 | ❌ | 없음 |
| 신뢰점수 | ❌ | 없음 |
| 보증보험 연계 | ⚠️ | 직방Pay (전세대출 연계만) |

### 2.5 강점/약점

| 강점 | 약점 |
|------|------|
| 압도적 MAU (500만) | 매물 중심, 세입자 무관심 |
| 높은 브랜드 인지도 | 중개사 광고 의존 수익 |
| 3D 홈투어 기술 | 세입자 차별화 불가 |
| 인수합병 능력 | 신뢰 인증 시스템 無 |

```
                    매물 중심
                       ↑
                       │
                  직방 ●  ● 다방
                       │
                       │
     중개사 ←─────────┼─────────→ 직거래
     연결               │              ● 피터팬
                       │
                       │
                       │     ★ 렌트미
                       │
                       ↓
                   세입자 중심
```

**렌트미 포지션**: 세입자 중심 × 직거래 영역
- 기존 플랫폼이 진출하지 않은 블루오션
- 매물 제공 없이 세입자 신뢰로 승부

## 8. 경쟁사 진입 장벽

### 8.1 직방/다방이 세입자 프로필 도입 시

| 장벽 | 상세 |
|------|------|
| **비즈니스 모델 충돌** | 중개사 광고 수익 vs 세입자 서비스 |
| **기술 투자 필요** | 인증 API 연동, 신뢰점수 알고리즘 |
| **문화 변화** | 기존 사용자 = 매물 검색 기대 |
| **데이터 축적** | 레퍼런스 데이터 = 시간 필요 |

**예상**: 2-3년 내 유사 기능 도입 가능
→ 렌트미는 그 전에 데이터 선점 필요

### 8.2 렌트미 선점 전략

```
[데이터 해자 구축]

1. 집주인 레퍼런스 DB 선점
   - 이전 집주인 평가 데이터 = 축적 시간 필요
   - 경쟁사가 따라오기 어려움

2. 신뢰점수 알고리즘 고도화
   - 인증 + 레퍼런스 + 활동 데이터 결합
   - 시간이 지날수록 정확도 향상

3. 보증보험 제휴 선점
   - SGI서울보증 독점 제휴 시도
   - 신뢰점수 연계 할인 = 차별화

4. 커뮤니티 구축
   - "좋은 세입자" 커뮤니티
   - 정보 공유 + 네트워크 효과
```

```
직방 또는 다방이 렌트미 인수
├── 가치: 신뢰 인증 기술 + 데이터
├── 렌트미 입장: Exit 옵션
```
