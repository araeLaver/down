## 2. 타겟 시장 분석 (TAM-SAM-SOM)

### 2.1 TAM (Total Addressable Market)

**전체 임대차 시장: 약 2조원/년**

```
산출 근거:
- 연간 임대차 거래: 200만 건
- 거래당 평균 서비스 가치: 10만원 (중개, 보험, 인증 등)
- 200만 × 10만원 = 2,000억원 (직접 시장)
- 관련 부가 서비스 포함: 약 2조원
```

### 2.2 SAM (Serviceable Addressable Market)

**1인 가구 월세 시장: 약 6,000억원/년**

```
산출 근거:
- 1인 가구 임차: 550만 가구
- 연간 이동률: 22% (약 120만 건 거래)
- 월세 비중: 70% (84만 건)
- 거래당 서비스 가치: 7만원
- 84만 × 7만원 = 약 600억원 (직접)
- 보험/인증/매칭 포함: 약 6,000억원
```

### 2.3 SOM (Serviceable Obtainable Market)

**3년 목표 시장: 약 30억원/년**

```
산출 근거:
- 수도권 1인 월세 거래: 연 50만 건
- 목표 점유율: 5% (25,000건)
- 건당 매출: 12,000원 (세입자+집주인 평균)
- 25,000 × 12,000 = 3억원 (직접 매출)
- 구독+보험 수수료 포함: 약 30억원
```

### 2.4 시장 성장성

| 동인 | 영향 |
|------|------|
| 1인 가구 증가 | 연 3% 성장 |
| 월세 전환 가속 | 연 5% 성장 |
| 신뢰 기반 거래 문화 | 신규 시장 창출 |
| 정부 규제 강화 | 인증 서비스 필요성 증가 |

**예상 시장 성장률: 연 10-15%**

### 5.2 전략 도출

#### SO 전략 (강점-기회)
- 월세 시장 성장에 맞춰 세입자 프로필 표준화 선점
- 전세사기 공포 활용한 신뢰 인증 마케팅

#### WO 전략 (약점-기회)
- B2B 제휴로 초기 네트워크 효과 확보 (셰어하우스)
- 바이럴 마케팅으로 낮은 예산 극복

#### ST 전략 (강점-위협)
- 빠른 실행으로 1-2년 선점
- 데이터 축적으로 전환비용 형성

#### WT 전략 (약점-위협)
- 틈새시장(셰어하우스) 집중으로 직접 경쟁 회피
- 개인정보 최소 수집 + 동의 기반 운영

```

세입자 → 고유코드 발급 → 이전 집주인 전달
                              ↓
                         간단 평가 작성
                         (5분, 익명 가능)
                              ↓
                         신뢰점수 반영

→ 미국에서는 일반화, 한국에서는 최초 시도
```

#### (3) 보증금 절감 연계

```
신뢰점수 80점+ → SGI서울보증 연계 → 보증금 50% 절감

예시:
기존 보증금: 1,000만원
렌트미 사용: 500만원 + 보증보험료 월 2만원

→ 세입자에게 명확한 금전적 혜택
```

### 6.2 경쟁 장벽 (Moat)

| 장벽 | 설명 | 형성 시점 |
|------|------|----------|
| 데이터 | 세입자 신뢰 데이터 축적 | 1년차 |
| 네트워크 | 세입자-집주인 양면 네트워크 | 2년차 |
| 브랜드 | "신뢰 프로필 = 렌트미" 인식 | 2-3년차 |
| 제휴 | 보험사, 셰어하우스 독점 계약 | 1-2년차 |

### 6.3 직방/다방 대응 전략

**시나리오**: 직방이 세입자 프로필 기능 추가

| 대응 | 내용 |
|------|------|

## 9. 핵심 인사이트

### 9.1 시장 기회

```
1. 월세 시장 60% 돌파 → 신뢰 기반 거래 필수
2. 전세사기 트라우마 → 양측 모두 검증 원함
3. 무보증 월세 등장 → 신뢰 > 돈 문화 태동
4. 1인 가구 804만 → 잦은 이사, 반복 Pain Point
```

### 9.2 경쟁 기회

```
1. 기존 서비스 모두 "매물 중심" → "사람 중심" 블루오션
2. 세입자 프로필/인증 서비스 전무 → 최초 진입자 이점
3. 글로벌에서는 검증된 모델 → 한국 적용 타이밍
```

### 9.3 실행 방향

```
1. 셰어하우스 B2B로 초기 사용자 확보
2. 집주인 레퍼런스로 차별화 (한국 최초)
3. 보증보험 연계로 명확한 혜택 제공
4. 1-2년 선점으로 네트워크 효과 + 데이터 장벽
```
