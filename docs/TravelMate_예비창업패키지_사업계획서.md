# PART 1. Problem (문제 인식)

## 1-1. 창업 아이템 개발 동기

### 개인적 경험에서 시작된 문제 인식

저는 2024년 동남아 배낭여행 중 혼자 여행하는 많은 한국인들을 만났습니다. 그들 대부분이 공통적으로 토로한 고민이 있었습니다:

> "혼자 여행은 자유롭지만, 가끔은 같이 밥 먹을 사람이 그립다"
> "여성 혼자 밤에 다니기 무섭다"
> "좋은 경치를 봐도 공유할 사람이 없으니 허전하다"

기존 여행 플랫폼(트립어드바이저, 에어비앤비)은 숙소와 관광지 정보는 많지만, **함께 여행할 동반자를 찾는 기능**은 전무했습니다.

### 시장 조사를 통한 문제 확인

| 조사 항목 | 결과 | 출처 |
|-----------|------|------|
| 1인 여행 경험자 중 "동행 있으면 좋겠다" 응답 | **67%** | 한국관광공사, 2024 |
| 여성 1인 여행자 "안전이 가장 큰 걱정" | **78%** | Booking.com, 2024 |
| 여행 후 "기록 남기고 싶다" 응답 | **82%** | 자체 설문 (n=200) |
| NFT에 관심 있는 MZ세대 | **41%** | 대학내일20대연구소, 2024 |

## 1-2. 문제 정의

### 핵심 문제 3가지

```
┌─────────────────────────────────────────────────────────────────┐
│                     1인 여행자의 3대 Pain Point                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  [Problem 1]         [Problem 2]         [Problem 3]           │
│   외로움/고립감        안전 우려            여행 기록 휘발         │
│                                                                 │
│  • 식사 혼자 하기 불편   • 밤 이동 불안        • SNS 사진만 남음     │
│  • 사진 찍어줄 사람 無   • 응급상황 대처 어려움  • 시간 지나면 잊혀짐   │
│  • 현지인 교류 한계     • 특히 여성 취약       • 방문 인증 불가       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### Problem 1: 1인 여행자의 외로움과 고립감

**현상**
- 혼자 여행하고 싶지만, 24시간 내내 혼자인 것은 부담
- 맛집에서 혼밥하기 어색함, 2인 이상 메뉴 주문 불가
- 인생샷 찍고 싶어도 부탁할 사람 없음
- 현지인과 교류하고 싶지만 언어/문화 장벽

**정량적 근거**
- 1인 여행 후 "외로웠다" 응답: 54% (트립어드바이저, 2024)
- 혼밥 스트레스로 여행 만족도 하락: 평균 -1.2점/5점

### Problem 2: 안전에 대한 불안

**현상**
- 여성 1인 여행자의 야간 이동 불안
- 응급상황 시 도움 요청 어려움
- 소지품 분실/도난 시 대처 곤란
- 여행 중 건강 이상 시 고립 위험

**정량적 근거**
- 여성 1인 여행자 78%가 "안전이 최대 걱정" (Booking.com)
- 1인 여행 중 범죄 피해 경험: 12% (한국관광공사)
- 안전 우려로 여행 취소한 경험: 34%

### Problem 3: 여행 기록의 휘발성

**현상**
- SNS 사진은 시간이 지나면 묻힘
- "내가 정말 거기 갔었나?" 증명 어려움
- 여행 기록을 체계적으로 보관하기 어려움
- 특별한 여행 경험을 자산화할 수 없음

**정량적 근거**
- 여행 후 1년 뒤 구체적 기억 유지율: 23%
- 여행 기록 앱 사용자 중 이탈률: 67% (3개월 내)
- "여행을 수집하고 싶다" 응답: 71% (자체 설문)

   ↓                      ↓                      ↓

## 4-3. 창업자의 열정과 비전

### 왜 이 문제를 해결하고 싶은가?

저는 2024년 혼자 떠난 동남아 여행에서 수많은 1인 여행자들을 만났습니다. 호스텔 로비에서, 야시장에서, 관광지에서 우연히 마주친 한국인들과 함께 밥을 먹고, 사진을 찍고, 이야기를 나눴습니다.

그 순간들이 여행의 하이라이트였습니다.

하지만 돌아와서 깨달았습니다. 그 만남들은 전부 "우연"이었다는 것을. 우연이 아닌, **계획된 만남**을 만들 수 있다면 어떨까요?

TravelMate는 그 물음에 대한 답입니다.

### 3년 후 비전

> "2029년, TravelMate는 아시아 1인 여행자 100만 명이 사용하는 플랫폼이 됩니다.
> 매일 1,000명의 여행자가 TravelMate를 통해 동행을 찾고,
> 10,000개의 여행 NFT가 발행됩니다.
>
> 혼자여도 괜찮은 여행,
> 하지만 함께라면 더 좋은 여행.
>
> TravelMate가 만들어갑니다."
