# 모두의 창업 지원아이디어 요약본

## 기본 정보

| 항목 | 내용 |
|---|---|
| 트랙 | 일반/기술 |
| 지원지역 | 경기 |
| 사업분야 | 프롭테크 |
| 멘토기관 | 성남산업진흥원 |
| 아이디어 | 세입자가 신뢰도를 증명하고, 집주인이 검증된 세입자를 찾을 수 있는 세입자 프로필 매칭 플랫폼 |

## 한줄 설명

세입자가 재직, 소득, 신용 인증과 이전 집주인 레퍼런스로 신뢰도를 증명하고, 집주인이 검증된 세입자를 탐색·연락할 수 있는 프롭테크 플랫폼.

## 문제

한국 임대차 시장에서 집주인은 세입자를 판단할 객관적 기준이 부족하고, 세입자는 자신이 좋은 세입자임을 증명할 수단이 없다. 이로 인해 집주인은 월세 미납, 시설 훼손, 이웃 민원 같은 리스크를 감수해야 하고, 성실한 세입자는 과거 거주 이력이나 신뢰도를 다음 거래에서 활용하지 못한다.

## 고객

세입자:

- 좋은 세입자임을 증명할 방법이 없는 사람
- 이전 임대 이력이나 성실한 거주 기록을 객관적으로 보여주고 싶은 사람
- 사회초년생, 이직자, 프리랜서 등 주거 신뢰 증명이 필요한 사람

집주인:

- 직감이 아니라 객관적 정보로 세입자를 선별하고 싶은 임대인
- 월세 미납, 시설 훼손, 민원 리스크를 사전에 낮추고 싶은 임대인
- 개별 재직·소득·신용 확인을 직접 하기 어려운 임대인

## 해결 방식

- 세입자 프로필 생성
- 재직, 소득, 신용 인증 기반 Trust Score 산정
- 이전 집주인 레퍼런스 요청 및 수집
- 집주인용 검증 세입자 탐색
- 매물 관리 및 실시간 메시지
- 카카오/네이버 소셜 로그인
- 리뷰 시스템

## 현재 진행상황

- Next.js + PostgreSQL 기반 웹 플랫폼 구축 완료
- 세입자 프로필 시스템 구현
- 신뢰 점수 알고리즘 구현
- 인증 체계 구현
- 이전 집주인 레퍼런스 요청·수집 시스템 구현
- 집주인용 세입자 탐색, 매물 관리, 실시간 메시지 구현
- Koyeb 클라우드 배포 완료

## 향후 계획

1. 대학가·원룸 밀집 지역 중심 베타 테스트
2. 집주인 대상 프리미엄 구독 모델 검증
3. NICE 본인인증 등 실명 인증 고도화
4. 부동산 중개 플랫폼 및 전·월세 보증보험 연계 검토

## 심사 대응 포인트

- 임대차 시장의 정보 비대칭 문제를 해결하는 명확한 프롭테크 아이템
- 단순 아이디어가 아니라 MVP가 이미 구축된 상태
- 세입자와 집주인 양면 시장 모두의 문제를 해결
- 인증, 레퍼런스, Trust Score를 결합해 차별화 가능
- 초기 시장을 대학가·원룸 밀집 지역으로 좁혀 실험 가능
