# 창업활동자금(200만원) 집행 및 검증 계획서

**프로젝트명:** 세입자 신뢰도 기반 임대 매칭 플랫폼
**수행기간:** 2026. 06. 15 ~ 2026. 08. 31
**작성일:** 2026. 06. 09

## 1. 개요
본 계획은 '모두의 창업(1기)' 1라운드 진출자에게 제공되는 창업활동자금 200만 원을 활용하여, 비즈니스 모델의 핵심 가설을 검증하고 2라운드 진출을 위한 유의미한 지표(Traction)를 확보하는 데 목적이 있음.

## 2. 예산 집행 세부 계획

| 항목 | 세부 내용 | 예산(원) | 비고 |
| :--- | :--- | :--- | :--- |
| **사용자 유입 강화** | 인스타그램/페이스북 타겟 광고 (수도권 대학가/원룸촌 세입자) | 1,000,000 | 가입자 및 프로필 생성 수 극대화 |
| **보안 및 인프라** | 자체 데이터 암호화 서버 구축 및 클라우드(Koyeb) 유료 운영 | 400,000 | 자체 스크래핑 데이터 보안 강화 |
| **PMF 검증** | 실거주자 및 임대인 FGI(심층인터뷰) 리워드 (30명) | 300,000 | 사용자 피드백 보고서 작성용 |
| **신뢰도 강화** | 개인정보 처리방침 및 이용약관 법률 검토 (전문가 자문) | 300,000 | 신용정보 취급 리스크 관리 |
| **합계** | | **2,000,000** | |

## 3. 주요 마일스톤 (6~8월)
- **6월 3주:** 창업활동자금 카드 수령 및 API 업체 계약
- **6월 4주:** 법률 검토 완료 및 이용약관 반영, AI 솔루션 신청
- **7월 1~2주:** 본인인증 시스템 연동 완료 및 베타 테스트 개시
- **7월 3주~8월 2주:** SNS 광고 집행 및 사용자 인터뷰(FGI) 병행
- **8월 3주:** 사용자 피드백 데이터 분석 및 2라운드 평가 대응 보고서 작성

## 4. 기대 효과
- **기술적 지표:** 실제 본인인증이 완료된 '검증된 세입자 프로필' 100건 이상 확보
- **사업적 지표:** 임대인의 세입자 탐색 및 매칭 성공 사례(Beta) 5건 이상 창출
- **평가 대응:** 단순 지출이 아닌 '비즈니스 마일스톤 달성'을 증빙하여 2라운드 진출 가능성 극대화
