| 항목 | 작성값 |

| 제출일 | 2026. 5. 7. |
| 도전자명 | 김다운 |

## 예비창업자 기본 제출 구성

[양식2]에서 아래 순서로 증빙자료를 붙인다.

1. 신분증 사본
2. 국세 납세증명서
3. 지방세 납세증명서
4. 신용정보 조회서
5. 사업자등록사실여부 사실증명

## 각 증빙자료 페이지에 붙일 메모

### 1. 신분증 사본

- 주민등록증 또는 운전면허증 사본
- 주민등록번호 뒷자리, 주소 등 불필요 민감정보는 제출처 요구 수준에 맞춰 마스킹

### 2. 국세 납세증명서

- 유효기간 내 발급본
- 발급 대상: 신청자 본인

### 3. 지방세 납세증명서

- 유효기간 내 발급본
- 발급 대상: 신청자 본인

### 4. 신용정보 조회서

- 2026-03-26 이후 발급본
- 채무불이행 규제 여부 확인 목적

### 5. 사업자등록사실여부 사실증명

- 2026-03-26 이후 발급본
- 공고일 기준 예비창업자 여부 확인 목적

## 사업자 이력이 있는 경우 추가 구성

사업자등록 이력이 있다면 `사업자등록사실여부 사실증명` 대신 또는 추가로 아래를 붙인다.

1. 총사업자등록내역 사실증명
2. 폐업사실증명원, 폐업 이력이 있는 경우

## 부동산임대업 보유자 추가 구성

부동산임대업 개인사업자만 보유한 경우 아래를 추가한다.

1. 총사업자등록내역 사실증명
2. 사업자등록증
3. 4대사회보험 가입내역 확인서
4. 건강보험 자격득실 확인서

## 기창업자 구성

기창업자로 제출하는 경우 아래를 붙인다.

공통:

1. 신분증 사본
2. 국세 납세증명서, 신청자 및 기업 모두
3. 지방세 납세증명서, 신청자 및 기업 모두
4. 신용정보 조회서

업력 확인:

1. 창업기업 확인서
2. 사업자등록증
3. 법인 등기부등본, 법인인 경우

이종창업 여부 확인:

1. 홈택스 조회화면
2. 현재 보유 사업 업종코드와 창업 예정 업종코드가 보이는 자료
