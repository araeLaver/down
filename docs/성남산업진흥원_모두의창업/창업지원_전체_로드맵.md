## 2단계: 핵심 가치 구현 (Strategic Execution / 6.17 ~ 7.15)
본인의 강점인 "기술력"을 증명하여 2라운드 진출 명분을 만드는 단계입니다.

*   **[기술] 자체 스크래핑 엔진 고도화**
    *   **Goal:** 홈택스/건강보험공단 데이터 연동 안정화.
    *   **Status:** 현재 보유한 기술력을 입주해 플랫폼에 완전히 녹여내기.
*   **[법률] 이용약관 및 보안 체계 수립**
    *   **Goal:** 자체 수집 데이터의 법적 정당성 확보. (예산 30만원 활용)
*   **[마케팅] 초기 가입자 확보(Traction)**
    *   **Goal:** 대학가/원룸촌 타겟 광고 집행. (예산 100만원 활용)
    *   **KPI:** 실제 'Trust Score'를 생성한 세입자 프로필 100건 확보.

## 3단계: 시장 검증 및 피드백 (Validation / 7.16 ~ 8.15)

*   **[FGI] 사용자 심층 인터뷰 실시**
    *   **Goal:** 집주인/세입자 각 15명 이상 인터뷰. (리워드 예산 30만원 활용)
    *   **Output:** "집주인의 80%가 Trust Score 도입 시 월세 조정 의향이 있음"과 같은 강력한 데이터 추출.
*   **[서비스] 역방향 매칭 베타 테스트**
    *   **Goal:** 실제 집주인이 세입자에게 연락하는 사례(Match) 발생시키기.
