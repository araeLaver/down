# [양식1] 도전자 신청자격 및 요건검토 사전 확인서 작성초안

## 기본정보

| 항목 | 작성값 |
|---|---|
| 도전자명 | 김다운 |
| 생년월일 | 직접 입력 |
| 신청구분 | 예비창업자 |
| 기업명 | 공란 |
| 법인번호 | 공란 |
| 사업자번호 | 공란 |
| 한줄아이디어 | 세입자가 신뢰도를 증명하고, 집주인이 검증된 세입자를 찾을 수 있는 세입자 프로필 매칭 플랫폼 |

## 자격 확인 체크 초안

아래 내용이 사실이면 [양식1]의 `예` 칸에 체크한다. 하나라도 사실과 다르면 제출 전에 수정 또는 문의가 필요하다.

| 항목 | 체크 | 확인 내용 |
|---|---:|---|
| 신청자격 | 예 | 공고일 2026-03-26 기준 사업자등록 및 법인 설립 등기를 하지 않은 예비창업자 |
| 채무 불이행 | 예 | 금융기관 등으로부터 채무불이행으로 규제 중인 자에 해당하지 않음 |
| 체납 | 예 | 국세 또는 지방세 체납 중이 아님 |
| 지원제외 업종 | 예 | 지원제외 업종에 해당하지 않음 |
| 동시수행 불가사업 | 예 | 2026년 동시수행 불가 창업지원사업을 수행 중이지 않음 |
| 수행대상 제외 | 예 | 보조금 관리에 관한 법률상 보조사업 수행 배제 대상이 아님 |

## 신청자격 문구 확인

예비창업자 기준:

공고일 2026-03-26 포함 기준 사업자등록 및 법인 설립 등기를 하지 않았거나, 부동산임대업 개인사업자만 보유한 경우여야 한다.

부동산임대업 개인사업자만 보유한 경우에는 아래 3개 요건을 모두 충족해야 한다.

- 공고일 기준 직원을 고용하지 않고 부동산임대업만 영위
- 사업자등록증상 부동산임대업 외 타 업종을 복수로 영위하지 않음
- 향후 이종 업종의 제품 또는 서비스를 영위하는 법인사업자로 창업 가능

창업기업 기준으로 제출하는 경우:

- 중소기업 대표자여야 함
- 사업 개시일 또는 법인설립일이 2023-03-27부터 2026-03-26까지여야 함
- 현재 업종과 창업 예정 업종이 이종창업이어야 함
- 현재 보유중인 사업의 한국표준산업분류 5자리 업종코드 입력 필요
- 창업 예정 업종의 한국표준산업분류 5자리 업종코드 입력 필요

## 지원 아이디어 문장

짧게 넣어야 하는 칸에는 아래 문장을 사용한다.

> 세입자가 신뢰도를 증명하고, 집주인이 검증된 세입자를 찾을 수 있는 세입자 프로필 매칭 플랫폼

조금 더 설명 가능한 칸이 있으면 아래 문장을 사용한다.

> 재직, 소득, 신용 인증과 이전 집주인 레퍼런스를 기반으로 세입자의 신뢰 프로필과 Trust Score를 만들고, 집주인이 검증된 세입자를 탐색·연락할 수 있게 하는 프롭테크 플랫폼

## 제출 전 직접 입력해야 하는 값

- 생년월일
- 서명 또는 도장
- 실제 사업자등록 여부에 따른 신청구분 최종 확인
- 기창업자인 경우 기업명, 법인번호, 사업자번호, 업종코드
