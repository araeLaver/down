# 검수결과 및 리스크

작성일: 2026-05-04

## 검수 결론

완전 검수 완료가 아니라 `부분 검수 완료` 상태다.

확실히 확인한 자료:

- 메일 본문: 제출 요청, 담당자, 마감 확인
- 신청서내용 RTF: 지원 아이디어, 지원지역, 사업분야, 현재 진행상황 확인
- [양식1] HWPX: 신청자격 확인서 항목 확인
- [양식2] HWPX: 증빙자료 구성 확인
- 참고 PDF: 지원제외 업종, 동시수행 불가사업, 창업여부 기준 확인

미검수 자료:

- `★모두의 창업 프로젝트 신속심사 추진 안내문.hwp`
- 사유: 현재 작업 환경에서 HWP 5.x 파일 본문 텍스트 추출 실패. `textutil`, LibreOffice 계열 도구, `hwp5txt` 사용 불가.

따라서 제출 전 한글 또는 한컴오피스에서 안내문 HWP를 직접 열어 아래 항목을 최종 확인해야 한다.

## 최우선 리스크

### 1. 예비창업자 여부

내가 작성한 문서는 `예비창업자` 기준이다. 하지만 저장소 안에 `다운랩` 사업자등록 준비 문서가 있으므로 실제 사업자등록 여부를 반드시 확인해야 한다.

확인 기준:

- 2026-03-26 포함 기준 사업자등록 또는 법인 설립등기가 없었다면: 예비창업자
- 2026-03-26 포함 기준 개인사업자 또는 법인이 있었다면: 기창업자 가능성 있음
- 부동산임대업 개인사업자만 있었다면: 별도 예비창업자 인정 요건 확인 필요

확인 방법:

- 홈택스에서 `사업자등록사실여부 사실증명` 또는 `총사업자등록내역 사실증명` 발급
- 증명서상 공고일 기준 사업자 보유 여부 확인

### 2. 동시수행 불가사업

2026년에 아래 창업지원사업을 수행 중이면 동시수행 불가 가능성이 있다.

- 예비창업패키지
- 초기창업패키지
- 창업도약패키지
- TIPS 창업사업화
- 프리팁스
- 포스트팁스
- 창업중심대학
- 재도전성공패키지
- 청년창업사관학교 등 참고 PDF 목록에 있는 사업

현재 저장소에는 여러 창업지원 준비 문서가 있지만, 준비 문서만으로는 수행 중 여부를 판단할 수 없다. 실제 협약 체결 또는 선정 후 수행 중인 사업이 있는지 확인해야 한다.

### 3. 체납/채무불이행

양식1에서 `예` 체크하려면 실제로 아래 상태여야 한다.

- 국세 체납 없음
- 지방세 체납 없음
- 금융기관 채무불이행 규제 없음
- 보조사업 수행 배제 대상 아님

증빙으로 확인해야 하며, 추정으로 체크하면 안 된다.

## HWP 안내문에서 직접 확인할 항목

한글에서 안내문을 열고 아래 항목을 확인한다.

- 최종 제출처 이메일 주소
- 제출 파일 형식: HWP/HWPX/PDF 허용 여부
- 파일명 규칙
- 서명 방식: 자필서명, 도장, 전자서명 허용 여부
- 증빙자료 원본/사본 요구 여부
- 신분증 마스킹 허용 범위
- 추가 제출서류 존재 여부
- 문의처 전화번호
- 선정 후 다음 절차: 멘토링, 발표, 협약, 사업화자금 등

## 내가 만든 문서의 사용 기준

예비창업자가 맞다면:

- `00_제출_액션체크리스트.md`
- `01_양식1_작성초안.md`
- `02_양식2_증빙자료_구성안.md`
- `03_회신메일_초안.md`
- `04_지원아이디어_요약본.md`

위 문서 그대로 사용 가능하다. 단, 생년월일과 서명은 직접 입력해야 한다.

기창업자가 맞다면:

- `01_양식1_작성초안.md`의 신청구분을 기창업자로 변경
- 기업명, 법인번호, 사업자번호 입력
- 현재 보유 사업 업종코드와 창업 예정 업종코드 입력
- `02_양식2_증빙자료_구성안.md`의 기창업자 구성으로 제출서류 변경
